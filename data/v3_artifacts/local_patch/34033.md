# Solution Note: Date Range Lookup with Condition Matching

The task required retrieving parameter values based on a date falling between two adjacent dates in a time series, while matching a specific item condition.

**Data Structure:**
- Sheet name: `ExcelForum help`
- Header row: Row 1 (with secondary headers in Row 2)
- Source data started at Row 3 in columns A-E:
  - Column A (`Start Date`): Date values defining time periods
  - Column B (`Item`): Item identifiers (e.g., 'A')
  - Columns C-E (`Parameter 1`, `Parameter 2`, `Parameter 3`): Numeric values to retrieve

**Lookup Criteria (starting Row 6):**
- Column I: Item to match
- Column J: Parameter name to retrieve
- Column K: Date to lookup
- Column L: Results position (K6:K10)

**Logic Applied:**
The solution read all source data rows (A3:E5) and for each lookup request in rows 6-10:
1. Extracted the target date from column K
2. Identified which item from column I to match against column B
3. Found the date range in column A where the target date fell between consecutive Start Dates
4. Matched the item condition from column B
5. Retrieved the corresponding parameter value from the appropriate column (C, D, or E) based on the parameter name in column J
6. Wrote the result to column L

The approach handled edge cases where dates fell before the first period or after the last period by selecting the nearest boundary row.
