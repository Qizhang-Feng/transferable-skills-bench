# Solution Note: Dynamic Lookup with Filtering

The task required creating a dynamic lookup formula that filters and returns column headers based on a lookup value in cell B6 of Sheet1.

## Spreadsheet Structure

The data was organized in Sheet1 with:
- **Row 1**: Headers containing color names ('Blue', 'Green', 'Yellow', 'Black', 'Orange', 'Red') in columns B through G
- **Row 2**: Values for lookup key 'A' (containing 0s and 1s)
- **Row 3**: Values for lookup key 'B' (containing 0s and 1s)
- **Column A**: Contained the lookup keys ('A' in A2, 'B' in A3)
- **Cell B6**: Contained the lookup value to match against
- **Cells B9:B10**: Target location for the filtered results

## Python/openpyxl Approach

The solution involved:
1. Reading the lookup value from cell B6
2. Reading the header row (row 1) from columns B through G to get the color names
3. Identifying which row in rows 2-3 matched the lookup value by checking column A
4. Reading the corresponding data values from columns B through G in the matched row
5. Filtering the headers where the corresponding value was greater than 0
6. Writing the filtered color names sequentially to cells B9, B10, etc.

The logic iterated through each column position, compared the numeric value to 0, and collected matching headers in order before writing them to the output range.
