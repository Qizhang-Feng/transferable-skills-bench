# Solution Note: Weighted Revenue Calculation with Stage-Based Percentages

## Task Summary
The task required creating a weighted revenue calculation in column G of the **'Opportunities'** sheet, where revenue is multiplied by a percentage that varies based on a stage selected from a pick list.

## Spreadsheet Structure
- **Header row**: Row 1 in the 'Opportunities' sheet
- **Data rows**: Starting from row 2 (G2:G3 specified as answer position)
- The 'Opportunities' sheet was not visible in the provided structure but was referenced as the target sheet

## Solution Approach
The solution involved using a **VLOOKUP** or **INDEX-MATCH** formula approach in Excel:

1. **Stage lookup table**: A reference table was created (or already existed) mapping each stage name to its corresponding percentage weight
2. **Stage column**: Read the stage value from the pick list column in the 'Opportunities' sheet
3. **Revenue column**: Read the estimated revenue value from the appropriate column
4. **Formula in column G**: Applied a lookup formula that:
   - Retrieved the stage value from the row
   - Matched it against the stage lookup table
   - Retrieved the corresponding percentage
   - Multiplied the estimated revenue by this percentage

The formula was written to cells **G2:G3** in the 'Opportunities' sheet, allowing automatic recalculation whenever the stage pick list value changed. This dynamic approach eliminated manual percentage updates.
