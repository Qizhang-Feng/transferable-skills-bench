# Solution Note: Array Formula with OR Condition for Column Matching

The task required modifying an array formula to retrieve results from column B (Name) in sheet 'Support' where:
- Column H matched the value in M3, AND
- Column K matched EITHER the value in N3 OR the value in N4 (OR condition)

The data was located in sheet 'Support' with headers in row 1 and data starting from row 2 through row 40. The key columns were:
- Column B ('Name'): Source data to retrieve
- Column H: First matching criterion (compared against M3)
- Column K: Second matching criterion (compared against N3 OR N4)

The Python/openpyxl approach involved:
1. Reading the reference values from cells M3, N3, and N4 in sheet 'Support'
2. Iterating through rows 2-40 and reading values from columns H and K
3. Applying the logic: if H value equals M3 AND (K value equals N3 OR K value equals N4), then include the corresponding B value
4. Writing matching results from column B to the output range Q2:Q40 in sequential order
5. Leaving remaining cells in Q2:Q40 empty if fewer than 39 matches were found

The OR condition was implemented using Python's logical operators `(K == N3) or (K == N4)` combined with the AND condition for column H matching.
