# Solution Note: Dynamic Column Reference with Fixed Row Reference in Formula

The task required modifying a formula in sheet **Sheet1** to allow the column reference to change when dragged down while keeping the row reference fixed.

The original formula `=SUM(INDEX(B3:B5,0,MATCH(A7,$B$2:$Z$2,0)))` was located in cell **B7**. The data structure consisted of:
- **Row 2** (B2:E2): Company headers ('Company A', 'Company EF', 'Company EG', 'Company H')
- **Rows 3-5** (B3:E5): Numeric data values for each company
- **Column A** (A7:A10): Company names to lookup

The solution involved using the INDIRECT function combined with ADDRESS to create dynamic column references. The modified formula structure used `INDIRECT(ADDRESS(3,COLUMN(),4))` to reference the starting row (3) with a relative column reference, and `INDIRECT(ADDRESS(5,COLUMN(),4))` for the ending row (5).

The Python/openpyxl approach involved:
1. Reading the lookup values from **column A, rows 7-10** in **Sheet1**
2. Reading the header row from **row 2, columns B-E**
3. Reading the data range from **rows 3-5, columns B-E**
4. For each lookup value, matching it against the headers and summing the corresponding column values
5. Writing the results to **cells B7:B10**

The formula was applied to the range **B7:B10** to produce the desired output.
