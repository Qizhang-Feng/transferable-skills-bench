# Solution Note: Counting Non-Pallet Bin Locations

The data was in sheet **'SKUs as Bins'** with headers in row 1. The task required reading column **'Current Location(s)'** (column B) starting from row 2 through row 6, and writing results to column **'# of Bins'** (column C) in the range **C2:C6**.

The logic applied was:
1. Read each cell value from column B (rows 2-6)
2. Split the string by commas to separate individual bin locations
3. For each location, check if it starts with 'X' or 'Z' (case-sensitive) after stripping whitespace
4. Count only locations that do NOT start with 'X' or 'Z' (these are pallet locations to exclude)
5. Write the count to the corresponding cell in column C

The implementation used Python with openpyxl to:
- Load the workbook and access the 'SKUs as Bins' sheet
- Iterate through rows 2-6
- Parse location strings by splitting on commas and extracting the location prefix before the colon
- Apply conditional logic to filter out locations starting with 'X' or 'Z'
- Write integer counts to cells C2:C6
- Apply formatting: Courier New font at 9pt and all borders (thin style) to the range C2:C6

The solution handled edge cases like single locations, multiple comma-separated locations, and locations with varying formats.
