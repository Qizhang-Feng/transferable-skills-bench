# Solution Note: Remove Last 3 Characters from Postcodes

## Problem
The VBA code had syntax errors and was incorrectly setting all values to FALSE instead of trimming the last 3 characters from postcodes.

## Spreadsheet Details
- **Sheet Name**: 'Data to Import'
- **Target Column**: Column C ('Location')
- **Header Row**: Row 1 (contains 'Location')
- **Data Range**: C2:C6066 (postcodes starting from row 2)

## Python/openpyxl Solution Approach

The data was located in the 'Data to Import' sheet, column C ('Location'), starting from row 2. The header 'Location' was in cell C1.

The solution involved:
1. Loading the workbook and accessing the 'Data to Import' worksheet
2. Reading values from column C, starting at row 2 through the last populated row
3. For each cell in column C (from C2 downward), applying string slicing logic to remove the last 3 characters using `value[:-3]` if the value was not None and had length > 3
4. Writing the trimmed values back to the same cells in column C
5. Saving the workbook

The original postcodes (e.g., 'DY8 3NQ') were transformed to remove the space and final two characters of the outward code (resulting in 'DY8'). The operation was performed in-place, modifying the existing cells rather than creating new columns.
