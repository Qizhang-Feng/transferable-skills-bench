# Solution Note: Extracting Unique Text from Column 3 Based on Column 1 and Column 2 Comparison

The task was completed on sheet **Sheet1** with headers in row 1 and data starting from row 2.

The logic applied was: when columns A (Column 1) and B (column 2) both contained "No", the value from column C (Column 3) was extracted and placed into column D (Column 4). If column C also contained "No", then column D remained empty for that row.

**Specific Implementation:**
- Read data from columns A, B, and C starting at row 2 through row 5
- For each row, checked if both column A and column B contained the text "No"
- When this condition was true, retrieved the value from column C
- If column C contained "No", no value was written to column D
- If column C contained actual text (not "No"), that text was written to the corresponding cell in column D
- The results were written to range **D2:D5**

The Python/openpyxl approach involved iterating through rows 2-5, comparing cell values in columns A and B, and conditionally copying non-"No" values from column C to column D. This effectively filtered unique text entries where the first two columns matched the "No" criteria.
