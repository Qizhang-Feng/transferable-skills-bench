# Solution Note: Extract 10-Digit Numbers Starting with '955' or 'CHK'

## Task Summary
The data was located in **Sheet1** with headers in **Row 2**. The input text strings were in **Column B** (labeled "Description") starting from **Row 3**, and the extracted results needed to be written to **Column C** (labeled "Expected Result") in the range **C3:C8**.

## Approach Used
The task required extracting 10-digit sequences that either:
1. Started with '955' (potentially preceded by '/TES-')
2. Started with 'CHK'

Priority extraction was specified, meaning the '955' pattern took precedence over 'CHK' patterns.

## Python/openpyxl Implementation
The solution read text values from **Column B** starting at **Row 3** through **Row 8** in **Sheet1**. For each cell, a regular expression pattern was applied to search for:
- First priority: 10-digit numbers starting with '955', optionally preceded by '/TES-'
- Second priority: 10-character alphanumeric sequences starting with 'CHK'

The extraction logic used regex pattern matching to identify these sequences within the description text. When a match was found, the 10-digit/character sequence was extracted and written to the corresponding cell in **Column C** (same row). If no match was found, the cell in Column C was left empty or set to an empty string.

The workbook was then saved with the extracted values populated in the specified range.
