# Solution Note: Deleting Rows Based on Column C Values

The task required removing rows from the **'imported Data'** sheet where Column C ('Current Balance') either contained blank values or values between -1 and 1 (inclusive).

## Spreadsheet Structure
- **Sheet Name**: 'imported Data'
- **Target Column**: Column C ('Current Balance')
- **Header Row**: Row 1
- **Data Range**: Rows 2-6 (C2:C6)

## Python/openpyxl Approach

The solution involved reading values from **Column C** starting from **Row 2** (first data row after the header in Row 1). The logic applied was:

1. Iterate through all rows in the 'imported Data' sheet from bottom to top (reverse order to avoid index shifting issues when deleting rows)
2. For each row, check the value in **Column C**:
   - If the cell is blank/None, mark the row for deletion
   - If the value is numeric and falls within the range -1 ≤ value ≤ 1, mark the row for deletion
3. Delete the identified rows using the `delete_rows()` method

The reverse iteration approach was critical because deleting rows from top to bottom would shift subsequent row indices, causing rows to be skipped. By iterating from the last row upward, each deletion didn't affect the indices of rows yet to be processed.

The final result preserved only rows where Column C values were outside the -1 to 1 range.
