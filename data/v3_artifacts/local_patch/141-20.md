# Solution Note: Delete Matching Rows Across Two Reconciliation Sheets

The task required creating a macro to find and delete matching rows between two sheets: 'PL Recon Items' and 'Statement Recon Items'.

**Data Structure:**
- Sheet 'PL Recon Items' had headers in row 1: 'Transaction Date', 'Branch', 'Invoice No.', 'Amount'
- Data started in row 2
- Key columns: 'Invoice No.' (column C) and 'Amount' (column D)

- Sheet 'Statement Recon Items' had headers in row 1: 'Code', 'Name', 'Transaction Category Type', 'Period Month', 'Period Year', 'Reference', 'Run', 'Date Processed', 'Total Invoice Value'
- Data started in row 2
- Key columns: 'Reference' (column F) and 'Total Invoice Value' (column I)

**Matching Logic:**
The macro needed to identify rows where:
- 'Invoice No.' (column C) in 'PL Recon Items' matched 'Reference' (column F) in 'Statement Recon Items'
- AND 'Amount' (column D) in 'PL Recon Items' matched 'Total Invoice Value' (column I) in 'Statement Recon Items'

**Python/openpyxl Approach:**
1. Loaded both sheets from the workbook
2. Iterated through data rows in 'PL Recon Items' starting from row 2
3. For each row, extracted values from columns C and D
4. Searched 'Statement Recon Items' for matching values in columns F and I
5. Collected matching row indices from both sheets
6. Deleted matching rows in reverse order to maintain index integrity
7. Saved the workbook
