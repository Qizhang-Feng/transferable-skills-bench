# Solution Note: Delete Blank Rows and Row Above in Column A

## Task Summary
The task required deleting blank rows in Column A AND the row directly above each blank row, while preserving data rows immediately following headers containing "Line No". Specifically, rows 2 and 9 were to be protected from deletion as they contained important data.

## Spreadsheet Structure
- **Sheet Name**: Sheet1 (starting point; logic applies to BR1 onwards in actual implementation)
- **Header Rows**: Row 1 and Row 5 contained "Line No" in Column A
- **Data Rows**: Row 2 contained data (2, BTY1, Inv, 9814) that must be preserved
- **Blank Rows**: Rows 3, 4, and 6 had empty Column A values
- **Column Range**: Columns A through E contained the relevant data

## Python/openpyxl Approach
The solution read Column A (column index 1) starting from row 1 to identify the pattern. The logic identified rows where Column A was blank, then marked both the blank row and the row immediately above it for deletion. However, a critical exception was implemented: if the row above a blank row immediately followed a header row containing "Line No" in Column A, that data row was protected from deletion. The deletion process worked backwards (from bottom to top) to maintain correct row indices during deletion. This logic was applied across multiple sheets starting with 'BR1'.
