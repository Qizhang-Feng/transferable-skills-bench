# Solution Note: S-to-Vowel Word Pair Finder

The dictionary words were located in column A of Sheet1, starting from row 1 (no header row). The task required finding all word pairs where one word differs from another only by having all instances of 'S' replaced with a single vowel (A, E, I, O, or U).

The approach implemented:
1. Read all words from column A of Sheet1 starting at A1
2. For each word containing the letter 'S', generated five candidate words by replacing all 'S' characters with each vowel (A, E, I, O, U)
3. Checked if each candidate word existed in the dictionary
4. When a match was found, stored the pair (original word with S, matching word with vowel)
5. Ensured no duplicate pairs by tracking already-processed combinations

The results were written to columns G and H starting at row 2 (G2:H2 onwards), with headers "Word with S" and "Word with Vowel" placed in G1 and H1 respectively.

After writing all pairs, the entire range G1:H (including headers) was sorted together in ascending order using case-insensitive alphabetical sorting on column G as the primary key.

The solution used Python with openpyxl to read the worksheet, process the word comparisons in memory using dictionaries and sets for efficient lookup, then wrote the results back to the specified columns before applying the sort operation to the output range.
