# Solution Note: Delete Blank Rows in Dataset

## Task Summary
The task required removing blank rows from a dataset in sheet **"SH"** to eliminate gaps between data rows.

## Spreadsheet Structure
- **Sheet Name**: "SH"
- **Header Row**: Row 1 contained column headers: 'Invoice', 'Type', 'Plan', 'Order', 'Code', 'Decription', 'Start Plan Date', 'Invoice Amount', 'Outstanding Amount', 'Plate', 'Registration Date', 'Ageing'
- **Data Range**: Started at row 2 and extended through the dataset
- **Target Output Range**: A6:L11 (where cleaned data was to be written)

## Solution Approach
The solution used Python with openpyxl to:

1. **Load the workbook** and access sheet "SH"
2. **Read all data rows** starting from row 2 (after the header in row 1)
3. **Identify non-blank rows** by checking if any cell in the row contained a non-empty value across all columns (A through L)
4. **Filter out blank rows** where all cells in columns A-L were empty
5. **Write the filtered data** to the target range starting at cell A6, preserving the original column structure (columns A through L)
6. **Save the workbook** with the cleaned dataset

The key logic involved iterating through each row, evaluating whether all cells were empty, and collecting only rows with at least one non-empty cell value for output.
