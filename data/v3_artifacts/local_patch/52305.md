# Solution Note: Time-Range Heat Map Table in Excel

The task required creating a count table that filters entries from **Sheet1** based on two criteria: account name and time range (ignoring date).

## Data Structure
- **Source data** was located in Sheet1 with headers in Row 1:
  - Column A: 'Type' (Reg/SL)
  - Column B: 'Time ' (datetime stamps)
  - Column C: 'Name' (Name One, Name Two, etc.)
  - Column D: 'Destination' (numeric codes)
- **Data rows** started at Row 2 and continued downward

## Output Table Structure
- **Time parameters** were in Row 5:
  - Cell H5: 'Time Start' label
  - Cell I5: 'Time End' label
  - Cell H6 onwards: Start times (e.g., '13:00:00')
  - Cell I6 onwards: End times (e.g., '13:30:00')
- **Name headers** were in Row 5, columns J through N (Name One through Name Five)
- **Results table** was written to range J6:N24

## Python/openpyxl Approach
The solution iterated through each cell in the output range (J6:N24). For each cell, it:
1. Extracted the corresponding name from the column header (Row 5)
2. Extracted the time range from columns H and I of the same row
3. Parsed datetime values from column B (Time), extracting only the time component
4. Counted rows where column C matched the name AND the time fell within the specified range
5. Wrote the count to the appropriate cell
