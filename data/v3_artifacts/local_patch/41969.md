# Solution Note: Shifting COUNTBLANK Formula Pattern

The task required applying a COUNTBLANK formula pattern across multiple cell ranges in Sheet1, where each range shifted by three columns. The source data was in Row 3 (A3:I3), and the results needed to be written to cells A6:C6.

## Approach Used

The data was located in Sheet1, Row 3, spanning from column A through column I. The pattern started with `=COUNTBLANK(A3:C3)` and needed to shift by three columns for each subsequent calculation: `=COUNTBLANK(D3:F3)`, `=COUNTBLANK(G3:I3)`, etc.

Using Python with openpyxl, the solution:

1. **Read the source data**: Extracted values from Row 3, columns A through I (cells A3:I3)
2. **Applied the logic**: For each group of three consecutive columns, counted the number of blank cells in that range. The first group was A3:C3, the second was D3:F3, and the third was G3:I3.
3. **Wrote the results**: Placed the three calculated COUNTBLANK values into cells A6, B6, and C6 respectively in Sheet1.

The solution iterated through the source range in steps of three columns, evaluating each three-column segment for blank cells (empty strings or None values), and populated the output row with the count results. This automated the pattern without requiring manual formula adjustment for each range.
