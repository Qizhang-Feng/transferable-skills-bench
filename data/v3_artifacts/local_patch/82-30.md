# Solution Note: Filtering Whole Numbers from Raw Data

The task required extracting only whole numbers (integers) from the 'Raw Data' sheet and arranging them in the 'Numbers' sheet with a maximum of 6 numbers per row, matching the order shown in 'Manual Result'.

## Data Structure
The data was located in the 'Raw Data' sheet, spanning columns A through F, starting from row 1 (no headers). Each row contained alternating whole numbers and decimal values across the six columns.

## Approach
Using Python with openpyxl:

1. **Read Data**: Loaded all values from columns A to F in the 'Raw Data' sheet, iterating through all populated rows starting from row 1.

2. **Filter Logic**: Checked each cell value to determine if it was a whole number by verifying:
   - The value was numeric
   - The value had no decimal component (either an integer type or a float equal to its integer conversion)
   - Excluded all decimal values (like 0.0253968253968254)

3. **Collect and Arrange**: Collected all whole numbers in their original order (reading left-to-right, top-to-bottom across the range).

4. **Write Output**: Wrote the filtered whole numbers to the 'Numbers' sheet starting at cell A1, filling columns A through F with a maximum of 6 values per row before moving to the next row.

The result preserved the exact sorting order as shown in the 'Manual Result' sheet.
