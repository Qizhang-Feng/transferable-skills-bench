# SUMPRODUCT Formula Correction - Solution Note

## Problem Context
The task involved fixing a SUMPRODUCT formula error in Excel that matched multiple criteria and computed values from a data range.

## Spreadsheet Structure
The data was located in **Sheet1** with:
- Header row at **row 3** containing age range categories ('3M-25yrs', '26-40yrs', '41-45yrs', etc.) spanning columns C through M
- Main data starting at **row 4** through **row 75**
- Column A (rows 4-75): Category values ('Individual')
- Column B (rows 4-75): Sum Insured amounts (150000, 200000, 300000, etc.)
- Columns C through M (rows 4-75): Premium values for different age ranges
- Lookup criteria in cells **R4** (Category), **S4** (Sum Insured), and **T4** (Age Range)
- Results to be written in column U (cells **U4:U75**)

## Python/openpyxl Solution Approach
The solution read the lookup criteria from cells R4, S4, and T4. For each row from 4 to 75 in column U:
1. Extracted the Category value from column A and Sum Insured from column B for that row
2. Compared these against criteria in R4 and S4
3. If both matched, identified which column (C-M) corresponded to the age range in T4 by matching against the header row 3
4. Retrieved the premium value from that specific column and row
5. Wrote the result (or 0 if no match) to the corresponding cell in column U
