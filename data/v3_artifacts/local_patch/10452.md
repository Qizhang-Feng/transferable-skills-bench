# Solution Note: Vertical Lookup with Prefix Filter

## Task Summary
The task required extracting values from column B in Sheet1 that begin with 'PK' and writing them to column E starting at E4.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Source Data**: Column B (starting from row 3, with header "Data" in B2)
- **Target Location**: Column E, rows 4-12 (with header "Material" in E3)
- **Header Row**: Row 2 contained column headers; Row 3 contained the first data label

## Python/openpyxl Approach
The solution involved:

1. **Reading Source Data**: Iterated through column B in Sheet1, starting from row 3 (B3 onwards) to capture all data values including "TOTAL COMPANY", "FABRIC CARE", "BONUS GOLD", and material codes

2. **Applying Filter Logic**: For each cell value in column B, checked if the value was not None and if it started with the prefix 'PK' using Python's `startswith()` method

3. **Writing Filtered Results**: Only values meeting the 'PK' prefix condition were collected into a list and then written sequentially to column E starting at cell E4

4. **Result**: The filtered list populated E4:E12, containing only the material codes that began with 'PK', eliminating all other entries like "TOTAL COMPANY", "FABRIC CARE", and "BONUS GOLD"

This approach automated the manual filtering process without requiring intermediate copy-paste operations or Excel filters.
