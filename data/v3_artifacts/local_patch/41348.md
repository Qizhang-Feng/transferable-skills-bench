# Solution Note: Marking Projects with Technology Components

The data was in sheet **Sheet1** with headers in row 1 and data starting in row 2 (rows 2-11).

## Columns Used:
- **Column H** (Project ID): Identified each project
- **Column I** (Linked to Budget): Indicated if project was linked to budget ('Y' or 'N')
- **Column J** (Budget ID): Grouped related project lines together
- **Column L** (Budget Type): Contained revenue types ('Technology' or 'Services')
- **Column Q** (Project includes tech?): Target column for output (Q2:Q11)

## Logic Applied:

1. **First pass**: Read all data rows and grouped them by Budget ID (column J). For each Budget ID group, checked if any line had 'Technology' in column L (Budget Type).

2. **Second pass**: For each row in Q2:Q11:
   - If column I (Linked to Budget) was 'N', wrote 'N' to column Q
   - If column I was 'Y', checked if the row's Budget ID (column J) belonged to a group containing at least one 'Technology' entry
   - Wrote 'Y' if technology was found in the group, 'N' otherwise

This approach ensured all lines sharing the same Budget ID received the same technology indicator, regardless of their individual Budget Type values.
