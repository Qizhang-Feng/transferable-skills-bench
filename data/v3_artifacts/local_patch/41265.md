# Solution Note: Root Word Matching and Search Volume Summation

## Task Summary
The data was located in **Sheet1** with headers in **Row 1**. The keyword data was in **column A (A2:A6)** with corresponding search volumes in **column B (B2:B6)**. Root words to match were listed in **column D (D2:D5)**, and the results needed to be written to **column E (E2:E5)**.

## Approach Used
The solution read the keywords from **column A (cells A2 through A6)** and their search volumes from **column B (cells B2 through B6)**. Root words were read from **column D (cells D2 through D5)**.

For each root word in column D, the logic performed exact word matching against the keywords in column A. The matching algorithm split each keyword into individual words and checked if the root word existed as a complete, standalone word (not as a substring). For example, "garden" as a root would match "garden tools" and "garden tool" but not "gardening".

When a match was found, the corresponding search volume from column B was added to a running sum for that root word. The final summed values were written to **column E (E2:E5)**, with each cell corresponding to its root word in the same row of column D.

The implementation used Python with openpyxl to read, process, and write the data back to the spreadsheet.
