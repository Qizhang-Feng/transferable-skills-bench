# Solution Note: Extract Check Numbers and Update Payment Status

## Task Summary
The data was located in **Sheet1** with headers in **row 1** and data starting from **row 2**. The task required processing column E (labeled "Payee") to identify cells containing the text "Check", extract numerical values from those cells, place the numbers in the corresponding cells in **column D** (labeled "Check"), and update the original text in column E to "Check Paid".

## Python/openpyxl Approach
1. **Read Operation**: Loaded the workbook and accessed **Sheet1**. Iterated through **column E** starting from **row 2** (first data row after headers).

2. **Logic Applied**: For each cell in column E:
   - Checked if the cell value contained the substring "Check"
   - Used regular expression pattern `r'\d+'` to extract numerical values from the text
   - If a number was found, converted it to an integer

3. **Write Operations**: 
   - Wrote the extracted numerical value to the corresponding cell in **column D** (same row)
   - Updated the original cell in **column E** to display "Check Paid"

4. **Data Range**: The operation processed rows 2 through 21 based on the presence of data in column E.

The solution successfully transformed entries like "Check 90108" by placing "90108" in column D and replacing the column E text with "Check Paid".
