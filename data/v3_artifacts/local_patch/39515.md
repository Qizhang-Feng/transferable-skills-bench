# Solution Note: Month-Year Based Value Lookup

The task required extracting values from monthly columns (JAN-DEC) in **Sheet1** based on both the month in column A ('Month Available') and year in column B ('Year Available'), with results written to column O ('Increase Based on Availability').

**Data Structure:**
- Header row: Row 1
- Data rows: Starting from row 2
- Monthly value columns: C through N (JAN through DEC)
- Lookup criteria: Column A (month name) and Column B (year)
- Output column: O (starting at O2)

**Python/openpyxl Approach:**

The solution read the 'Month Available' values from column A and 'Year Available' values from column B for each data row (rows 2-6). For each row, the logic:

1. Extracted the month abbreviation from column A (e.g., "Mar", "Oct")
2. Mapped the month abbreviation to the corresponding column letter (C=JAN, D=FEB, etc.)
3. Read the value from the matched month column in the same row
4. Wrote the extracted value to column O in the corresponding row

This approach correctly handled year-specific lookups by processing each row independently, ensuring that 2022 rows retrieved their respective month values from 2022 data rows, and 2023 rows retrieved from 2023 data rows. The month-to-column mapping was implemented using a dictionary or conditional logic to translate three-letter month abbreviations to column positions C through N.
