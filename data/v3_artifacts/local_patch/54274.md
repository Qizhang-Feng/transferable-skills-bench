# Solution Note: Nested IF Formula for Performance Points Calculation

The task required creating a nested IF formula in cell G4 of Sheet1 to calculate performance points based on comparing a Total value against Target, Stretch, and Over Stretch thresholds.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Header Row**: Row 3
- **Data Row**: Row 4
- **Key Columns**:
  - Column B: Target (value in B4)
  - Column C: Stretch (value in C4)
  - Column D: Over Stretch (value in D4)
  - Column E: Weight (value in E4)
  - Column F: Total (value in F4)
  - Column G: Points (formula destination in G4)

## Logic Applied
The formula implemented a three-tier conditional structure:
1. If Total (F4) ≤ Target (B4): Points = Total / Target
2. If Total (F4) > Target but < Stretch (C4): Points = 1 + ((Total - Target) / (Stretch - Target)) × Weight
3. If Total (F4) ≥ Stretch: Points = 1 + Weight + ((Total - Stretch) / (Over Stretch - Stretch)) × additional weight factor

## Python/openpyxl Approach
The solution read values from cells B4, C4, D4, E4, and F4 in Sheet1, applied the nested conditional logic using Python if-elif-else statements, and wrote the calculated result to cell G4. The formula structure was preserved as an Excel formula string for native spreadsheet functionality.
