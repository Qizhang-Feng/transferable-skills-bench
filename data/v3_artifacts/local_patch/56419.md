# Solution Note: Extract Non-Zero Quantity Types with Conditional Formatting

## Task Summary
The data was located in **Sheet1** with headers in **row 1**. Column **A** ('Type') contained text values (A, B, C, D, E, etc.) and column **B** ('Quantity') contained numeric values. The task required extracting only the 'Type' values where 'Quantity' was non-zero, outputting results to column **H** ('Unique List') starting from **H2** through **H27**.

## Python/openpyxl Approach

1. **Read Data**: Loaded the workbook and accessed **Sheet1**. Read column **A** ('Type') and column **B** ('Quantity') starting from **row 2** (first data row after headers).

2. **Filter Logic**: Iterated through rows 2-27, checking if the corresponding 'Quantity' value in column **B** was non-zero. For each non-zero quantity, captured the associated 'Type' value from column **A**.

3. **Write Results**: Wrote the filtered 'Type' values sequentially to column **H** starting at **H2**. For rows where no more non-zero values existed, left cells empty or applied error handling.

4. **Formatting**: Applied background color **#92D050** (light green) to all cells in the range **H2:H27** using openpyxl's `PatternFill` with `fill_type='solid'`.

5. **Error Handling**: Ensured the formula handled cases where fewer non-zero values existed than the output range by leaving remaining cells blank.
