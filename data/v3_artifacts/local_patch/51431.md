# Solution Note: Extracting First Name from Semicolon-Separated List

The data was located in **Sheet1**. The source column was **"Names:"** (column A), with the header in row 1 and data starting in row 2. The target column was **"Primary Provider"** (column B), where the extracted first name needed to be written.

## Task Requirements
The task required extracting the first full name from a semicolon-separated list of names in column A. Each name followed the format "LastName, FirstName" and multiple names were separated by semicolons (e.g., "Nizghi, Righat; Maher, Tammy; Ayer, Janet").

## Python/openpyxl Approach
The solution read values from column A (the "Names:" column) starting at row 2. For each cell containing name data:

1. The string was split by semicolon (";") to separate individual names
2. The first element from the resulting list was extracted
3. Leading and trailing whitespace was stripped from the extracted name
4. The cleaned first name was written to the corresponding cell in column B (the "Primary Provider" column)

The formula-based approach would have used `=TRIM(LEFT(A2,FIND(";",A2)-1))` but the Python solution provided more flexibility for handling edge cases like missing semicolons or empty cells. The openpyxl library was used to read from column A and write results to column B in Sheet1.
