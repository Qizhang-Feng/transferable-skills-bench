# Solution Note: Delete Rows Above "Invoice No." in Column A

The task required searching for the text "Invoice No." in column A of sheet 'Sheet1' and deleting all rows above its first occurrence, while preserving the "Invoice No." row and everything below it.

## Approach

The data was located in sheet 'Sheet1'. The search was performed in column A (the first column) to find the case-insensitive match for "Invoice No."

The solution involved:
1. Loading the workbook and accessing sheet 'Sheet1'
2. Iterating through column A cells starting from row 1
3. Performing a case-insensitive search by converting cell values to lowercase and checking if "invoice no." was present in the text
4. Once the first occurrence was found, recording that row number
5. Deleting all rows from row 1 up to (but not including) the row containing "Invoice No."
6. Using the `delete_rows()` method with the appropriate range

The spreadsheet initially had data starting from row 1, with empty rows at the top (rows 1-3) and actual content beginning at row 4. The "Invoice No." text was located further down in the sheet. After deletion, the "Invoice No." row became the new first row, with all subsequent data (including rows with values like 'Account Number: 25791QW', 'BR1', 'SW') preserved in their relative positions below it.
