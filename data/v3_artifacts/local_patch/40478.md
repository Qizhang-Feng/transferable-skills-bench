# Solution Note: Text Extraction Between Second Space and Dash

## Task Summary
The data was located in **Sheet1**, with source text in **column A** and target extraction results in **column B**, specifically cells **B1:B3**.

## Data Structure
- **Header row**: Row 1 contained data (no separate header)
- **Data rows**: Rows 1-3 contained the text strings to process
- **Source column**: Column A contained strings in the format "PO [number] [text] - [name]"
- **Target column**: Column B required extracted text between the second space and the dash character

## Python/openpyxl Approach
The solution read values from **column A (cells A1:A3)** in **Sheet1**. For each cell, the logic applied was:

1. Located the position of the second space character in the string
2. Located the position of the dash character ("-")
3. Extracted the substring starting after the second space and ending before the dash
4. Trimmed any leading/trailing whitespace from the extracted text

The extracted results were written to **column B (cells B1:B3)** in the same sheet. The formula logic used string manipulation methods to find character positions and perform substring extraction, handling cases where the pattern might not exist by returning an empty string.
