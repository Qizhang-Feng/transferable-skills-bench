# Solution Note: Dynamic Multi-Column Sort with VBA

## Task Summary
The task required creating a VBA macro to dynamically sort data in **Sheet1** without hardcoding column positions, with the sorted result matching the structure shown in **Sheet2**.

## Data Structure
- **Source Sheet**: Sheet1
- **Header Row**: Row 1 (columns A-F: 'Sr. No', 'Team', 'Task', 'Start Date', 'End Date', 'Responsibility')
- **Data Range**: Rows 2-14 (based on the answer position A1:F14)
- **Sort Columns**: 'Task' (column C) and 'Responsibility' (column F)

## Solution Approach
The VBA macro dynamically located the 'Task' and 'Responsibility' column positions by searching through the header row (Row 1) rather than hardcoding column letters. The data range was identified by finding the last row with data in Sheet1.

The sort operation applied two levels:
1. **Primary Sort**: Column 'Task' in ascending order (alphabetically from small to large: CT1, CT2, NT1, etc.)
2. **Secondary Sort**: Column 'Responsibility' in ascending order (R2, R3, R4, etc.)

The macro used Excel's `Sort` object with `SortFields` collection to define the sorting criteria. The `Match` or `Find` function was used to dynamically determine column indices for 'Task' and 'Responsibility' headers, ensuring the macro remained flexible if column positions changed in the future.
