# Solution Note: Extract and Display Month from Date Strings

## Task Summary
The data was in sheet **'Sheet1'**, column **'Data'** (column A), rows 2-6. The task required extracting the month abbreviation from date strings at the end of text entries and writing results to column **'Desired Result'** (column E), rows 2-6.

## Approach Used
The solution read values from column A starting at row 2 (first data row after header in row 1). Each text string in column A contained a date in "MMM YY" format at the end, either preceded by " - " or just a space.

The logic applied was:
1. Extract the last 6 characters from each string in column A using Python string slicing `[-6:]`
2. Split the extracted substring by space to separate month and year components
3. Take the first element (index 0) from the split result to isolate the month abbreviation (e.g., "Oct", "Nov", "Dec")
4. Write the extracted month abbreviation to the corresponding cell in column E

The solution iterated through cells E2 through E6, reading the corresponding A column value for each row, applying the extraction logic, and writing the month abbreviation back to column E.

This approach handled both formats consistently (with or without " - ") since the date was always positioned in the last 6 characters of the string.
