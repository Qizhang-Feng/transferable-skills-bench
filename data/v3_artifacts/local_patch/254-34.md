# Solution Note: Subset Sum Problem - Highlighting Cells

## Task Summary
The task was to identify which values from the 'Before' sheet, when summed together, equal a target sum between 994108 and 994112. The answer was to be written to cell C2 in the 'Before' sheet.

## Spreadsheet Structure
- **Data Location**: Sheet 'Before', column A, rows 1-6 (no headers, data starts at row 1)
- **Values**: Six numeric values ranging from ~22,704 to ~1,238,917
- **Answer Location**: Cell C2 in sheet 'Before'

## Solution Approach
This was a **subset sum problem** requiring identification of which combination of values from column A summed to the target range. The approach involved:

1. **Reading Data**: Used openpyxl to read all numeric values from column A (cells A1:A6) in sheet 'Before'
2. **Algorithm**: Implemented a subset sum algorithm to find all combinations of values that sum to a value within the target range (994108-994112)
3. **Logic**: Iterated through all possible subsets (2^n combinations) and calculated their sums to find matches
4. **Output**: Wrote the identifying information (row numbers or cell references) of the matching cells to cell C2 in sheet 'Before'

The solution required handling floating-point precision issues due to values like "30424.000000000004" and "406152.99999999994". Conditional formatting alone cannot solve this; VBA or Python with proper subset sum logic was necessary.
