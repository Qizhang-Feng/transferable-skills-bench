# Solution Note: Delete Rows with References Starting with 'Z1' or 'X1'

## Task Summary
The task required deleting all rows from **Sheet1** where column E ('Reference') contained values beginning with 'Z1' or 'X1', while preserving the original data in the **Original Data** sheet.

## Spreadsheet Structure
- **Sheet1**: Working sheet with headers in row 1 and data starting from row 2
- **Original Data**: Backup sheet with identical structure
- Column E contained the 'Reference' values to be evaluated
- Headers: Company Name (A), Period Year (B), Period Month (C), Date Processed (D), Reference (E)

## Python/openpyxl Approach
The solution loaded the workbook and accessed **Sheet1**. Starting from row 2 (first data row after headers in row 1), the code iterated through all rows in reverse order to avoid index shifting issues during deletion. For each row, it read the value in column E ('Reference'). If the value started with 'Z1' or 'X1' (using string prefix matching), that entire row was deleted from the worksheet. The reverse iteration was critical to maintain correct row indices during the deletion process.

The **Original Data** sheet remained untouched as a reference. After processing all rows, the modified workbook was saved, with **Sheet1** containing only rows where the Reference column did not begin with 'Z1' or 'X1'.
