# Solution Note: Accounts Receivable Remarks Formula

## Task Summary
The task required updating the Remarks column (J) in an accounts receivable tracking spreadsheet on **Sheet1** to properly handle prepaid scenarios based on the Amount Outstanding value.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Header Row**: Row 5
- **Data Range**: Rows 6-10
- **Key Columns**:
  - Column H: "Amount Outstanding"
  - Column I: "Over Due Days"
  - Column J: "Remarks" (target column)

## Solution Approach
The solution used Python with openpyxl to:

1. Read the "Amount Outstanding" values from column H (starting at H6)
2. Read the "Over Due Days" values from column I (starting at I6)
3. Apply the following logic for each row:
   - If Amount Outstanding is negative → "Prepaid"
   - Else if Over Due Days is empty/blank → empty string
   - Else if Over Due Days < 90 → "Call Customer"
   - Else → "Bad Debts"
4. Write the calculated remarks to column J (J6:J10)

## Implementation Details
The Python script iterated through rows 6-10, evaluated the conditions in order (checking for negative amounts first, then empty overdue days, then the threshold comparison), and populated the Remarks column accordingly. This corrected the original formula which failed to account for prepaid (negative outstanding) scenarios.
