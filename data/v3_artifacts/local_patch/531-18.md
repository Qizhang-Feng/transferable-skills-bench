# Transpose Data Solution Note

The task required transposing data from the 'Data' sheet into three columns (ITEM, DATE, QTY) on the 'Transpose' sheet, with results written to range A3:C970.

## Source Data Structure
The 'Data' sheet contained a header row at row 6 with:
- Column A: 'ITEM' 
- Columns G onwards: Date headers (starting with '2020-11-16 00:00:00', '2020-11-23 00:00:00', etc.)
- Data rows started at row 7

The data was in a pivoted/wide format where each item had one row, and quantities were spread across multiple date columns (columns G through the end of the dataset).

## Python/openpyxl Approach
The solution read from the 'Data' sheet starting at row 6 (headers) and row 7 (data). For each data row:
1. Read the ITEM value from column A
2. Iterated through columns G onwards to get date headers from row 6
3. For each non-empty quantity value in these columns, created a new row with: ITEM (from column A), DATE (from the column header in row 6), and QTY (the cell value)

The transposed results were written to the 'Transpose' sheet starting at cell A3, with headers ITEM, DATE, QTY in row 3, and data beginning at row 4. This converted the wide format into a long/normalized format with 968 data rows.
