# Solution Note: Counting Unique Item Numbers in Bin Tracking

The data was located in **Sheet1** with headers in **row 1** and data starting from **row 2**.

The task required counting unique values in the **'Item No.' column (column E)**, specifically from the range **E2:E10**, and placing the result in cell **H10**.

## Approach Used

The solution used Python with openpyxl to:

1. **Unmerge any merged cells** in the worksheet first to ensure clean data access
2. **Read the 'Item No.' values** from column E (specifically cells E2 through E10)
3. **Extract unique item numbers** by collecting all non-empty values from this range into a set data structure, which automatically handles uniqueness
4. **Count the unique items** by taking the length of the set
5. **Write the count** to cell **H10**

The logic focused on the **'Entry Type' column (column C)** which contained "Place" entries, indicating items being added to the bin. The **'Bin Code' column (column G)** showed all items were associated with bin "BD026".

The solution preserved the ability to track counts across cycles by writing to specific cells in the **H2:H24 range**, allowing historical counts to remain visible when new counting cycles begin after a "Take" event from another department.
