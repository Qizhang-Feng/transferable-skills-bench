# Solution Note: Counting Item Occurrences in Sheet2

The task was to automate counting occurrences of each unique number in Column A of Sheet2, writing the counts to Column B without manually creating a sequential list.

**Spreadsheet Structure:**
- Sheet Name: `Sheet2`
- Header Row: Row 1 containing `['Item', 'Count Of Item']`
- Data Range: Rows 2-21 in Column A contained the item numbers
- Output Range: B2:B21 for the occurrence counts

**Python/openpyxl Approach:**

The solution read values from Column A (specifically cells A2:A21) in Sheet2. A dictionary was used to track the occurrence count for each unique item number as the column was iterated through sequentially.

The logic worked as follows:
1. Loaded the workbook and accessed Sheet2
2. Iterated through cells A2 to A21
3. For each value in Column A, checked if it had been seen before in a tracking dictionary
4. If the item was encountered for the first time, initialized its count to 1; otherwise, incremented the existing count
5. Wrote the current count value to the corresponding cell in Column B (same row)

This approach automatically tracked how many times each item had appeared up to that point in the list, creating a running count that incremented each time a duplicate item was found. The counts were written directly to cells B2:B21 in Sheet2.
