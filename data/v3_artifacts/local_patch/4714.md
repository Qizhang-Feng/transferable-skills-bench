# Solution Note: 4-Month Moving Average Calculation

The data was located in **Sheet2** with headers in row 1 and data starting from row 2. The key columns were:
- **Column A (Emp Code)**: Employee identifier
- **Column C (Month)**: Date values for chronological ordering
- **Column D (Hours)**: Numeric values to calculate averages from
- **Column E (Moving Average Amount)**: Output column for calculated averages (E2:E25)
- **Column F (Moving Aveage (four months exceed 48))**: Output column for threshold checks

The Python/openpyxl approach involved:

1. Reading all data rows from columns A, C, and D in Sheet2
2. For each row starting at row 2, looking back at the current and previous 3 rows (4-month window) that belonged to the same employee (matching Emp Code in column A)
3. If fewer than 4 months of data existed for that employee, writing "n/a" to column E and "n/a - less that 4 weeks" to column F
4. If 4 months existed, calculating the average of the Hours values (column D) from those 4 months and writing to column E
5. Formatting the average as a whole number when the decimal was .0 (e.g., "50" instead of "50.0")
6. Writing "Less that 48" or appropriate text to column F based on whether the average exceeded 48
7. The calculation respected employee boundaries, only averaging within the same Emp Code
