# Solution Note: Extracting Lighting Data from Source Table

The task involved filtering and extracting specific category data from a source table in **Sheet1**.

**Source Data Structure:**
- Located in range **B3:D22** on **Sheet1**
- Header row at **row 3** with columns: "Item" (column B), "kW rating" (column C), "Number" (column D)
- Data started from **row 4** onwards

**Target Output Structure:**
- Located in range **F3:H6** on **Sheet1**
- Header row at **row 3** with columns: "Item" (column F), "kW rating" (column G), "Number" (column H)
- Data output area: **F4:H6** (3 rows for filtered results)

**Python/openpyxl Approach:**
1. Read the source data from columns B, C, and D starting from row 4 through row 22
2. Filter rows where the "Item" column (column B) contained the text "Lighting"
3. Extract the matching rows' values from all three columns (Item, kW rating, Number)
4. Write the filtered results sequentially into the output range starting at cell F4
5. Populate columns F, G, and H with the corresponding Item, kW rating, and Number values from each matching row
6. Leave remaining rows in the output range empty if fewer than 3 lighting items were found

The solution automated the extraction process by reading, filtering, and writing data programmatically without hardcoding specific values.
