# Solution Note: Deleting Matching Row Pairs in Bank Transactions

The data was located in sheet **'Imported Data'** with headers in row 1 and data starting from row 2. The relevant columns were:
- Column A: 'Reference'
- Column C: 'Debit'
- Column D: 'Credit'
- Column F: 'Narrative'

The task required identifying and deleting row pairs where:
1. One row's Debit value (Column C) matched another row's Credit value (Column D)
2. Both rows had the same Reference number (Column A)
3. Both rows had the same Narrative text (Column F)

**Python/openpyxl Approach:**

The solution read all data rows from the sheet starting at row 2. For each row, the values from columns A (Reference), C (Debit), D (Credit), and F (Narrative) were extracted and stored with their row indices.

The logic compared each pair of rows to find matches where:
- Row i's Column C value == Row j's Column D value
- Row i's Column A == Row j's Column A
- Row i's Column F == Row j's Column F

Matching row pairs were collected and then deleted from the worksheet in reverse order (highest row index first) to avoid index shifting issues. The workbook was saved after all deletions were complete.

The final result was written back to the same range **'Imported Data'!A1:F10**, with matching pairs removed and remaining rows preserved.
