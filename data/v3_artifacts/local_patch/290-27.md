# Solution Note: Remove Alphabetic Characters from Column B

The task required creating a macro to remove alphabetic characters from alphanumeric strings in column B of Sheet1, leaving only numeric values.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Target Column**: Column B (index 2 in openpyxl)
- **Answer Range**: A14:I137
- **Data Characteristics**: Alphanumeric strings with 2-3 uppercase letter codes followed by numbers (e.g., 'PID1', 'GG 1')

## Python/openpyxl Approach

The solution read data from column B in Sheet1, starting from row 14 (the first row of the answer range) through row 137. For each cell value in column B:

1. The string was processed to remove all alphabetic characters (A-Z, a-z) and spaces
2. Only numeric characters (0-9) were retained
3. The cleaned numeric value was written back to the same cell in column B

The logic used regular expressions or string filtering to extract digits from each cell value. Empty cells or cells without numeric content were handled appropriately to avoid errors.

The processing loop iterated through rows 14 to 137 in column B, applying the character removal logic to each non-empty cell and updating the cell value in place. The workbook was saved after all transformations were complete.

This approach preserved all other data in the spreadsheet while specifically targeting only the alphanumeric strings in column B within the specified range.
