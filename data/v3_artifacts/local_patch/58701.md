# Solution Note: Auto-populate Office Code from Location Lookup

## Task Summary
The task required automatically populating office codes in the 'Entry Tab' sheet based on location names entered by users, using a lookup table stored in the 'Table Tab' sheet.

## Spreadsheet Structure
- **Entry Tab**: Contains user-entered locations in column B (starting at B2) with headers in row 1. Office codes needed to be populated in column E (E2:E3).
- **Table Tab**: Contains the lookup reference table with 'Location Name' in column A and 'Office Code' in column B, with headers in row 1 and data starting at row 2.

## Python/openpyxl Approach
The solution read location names from column B of the 'Entry Tab' sheet (cells B2:B3). It then loaded the lookup table from the 'Table Tab' sheet by reading column A ('Location Name') and column B ('Office Code'), starting from row 2 to build a dictionary mapping locations to their corresponding office codes.

For each location value found in 'Entry Tab' column B, the code performed a dictionary lookup to retrieve the matching office code from the reference table. The matched office codes were then written to column E of the 'Entry Tab' sheet (E2:E3).

The solution handled the header row at row 1 and processed data rows starting from row 2 in both sheets.
