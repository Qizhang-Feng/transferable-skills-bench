# Solution Note: Index/Match for Deal Name Lookup

## Task Summary
The task required populating column B ('deal name') in **Sheet2** by looking up values from **Sheet1** based on matching criteria in the 'Existing Data Fields' column.

## Data Structure
- **Sheet1** contained the source data with headers in Row 1:
  - Column A: 'Count'
  - Column B: 'Deal Name' (the values to retrieve)
  - Column C: 'Type'
  - Column D: 'Existing Data Fields' (the lookup key)
  - Data rows: 2-6

- **Sheet2** contained the target structure with headers in Row 1:
  - Column A: 'Existing Data Fields' (the lookup key)
  - Column B: 'deal name' (destination for results)
  - Data rows: 2-6

## Python/openpyxl Approach
The solution read column A ('Existing Data Fields') from **Sheet2** starting at row 2. For each value, it searched through column D ('Existing Data Fields') in **Sheet1** (rows 2-6) to find a matching entry. When a match was found, the corresponding value from column B ('Deal Name') in **Sheet1** was retrieved and written to column B ('deal name') in **Sheet2** at the same row index.

The implementation used a dictionary mapping approach for efficient lookups, iterating through Sheet1 data rows to build the lookup table, then populating Sheet2's column B with the matched values.
