# Solution Note: Deleting Rows Based on Column J and K Criteria in Sheet2

The task required deleting entire rows from **Sheet2** where **both Column J ('sub')** and **Column K ('total')** were either blank or contained specific values: `-`, `0`, `$`, `$0`, or `$0.0`.

## Spreadsheet Structure
- **Sheet Name**: Sheet2
- **Header Row**: Row 1
- **Data Start Row**: Row 2
- **Target Columns**: Column J ('sub') and Column K ('total')
- **Data Range**: A1:K54

## Python/openpyxl Approach

The solution involved reading data from **Sheet2** starting at **Row 2** (after the header in Row 1). For each row, the values in **Column J** and **Column K** were evaluated against the deletion criteria.

The logic checked if both cells met the criteria (blank or containing `-`, `0`, `$`, `$0`, or `$0.0`). Rows were marked for deletion only when **both conditions were true simultaneously**. Rows where at least one column (J or K) contained actual data were preserved.

The implementation iterated through rows in reverse order (from bottom to top) to safely delete rows without affecting subsequent row indices. The `worksheet.delete_rows()` method was used to remove qualifying rows.

After deletion, the modified workbook was saved, preserving all rows where either Column J or Column K contained meaningful data values, as demonstrated by Row 4 and Row 6 which had actual numeric values in at least one of these columns.
