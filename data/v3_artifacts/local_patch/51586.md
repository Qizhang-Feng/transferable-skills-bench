# Solution Note: Store Name City Validation Formula

## Task Summary
The task required checking if the second part of a store name in column A exactly matches the city name in column B, with no additional text after the city name.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Headers** (Row 1): Column A = "Store Name", Column B = "City", Column C = "Formula Required to check if 2nd part of Store Name does not equal City"
- **Data Range**: Rows 2-6
- **Answer Position**: C2:C6

## Python/openpyxl Approach

The solution read data from **Sheet1** starting at row 2 (first data row after headers in row 1). 

For each row:
1. Read the store name from **column A** (Store Name)
2. Read the city name from **column B** (City)
3. Applied logic to extract the text after "ABC " prefix in the store name
4. Checked if the extracted text exactly equals the city name (no additional text allowed)
5. Wrote "True" to **column C** if they matched exactly, "False" otherwise
6. Results were written to cells **C2 through C6**

The formatting requirements included:
- Recreating the original sheet with matching fonts and fill colors
- Changing the fill color specifically for the values in column C (C2:C6)

The validation logic ensured that store names like "ABC Melbourne Central" would return False because "Melbourne Central" contains additional text beyond just "Melbourne".
