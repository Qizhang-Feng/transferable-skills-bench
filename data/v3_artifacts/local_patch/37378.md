# Solution Note: Auto-Incrementing Numbers Based on Column B Entries

The task was completed on **Sheet1** with headers in **Row 1** (columns A and B labeled as '#' and 'A' respectively).

The data started at **Row 2** and extended through **Row 6**. The goal was to populate **column A (cells A2:A6)** with sequential numbers that increment by 1 only when the corresponding cell in **column B (cells B2:B6)** contained data.

## Python/openpyxl Approach:

1. **Read the data**: Accessed **Sheet1** and read values from **column B starting at B2** through the target range (B2:B6)

2. **Applied logic**: Initialized a counter starting at 1. For each row from row 2 to row 6:
   - Checked if the corresponding cell in **column B** was not empty (not None and not an empty string)
   - If the cell contained data, assigned the current counter value to **column A** in that row
   - Incremented the counter by 1 for the next non-empty entry
   - If the cell was empty, left **column A** blank or empty

3. **Wrote results**: Wrote the calculated sequential numbers back to **column A (cells A2:A6)** in **Sheet1**

This approach ensured that only rows with data in column B received sequential numbering in column A, while empty rows in column B resulted in blank cells in column A.
