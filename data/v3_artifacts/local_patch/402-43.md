# Solution Note: Duplicate Rows Based on Count Column

## Task Summary
The task required duplicating rows from **Sheet1** based on a count value specified in column **AG** (labeled 'COUNT'), and writing the expanded results to **Sheet2**.

## Spreadsheet Structure
- **Source sheet**: Sheet1
- **Target sheet**: Sheet2
- **Header row**: Row 1 (columns A through AG: Col1, Col2, Col3...Col32, COUNT)
- **Data rows**: Starting from row 2
- **Count column**: Column AG (33rd column, labeled 'COUNT')
- **Total columns**: 33 columns (A through AG)

## Solution Approach
The solution used Python with openpyxl to:

1. **Read the header row** from Sheet1, row 1 (columns A:AG) and wrote it to Sheet2, row 1
2. **Iterated through each data row** starting from row 2 in Sheet1
3. **Read the COUNT value** from column AG for each row
4. **Duplicated each row** the number of times specified by the COUNT value (e.g., COUNT=1 meant 1 copy, COUNT=2 meant 2 copies)
5. **Wrote duplicated rows sequentially** to Sheet2, appending them at the bottom of the dataset

The logic ensured that all 33 columns (A through AG) were copied for each duplicate, maintaining the complete row structure. The duplicates were inserted at the bottom rather than directly below each original row, creating a clean expanded dataset in Sheet2.
