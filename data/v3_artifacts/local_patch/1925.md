# Dynamic Index Lookup Solution Note

## Task Summary
The task required creating a dynamic lookup that matches a date in `Sheet1!C5` to find the corresponding column in `Sheet2`, then checks if items from `Sheet1!B6:B31` exist in that matched column.

## Spreadsheet Structure
- **Sheet1**: Contains the target date in cell `C5` (2020-02-28) and items list in column B starting from row 6 (B6:B31). Results were written to column C (C6:C31).
- **Sheet2**: Contains dates in row 5 starting from column C (C5:M5) and corresponding item values in row 6 onwards (C6:M6, etc.).

## Solution Approach
The solution used Python with openpyxl to:

1. Read the target date from `Sheet1!C5`
2. Read all dates from `Sheet2` row 5 (columns C through M) to find the matching column index
3. Use `MATCH` logic to locate which column in Sheet2 contains the matching date
4. For each item in `Sheet1!B6:B31`, check if it exists in the corresponding column of `Sheet2` (starting from row 6)
5. Write "NEW" to `Sheet1!C6:C31` if the item was not found in the matched Sheet2 column, otherwise write the item name itself
6. Applied the formula using `IFNA(INDEX(matched_column, MATCH(item, matched_column, 0)), "NEW")` logic programmatically

The header row was row 5, and data started from row 6 in both sheets.
