# Solution Note: Assembling Three-Digit Boxed Numbers

The data was in sheet **'3 straights to 3 boxed'** with headers in row 1 and data starting from row 2. The task required reading three individual digit columns: **'strai_1'** (column B), **'strai_2'** (column C), and **'strai_3'** (column D).

The logic applied was to sort these three digits in ascending order (smallest to largest) and concatenate them into a three-digit string with leading zeros preserved. For example, digits 4, 1, 0 were sorted to 0, 1, 4 and formatted as "014".

The implementation used Python with openpyxl to:
1. Load the workbook and access the **'3 straights to 3 boxed'** sheet
2. Iterate through rows 2 to 16 (the data range)
3. For each row, extract the three digit values from columns B, C, and D
4. Sort the three digits in ascending order
5. Format them as a zero-padded three-character string
6. Write the result to column **G** ('boxed') in the corresponding row

After populating column G (range **G2:G16**), the cells were formatted with:
- Bold font style
- Fill color **#EBF1DE** (light green)

The solution preserved the three-digit format even when digits included zeros, ensuring consistent output formatting across all rows.
