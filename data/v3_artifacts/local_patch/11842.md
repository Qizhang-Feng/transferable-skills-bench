# Solution Note: Staff Attendance Full-Year Calculation

## Task Summary
The task required calculating full-year attendance (January to December) for each staff member based on a selected name in cell 'O3' of the 'Total' sheet, with results written to range 'O5:Z12'.

## Spreadsheet Structure
- **Monthly Sheets**: 'Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec'
- **Header Row**: Row 5 contained column headers including 'NAME' (column A) and calendar dates (columns B through AF/AG depending on month length)
- **Attendance Codes**: Column 'P' contained present days count for each person
- **Data Structure**: Each person had two rows per month (representing first and second half of workday)
- **Data Start**: Row 6 onwards contained staff attendance data

## Python/openpyxl Approach
1. Read the selected person's name from 'Total'!O3
2. For each monthly sheet (Jan through Dec):
   - Located the person's name in column A (starting from row 6)
   - Read the 'P' (Present) count from column 'P' for matching rows
   - Summed present days across both rows for that person in that month
3. Wrote the monthly totals horizontally across 'Total'!O5:Z5 (or appropriate row based on person selection)
4. The solution handled multiple staff members by iterating through rows O5:O12 if needed, calculating each person's monthly attendance across columns O through Z
