# Solution Note: Text Lookup Formula for BA/MA Designation

## Task Summary
The task required creating a formula to extract "BA" or "MA" text from degree lane codes and populate column F (BA/MA) in Sheet1.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Header Row**: Row 1
- **Data Rows**: Rows 2-5
- **Source Column**: Column D (FY20 Lane) containing values like "BA", "BA+10", "MA", "MA+30"
- **Target Column**: Column F (BA/MA) where the extracted values needed to be written
- **Target Range**: F2:F5

## Python/openpyxl Approach
The solution read data from column D (FY20 Lane) starting at row 2. For each cell in D2:D5, the logic checked whether the cell value contained "MA" or "BA" as a substring. The search prioritized "MA" first (since "MA" should be detected even in values like "MA+30"), then checked for "BA" (to catch "BA", "BA+10", etc.). 

The extracted text ("MA" or "BA") was then written to the corresponding cell in column F (BA/MA) at the same row position. If neither "MA" nor "BA" was found in the source cell, an empty string was written to column F.

This approach used conditional logic with substring matching to parse the degree designation from the lane code and populate the BA/MA column accordingly.
