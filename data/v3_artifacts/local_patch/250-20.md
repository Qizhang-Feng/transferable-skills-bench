# Solution Note: Consolidating Duplicate Rows by Summing Column J

## Task Summary
The task required identifying rows with duplicate combinations in columns B (NUMBER) and C (LINE NO) on the 'RNM' sheet, summing their corresponding values in column J (MATCHED_QTY), and retaining only one row per unique B-C pair with the summed value.

## Spreadsheet Structure
- **Sheet name**: 'RNM'
- **Header row**: Row 1
- **Data start row**: Row 2
- **Key columns**: 
  - Column B: 'NUMBER'
  - Column C: 'LINE NO'
  - Column J: 'MATCHED_QTY' (values to sum)

## Python/openpyxl Approach
The solution read all data from the 'RNM' sheet starting at row 2. For each row, a composite key was created by concatenating the values from column B (NUMBER) and column C (LINE NO). 

A dictionary was used to track unique B-C combinations. For each composite key:
- If encountered for the first time, the entire row data was stored
- If already present, the value in column J (MATCHED_QTY) was added to the existing sum for that key

After processing all rows, the sheet was cleared (preserving the header in row 1), and the consolidated rows were written back starting at row 2. Each unique B-C pair appeared only once with the summed MATCHED_QTY value in column J, while all other column values were preserved from the first occurrence of each pair.
