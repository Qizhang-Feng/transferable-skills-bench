# Solution Note: Conditional Formula for Folder Created Status

The task required creating a formula in column J ("Folder Created") on sheet "Sheet1" that evaluates column H ("Docusign Sent") with three conditions:
- Display "Yes" if H contains a date or the text "Yes"
- Display "N/A" if H contains "N/A"
- Display "No" if H is blank

The data structure had headers in row 1 and data starting in row 2. The target cells were J2:J4.

**Python/openpyxl Approach:**

The solution read values from column H ("Docusign Sent") starting at row 2 on sheet "Sheet1". For each cell in H2:H4, the logic applied was:
1. First checked if the cell value equals "N/A" (string comparison)
2. Then checked if the cell contained a datetime object or the string "Yes"
3. Otherwise (blank cells), returned "No"

The results were written to the corresponding cells in column J ("Folder Created"), specifically J2:J4.

The key was handling mixed data types in column H: datetime objects (like 2022-01-02), text strings ("Yes", "N/A"), and empty cells. The isinstance() function was used to detect datetime objects, while string comparisons handled text values. This approach correctly evaluated all three conditions without the limitations of the original Excel formula.
