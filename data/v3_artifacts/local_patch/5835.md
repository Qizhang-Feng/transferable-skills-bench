# Solution Note: Populating Log Values Using Lookup Logic

The task required populating the "Log Value" column (column C) in the "Requested table" section of sheet **גיליון1** by matching data from the "Basic Table" section.

**Data Structure:**
- Headers were in row 2
- Requested table occupied columns A-C, with data starting at row 3
- Basic Table occupied columns G-I, with data starting at row 3
- Column A (Stock) and Column B (Year) in the Requested table needed matching
- Column H (Stock) and Column G (Year) in the Basic Table contained lookup keys
- Column I (Log Value) in the Basic Table contained the values to retrieve

**Python/openpyxl Approach:**
The solution read data from the Basic Table (columns G, H, I starting at row 3) and created a lookup dictionary using tuples of (Year, Stock) as keys mapped to their corresponding Log Value from column I. 

For each row in the Requested table (starting at row 3, range C3:C19), the code extracted the Stock value from column A and Year value from column B, created a lookup key tuple (Year, Stock), and retrieved the matching Log Value from the dictionary.

The retrieved values were then written to column C (Log Value) in the Requested table at the corresponding row positions. Empty or missing matches resulted in empty cells or zero values being written.
