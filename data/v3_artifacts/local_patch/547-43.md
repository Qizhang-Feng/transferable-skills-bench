# Solution Note: Transpose and Enrich Employee Data

## Task Summary
The task required transposing selected columns from the 'Emp' sheet into a long format while enriching each row with lookup data from the 'Lookup' sheet.

## Data Structure
**Source Sheet: 'Emp'**
- Header row: Row 1
- Data started: Row 2
- Key columns to transpose: 'Comp1', '2_comp', 'dept1', 'avg sal', 'hike' (columns E through I)
- Columns to keep constant: 'seqno', 'Empno', 'ename', 'Jobid' (columns A through D)
- Column to exclude: 'performance' (column J)

**Lookup Sheet: 'Lookup'**
- Header row: Row 1
- Data started: Row 2
- Columns: 'Variable' (A), 'Category' (B), 'Subcategory' (C)

**Output Sheet: 'Expected'**
- Started at cell A2
- Columns: 'seqno', 'Empno', 'ename', 'Jobid', 'Variable', 'Category', 'Subcategory', 'Value'

## Approach
The solution iterated through each row in the 'Emp' sheet (rows 2-4, excluding empty rows). For each employee row, it created multiple output rows—one for each transposed variable. The constant columns (A-D) were repeated for each output row. The variable name became the 'Variable' column value, and its corresponding cell value became the 'Value' column. A VLOOKUP-style match was performed against the 'Lookup' sheet to populate 'Category' and 'Subcategory' columns based on the variable name.
