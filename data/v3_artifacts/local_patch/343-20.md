# Solution Note: Extract Duplicate Records from Column C

## Task Summary
The task required identifying duplicate values in column C ('Reference') from the 'Imported Data' sheet and copying entire rows containing duplicates to the 'Duplicates' sheet, sorted by 'Balance' and 'Invoiced Date' in ascending order.

## Spreadsheet Structure
- **Source Sheet**: 'Imported Data'
- **Target Sheet**: 'Duplicates'
- **Header Row**: Row 1 (columns A-E: Division, Stock Number, Reference, Invoiced Date, Balance)
- **Data Start Row**: Row 2

## Python/openpyxl Approach

1. **Read Data**: Loaded the 'Imported Data' sheet and extracted all data starting from row 2. Column C ('Reference') was the key column for identifying duplicates.

2. **Identify Duplicates**: Used Python's `collections.Counter` to count occurrences of each value in column C. Rows where the Reference value appeared more than once were flagged as duplicates.

3. **Sort Duplicates**: Sorted the duplicate rows first by column E ('Balance') then by column D ('Invoiced Date'), both in ascending order.

4. **Write to Target**: Wrote the header row to 'Duplicates'!A1:E1, then populated the sorted duplicate rows starting from row 2 in the 'Duplicates' sheet.

5. **Preserve Formatting**: Maintained cell alignment and data types when copying rows to ensure consistency with the source data.
