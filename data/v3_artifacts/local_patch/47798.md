# Solution Note: Extracting Collection Names from Tag Strings

## Task Summary
The data was located in sheet **'Sheet1'** with headers in row 1. Column **'Tags'** (column A) contained comma-separated tag strings, and column **'Collection'** (column B) was the target output column where extracted collection names needed to be written.

## Data Structure
- **Header row**: Row 1
- **Data rows**: Rows 2-6
- **Input column**: 'Tags' (column A, cells A2:A7)
- **Output column**: 'Collection' (column B, cells B2:B7)

## Solution Approach
The solution used Python with openpyxl to:

1. Load the workbook and access sheet 'Sheet1'
2. Iterate through cells A2:A7 in the 'Tags' column
3. For each cell, search for the substring 'Collection-' within the tag string
4. Extract the text immediately following 'Collection-' up to either:
   - The first comma character, or
   - The end of the string if no comma follows
5. Write the extracted collection name to the corresponding cell in column B (cells B2:B7)

The logic handled cases where 'Collection-' appeared at different positions within the comma-separated tag string. The extraction used string manipulation methods (find/split operations) to locate the tag prefix and isolate the collection name value. Empty cells in column B were populated with the extracted values where they were missing.
