# Solution Note: Expanding Average Calculation

## Task Summary
The task required creating expanding averages in column B of sheet **Blad1**, where each cell contains the average of all values from B11 down to the current row, formatted to 3 decimal places.

## Spreadsheet Structure
- **Sheet Name**: Blad1
- **Header Row**: Row 1 contained column headers "Value" and "Average"
- **Data Column**: Column B ("Value") contained the source values starting from row 11
- **Output Range**: B12:B15 in column B ("Average")

## Python/openpyxl Approach
The solution involved reading values from column B (the "Value" column) in sheet **Blad1** starting from row 11. For each output row from B12 to B15, the logic calculated the average of an expanding range:
- B12: average of B11:B12
- B13: average of B11:B13
- B14: average of B11:B14
- B15: average of B11:B15

The implementation used `openpyxl` to:
1. Access sheet **Blad1**
2. Read numeric values from column B starting at row 11
3. For each target row (12-15), calculate the cumulative average from row 11 to the current row
4. Write the formatted result (3 decimal places) to the corresponding cell in column B
5. The averages were computed programmatically rather than using Excel formulas, ensuring the values updated based on the expanding range logic
