# Solution Note: Extract COS Rows with "Used" Keyword

## Task Summary
The task required extracting specific rows from sheet **'BZTB'** where column B descriptions started with 'COS Pass', 'COS Comm', or 'COS W/S' AND contained the word 'Used', while excluding any rows with ID '398726'. The filtered data from columns A through D was written to sheet **'Tema UV COS'** starting at cell A1.

## Implementation Approach
The solution used Python with openpyxl to:

1. **Read data from sheet 'BZTB'**: Iterated through all rows starting from row 1 (no header row present, data starts immediately at row 1)
2. **Applied filtering logic on column B**: Checked if the description text in column B started with any of the three prefixes ('COS Pass', 'COS Comm', 'COS W/S') using string matching
3. **Secondary filter**: Verified that the description also contained the word 'Used' anywhere in the text
4. **Exclusion rule**: Skipped any rows where column A contained the value '398726'
5. **Data extraction**: For matching rows, extracted values from columns A (ID), B (Description), C (numeric value), and D (numeric value)
6. **Write to destination**: Wrote the filtered results to sheet **'Tema UV COS'** in columns A through D, starting from row 1, with each matching row appended sequentially

The final output range was A1:D22 in the 'Tema UV COS' sheet.
