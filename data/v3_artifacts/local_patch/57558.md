# Solution Note: Multi-Criteria Commission Rate Lookup with Date Range

The task required populating the "Commission Rate" column in the **Deposits** sheet by matching multiple criteria against the **RateHurdles** sheet, including a date range check.

## Data Structure

**RateHurdles sheet:**
- Headers in Row 1: 'Start Date' (Column A), 'End Date' (Column B), 'Type' (Column C), 'LeadsSalesPerson' (Column D), 'Rate' (Column E)
- Data started in Row 2

**Deposits sheet:**
- Headers in Row 1: 'Commission Rate' (Column A), 'Type' (Column B), 'LeadsSalesPerson' (Column C), 'PODateReceived' (Column D)
- Data to process started in Row 2 (A2:A3 were the target cells)

## Python/openpyxl Approach

The solution read all data from the **RateHurdles** sheet (rows 2 onwards) into memory. For each row in the **Deposits** sheet starting from Row 2:

1. Read the 'Type' value from Column B
2. Read the 'LeadsSalesPerson' value from Column C
3. Read the 'PODateReceived' date from Column D

The logic iterated through the RateHurdles data to find a matching row where:
- 'Type' matched exactly
- 'LeadsSalesPerson' matched exactly
- 'PODateReceived' fell between 'Start Date' and 'End Date' (inclusive)

When a match was found, the corresponding 'Rate' value from Column E of RateHurdles was written to Column A of the Deposits sheet at the current row.
