# Solution Note: Moving Dynamic '0-15' Column to Column B

## Task Summary
The task required locating and moving a column labeled '0-15' from its variable position in **Sheet1** to column B. The '0-15' column position changes daily, making fixed column references unreliable.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Header Row**: Row 5 contained the column headers including 'Row Labels' in column A and various age bucket labels ('16', '19', '21', '23', '24', '26', '29', '0-15', '31-40', '41-50', 'Grand Total')
- **Data Rows**: Started from row 6 onwards with segment labels ('Seg - 1', etc.) in column A
- **Target Location**: Column B, rows 6-10 (B6:B10)

## Python/openpyxl Approach
1. **Located the '0-15' column**: Iterated through row 5 cells to find the column index where the cell value equals '0-15'
2. **Read the source data**: Extracted values from the identified '0-15' column for rows 6-10
3. **Inserted new column**: Used `worksheet.insert_cols(2)` to insert a new column at position B, shifting existing columns right
4. **Wrote header and data**: Set cell B5 to '0-15' and populated cells B6:B10 with the extracted values
5. **Deleted original column**: Removed the old '0-15' column from its original position (now shifted right by one position)

This dynamic approach successfully handled the variable column position by searching for the header value rather than using fixed column references.
