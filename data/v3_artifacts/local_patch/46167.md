# Solution Note: SUMIF for Product ID Matching

## Task Summary
The task required calculating the sum of quantities for rows matching a specific PRODUCT ID value (111111) and placing the result in cell D2 of Sheet1.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Header Row**: Row 1 contained column headers: 'PRODUCT ID', 'QTY', '', 'answer'
- **Data Range**: Data started from Row 2 and continued through Row 6
- **Input Columns**: 
  - Column A: 'PRODUCT ID' 
  - Column B: 'QTY'
- **Output Location**: Cell D2 (under the 'answer' column header)

## Python/openpyxl Approach
The solution was implemented by:
1. Loading the workbook and accessing the worksheet named 'Sheet1'
2. Reading the 'PRODUCT ID' values from column A (cells A2:A6)
3. Reading the corresponding 'QTY' values from column B (cells B2:B6)
4. Iterating through each row starting from row 2 to row 6
5. Checking if the PRODUCT ID in column A matched the target value (111111)
6. Summing the QTY values from column B where the condition was met
7. Writing the calculated sum directly to cell D2

This approach replicated the functionality of Excel's SUMIF formula using Python logic to conditionally sum values based on the PRODUCT ID criterion.
