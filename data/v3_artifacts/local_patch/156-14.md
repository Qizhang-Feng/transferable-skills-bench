# Solution Note: Delete Rows with "to be processed" Status

## Task Summary
The task required deleting rows from row 6 onward where column F (Status) contains "to be processed" on three sheets: "99250", "99251", and "99252". The original VBA code failed because `Sheets(Array(...))` doesn't support direct method calls on multiple sheets simultaneously.

## Spreadsheet Structure
- **Sheets**: "99250", "99251", "99252"
- **Header Row**: Row 5 containing ['Ref', 'Invoice', 'Type', 'Amount', 'Transaction Number', 'Status']
- **Data Start Row**: Row 6
- **Target Column**: Column F (Status)

## Python/openpyxl Approach
The solution iterated through each sheet individually rather than attempting to process them as an array. For each sheet ("99250", "99251", "99252"):

1. Loaded the workbook and accessed each sheet by name
2. Read column F (Status) starting from row 6 onward
3. Identified rows where the Status column contained the text "to be processed"
4. Deleted these rows by iterating backwards (from bottom to top) to avoid index shifting issues
5. Saved the workbook after processing all three sheets

The key was processing sheets in a loop rather than as a collection, and deleting rows in reverse order to maintain correct row indices during deletion. The header row (row 5) remained intact throughout the operation.
