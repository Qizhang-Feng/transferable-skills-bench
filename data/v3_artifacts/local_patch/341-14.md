# Solution Note: Transposing Multi-line Cell Data

## Task Summary
The data was in sheet **'problem'** with category labels in column A (rows 1-6) and corresponding multi-line numeric values in column B. Each cell in column B contained multiple numbers separated by newline characters (`\n`).

## Approach Used
The solution read data from sheet **'problem'**, columns A and B, starting from row 1 (no header row). For each row in the problem sheet:

1. The category label was extracted from column A
2. The corresponding value in column B was split by newline characters (`\n`) to separate individual numbers
3. For each number found, a new row was created in the result with the category label and the number
4. If column B was empty or contained only whitespace (like row 5 with 'E'), a single row was created with the category label and an empty value

## Output Location
The transformed data was written to sheet **'result'** starting at cell **A1** (range A1:B28). Each row in the result contained:
- Column A: The category label (repeated for each number)
- Column B: A single number value (or empty if no numbers existed)

The solution used Python with openpyxl to iterate through the problem data, split multi-line values, and write the expanded rows to the result sheet in-place, clearing any existing content first.
