# Solution Note: Extract Text Within Parentheses in Column B

## Task Summary
The task required extracting only the text within parentheses from cells in column B of Sheet1, leaving cells without parentheses unchanged and preserving blank cells.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Data Range**: Column B, rows 3-26 (B3:B26)
- **Header Position**: Row 2 was empty; data started at row 3
- **Column A**: Contained row numbers (1, 2, 3, 4...)
- **Column B**: Contained text strings with parentheses like "(tig) this is good"

## Python/openpyxl Approach
The solution read data from column B (index 2) in Sheet1, starting from row 3. For each cell in column B:

1. **Blank cells**: Were left blank (no value written, avoiding "0" or empty strings)
2. **Cells with parentheses**: Applied regex pattern `\(([^)]+)\)` to extract only the text between the first opening and closing parentheses
3. **Cells without parentheses**: Were left unchanged with their original value

The logic iterated through rows 3-26, checking each cell in column B. Results were written back to the same cells in column B. The implementation used `re.search()` to find parenthetical content and extracted group(1) when a match was found. Blank cells were explicitly skipped to maintain their empty state.
