# Solution Note: Delete Duplicate Words Based on Lookup Column

The task required removing entire rows from columns A through E in Sheet1 where words in column A matched any word found in column I.

**Data Structure:**
- Sheet name: `Sheet1`
- Primary data: Columns A through E (A1:E1102)
- Column A contained words to check for duplicates
- Columns B, C, D, E contained associated details (some empty)
- Column I contained the lookup list of words to match against
- Data started at row 1 (no separate header row)

**Python/openpyxl Approach:**

1. Read all values from column I (I1 downward) into a set for efficient lookup
2. Iterate through column A starting from row 1
3. For each word in column A, check if it exists in the column I lookup set
4. If a match was found, clear cells A through E in that row (set to empty string or None)
5. After processing all rows, shift cells up by iterating from top to bottom and moving non-empty rows upward
6. Alternatively, collect non-matching rows into a new list and write them back starting from A1

The solution preserved all rows where column A words did NOT appear in column I, maintaining the data in columns B through E intact. The final range A1:E1102 contained only the filtered results with duplicates removed.
