# Tax Recoupment and Scrapping Allowance Calculation Solution Note

The task was completed on sheet **Sheet1** with data starting in row 3 (header in rows 1-2).

## Data Sources
The following columns were read:
- **Column C** ('Cost'): Original asset cost
- **Column E** ('ITV'): Investment Tax Value
- **Column F** ('Profit'): Calculated profit/loss on sale
- **Column G** ('Proceeds'): Sale proceeds

## Logic Applied
The formula calculated tax recoupment based on three conditions:

1. **Loss scenario** (Profit < 0): Recoupment = ITV + Profit (effectively reducing ITV by the loss amount)
2. **Profit less than original cost** (0 ≤ Profit < Cost): Recoupment = Profit (full profit recouped)
3. **Profit exceeds or equals cost**: Recoupment = Cost - ITV (cost basis reduced by ITV)

## Implementation
Using Python with openpyxl:
- Read values from columns C, E, and F for rows 3-6
- Applied conditional logic to determine the appropriate recoupment calculation
- Wrote results to **column H** (cells H3:H6)
- Preserved existing cell formatting in column I by not modifying those cells

The solution handled all four asset records (Caterpillar, Utility truck, Computer equipment, and Compressor) with their respective purchase years and financial data.
