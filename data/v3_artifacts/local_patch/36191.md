# Solution Note: Extracting First and Last Name from Employee Data

The task required extracting both first name and surname from employee data in column **'Employee'** (column B) on sheet **'Sheet1'**, while omitting email addresses. The results were written to column **'Result Should Be'** (column C).

## Spreadsheet Structure
- **Sheet name**: Sheet1
- **Header row**: Row 1
- **Data rows**: Rows 2-3
- **Source column**: 'Employee' (column B, cells B2:B3)
- **Target column**: 'Result Should Be' (column C, cells C2:C3)

## Python/openpyxl Approach
The solution read data from column B starting at row 2. Each cell in column B contained a string with the format "FirstName LastName email@domain.com". 

The logic applied was to extract everything before the email address by:
1. Finding the position of the last space character in the string (which separates the surname from the email)
2. Extracting the substring from the beginning up to (but not including) that last space
3. This captured both the first name and surname while excluding the email address

The extracted values were written to the corresponding cells in column C (C2:C3). The formula approach used `FIND()` to locate the first space, but the correct solution required finding the **last** space before the email address, which was accomplished using string manipulation methods like `rfind()` or `rsplit()` in Python.
