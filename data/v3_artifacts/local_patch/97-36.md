# Solution Note: Delete Rows Above Substring in Excel

## Task Summary
The task required creating a VBA solution to delete all rows above a specified substring ('domestic') in an Excel sheet, searching from the first row and first column.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Data Column**: Column A (URLs)
- **Data Range**: A1:A22
- **Header Row**: None (data starts at row 1)

## Approach
The data was located in Sheet1, column A, containing URLs starting from row 1. The task required searching through column A to find the first occurrence of the substring 'domestic' within any cell value.

The VBA logic involved:
1. Loading the workbook and accessing Sheet1
2. Iterating through column A cells from row 1 downward
3. Using the `InStr` function to check if 'domestic' appeared anywhere in each cell's text
4. Identifying the first row containing 'domestic'
5. Deleting all rows from row 1 up to (but not including) the row containing 'domestic'
6. Using the `Rows` collection with the `Delete` method to remove the specified range

The solution used a loop to scan each cell in column A sequentially, stopping at the first match. Once found, the `Range.Delete` method was applied to remove rows 1 through (match_row - 1), preserving the row containing 'domestic' and all rows below it.
