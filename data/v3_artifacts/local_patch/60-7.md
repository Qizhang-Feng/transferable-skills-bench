# Solution Note: Consolidated Tracker Update

## Task Summary
The task required combining data from three source sheets to populate the "Consolidated Tracker" sheet in the range A3:E11.

## Data Sources
- **Existing Task** sheet: Data rows 3-6 (after header in row 2) containing Category, Activity Name, AU, Frequency, and Team Member in columns A-E
- **Additions** sheet: Data rows 3-6 (after header in row 2) with the same five columns A-E
- **Retired** sheet: Data rows 3+ (after header in row 2) with matching column structure

## Approach
The solution used Python with openpyxl to:

1. Read all data rows from "Existing Task" sheet starting at row 3, columns A through E
2. Read all data rows from "Additions" sheet starting at row 3, columns A through E
3. Read all data rows from "Retired" sheet starting at row 3, columns A through E
4. Combine "Existing Task" and "Additions" rows into a single list
5. Filter out any rows from the combined list that matched entries in "Retired" (comparing all five columns: Category, Activity Name, AU, Frequency, Team Member)
6. Write the resulting net list to "Consolidated Tracker" sheet starting at cell A3

The operation effectively calculated: Consolidated Tracker = Existing Task + Additions - Retired, where subtraction meant removing matching rows based on all column values.
