# Solution Note: School Data Consolidation with INDEX/MATCH

## Problem Summary
The task involved fixing a failing INDEX/MATCH formula that was attempting to look up DFES numbers from a 'URN lookup' sheet by matching values in column L of the main dataset.

## Spreadsheet Structure
- **Main Sheet**: 'Compiled and located schools da'
- **Header Row**: Row 1
- **Data Start Row**: Row 2
- **Key Columns**:
  - Column A: 'URN'
  - Column B: 'DFES check' (target output column)
  - Column C: 'DfE Number'
  - Column L: Not visible in sample but referenced as lookup value source

## Python/openpyxl Approach

The solution required reading data from the 'URN lookup' sheet:
- Column D (range D2:D1461) contained the DFES numbers to be returned
- Column K (range K2:K1461) contained the lookup keys to match against

The process involved:
1. Loading both sheets using openpyxl
2. Reading column L values from 'Compiled and located schools da' sheet (rows 2-1461)
3. Reading the lookup table from 'URN lookup' sheet: column K as keys, column D as values
4. For each row in the main sheet, matching the column L value against 'URN lookup' column K
5. Writing the corresponding column D value to column B ('DFES check') in the main sheet
6. Handling #N/A cases where no match was found

This replaced the failing Excel formula with a programmatic lookup operation.
