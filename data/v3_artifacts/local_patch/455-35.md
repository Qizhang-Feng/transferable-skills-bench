# Solution Note: Filtering Rows by First Character in Column B

The task required filtering rows in **Worksheet1** where the first character in **column B (User)** was not 'H' or 'A'. The data started at **row 2** (row 1 contained headers: Action, User, Date, Client, Packslip, PO Number, Quantity, Packsize, Extended Product Code, Bin Label).

## Approach Used

The solution used Python with openpyxl to:

1. **Read the data**: Loaded the workbook and accessed **Worksheet1**
2. **Identify rows to keep**: Iterated through rows starting from **row 2** through the last row with data
3. **Apply filtering logic**: For each row, checked if the first character of the cell in **column B (User)** started with 'H' or 'A' using string indexing or the `startswith()` method
4. **Delete rows in reverse**: Collected row indices where column B did NOT start with 'H' or 'A', then deleted these rows by iterating in reverse order (from bottom to top) to avoid index shifting issues
5. **Write results**: The filtered data remained in **Worksheet1** with the same structure, maintaining the header row at row 1

The key difference from the VBA approach was iterating backwards when deleting rows to prevent index misalignment. The solution preserved all rows where column B started with 'H' or 'A' and removed all others.
