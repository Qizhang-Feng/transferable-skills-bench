# Solution Note: Extracting Unticked Branches

The task required identifying branches from **Sheet1** that did not have a tick mark in the "checked" column and listing them starting at cell **A13**.

## Data Structure
- **Header row**: Row 1
- **Data range**: Rows 2-6
- **Branch names**: Column A ("Branches")
- **Check marks**: Column I ("checked")
- **Output location**: Starting at A13

## Python/openpyxl Approach

The solution read data from **Sheet1** starting at row 2 (first data row after headers). For each row in the range A2:I6:

1. Read the branch name from **column A**
2. Read the corresponding value from **column I** (the "checked" column)
3. Applied logic to check if column I was empty or did not contain a tick mark ("ü")
4. If the condition was true (no tick mark present), the branch name was added to a list of unticked branches

The filtered list of unticked branches was then written to **column A** starting at **cell A13** (rows 13-14 in this case).

The key logic was identifying rows where column I was either empty or did not equal "ü", then extracting the corresponding branch name from column A and writing it sequentially to the output range.
