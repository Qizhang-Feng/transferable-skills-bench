# Solution Note: If UNLESS Conditional Logic for Department Assignment

The data was located in **Sheet1** with headers in row 1. Column A contained "Name" (department values: 'Sales' or 'Sales-Marketing'), Column B contained "Sub-Dept" (either empty or 'General'), and Column C "Excel Formula" was the target output column.

The data rows started at row 2 and extended through row 6.

## Logic Applied:

The conditional logic implemented was:
- If Column A = 'Sales' AND Column B is empty → output 'Sales'
- If Column A = 'Sales-Marketing' AND Column B is empty → output 'Marketing'
- If Column A = 'Sales' AND Column B = 'General' → output 'Sales'
- If Column A = 'Sales-Marketing' AND Column B = 'General' → output 'Marketing'

## Python/openpyxl Approach:

1. Read values from Column A (Name) starting at A2
2. Read values from Column B (Sub-Dept) starting at B2
3. For each row, apply the conditional logic:
   - Check if Column B is empty or None
   - If empty: output 'Sales' for 'Sales', 'Marketing' for 'Sales-Marketing'
   - If 'General': output based on Column A value ('Sales' → 'Sales', 'Sales-Marketing' → 'Marketing')
4. Write the results to Column C (Excel Formula) starting at C2

The formula essentially treated Column B as an override condition, where the presence of 'General' maintained certain mappings while empty values triggered department-specific assignments.
