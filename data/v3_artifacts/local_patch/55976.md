# Solution Note: Survey Response Tallying

The task involved counting 'Yes' responses (marked with 'x') from an 80-question survey in **Sheet1** and categorizing them into four columns.

## Spreadsheet Structure
- **Header row**: Row 4 contained column headers including 'Number', 'Statement', 'Yes', 'No'
- **Data started**: Row 5 onwards (questions 1-80)
- **Yes column**: Column C (index 2) contained 'x' marks for affirmative responses
- **Output location**: Cells F25:I25 for the four category totals

## Python/openpyxl Approach

The solution read data from **Sheet1** starting at row 5 through row 84 (80 questions total). For each row, the script checked **column C** ('Yes' column) for the presence of an 'x' mark.

Each question was pre-assigned to one of four categories (columns F, G, H, or I). The logic involved:
1. Iterating through rows 5-84
2. Checking if column C contained 'x'
3. Mapping each question number to its corresponding category column based on the survey design
4. Incrementing the appropriate category counter when an 'x' was found
5. Writing the four totals to cells **F25, G25, H25, and I25**

The categorization mapping determined which questions contributed to which of the four summary columns, ensuring each 'Yes' response was tallied in its designated category total.
