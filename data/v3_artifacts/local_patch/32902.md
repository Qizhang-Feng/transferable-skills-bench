# Bonus Calculation Solution Note

The task involved calculating tiered bonuses in **Sheet1** based on income levels. The data structure had headers in **row 5** with actual data starting in **row 6**.

## Data Structure
- **Column A (Quarter)**: Quarter identifiers
- **Column B (Income)**: Income values to calculate bonuses from
- **Columns C, D, E (Tier 1, Tier 2, Tier 3)**: Individual tier bonus calculations
- **Column F (Total)**: Sum of all tier bonuses

## Calculation Logic Applied

The bonus tiers were calculated as follows:

**Tier 1 (Column C)**: For income over $60,000 up to $90,000, calculated as `(min(income, 90000) - 60000) * 0.15` if income exceeded $60,000, otherwise 0.

**Tier 2 (Column D)**: For income over $90,000 up to $120,000, calculated as `(min(income, 120000) - 90000) * 0.25` if income exceeded $90,000, otherwise 0.

**Tier 3 (Column E)**: For income over $120,000, calculated as `(income - 120000) * 0.35` if income exceeded $120,000, otherwise 0.

## Implementation Approach

Using Python/openpyxl, the solution read income values from **column B** starting at **row 6**, applied the tiered calculation logic with appropriate conditional checks and min/max functions, then wrote the calculated tier bonuses to **columns C, D, and E** for each data row. The formulas ensured that each tier only calculated bonuses for its specific income range.
