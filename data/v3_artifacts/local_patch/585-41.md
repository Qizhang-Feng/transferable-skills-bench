# Solution Note: Remove Non-Numeric Characters While Preserving Decimals

The task was to clean cell values in the range **'A1:H1'** on sheet **'Sheet1'** by removing all non-numeric characters while preserving numeric digits and decimal points.

## Approach

The data was located in **Sheet1**, specifically in **row 1, columns A through H** (A1:H1). There were no headers in this case, as the data to be cleaned was directly in row 1.

The Python/openpyxl solution involved:

1. **Loading the workbook** and accessing sheet **'Sheet1'**
2. **Reading each cell** in the range A1 through H1
3. **Applying a filtering logic** that:
   - Converted each cell value to a string
   - Iterated through each character in the string
   - Kept only characters that were either digits (0-9) or decimal points (.)
   - Joined the filtered characters back into a cleaned string
4. **Writing the cleaned values** back to the same cells (A1:H1) in place

The key difference from a script that removes decimal points was using a character filter that explicitly checked if each character was either `char.isdigit()` OR `char == '.'`, rather than only checking for digits. This preserved decimal points while removing currency symbols, letters, and special characters.

The results were written back to the original range **A1:H1** on **Sheet1**.
