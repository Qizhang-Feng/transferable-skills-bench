# Solution Note: Multi-Sheet Balance Calculation and Collection

The task required consolidating data from four source sheets (STA, RPA, SR, SS) into the COLLECTION sheet with calculated BALANCE values.

**Source Data Structure:**
- Sheets STA and RPA contained columns: ITEM, BR, TY, OR, SALE, RET (columns A-F, headers in row 1, data starting row 2)
- Sheet SR contained columns: ITEM, BR, TY, OR, SALE (columns A-E, no RET column)
- Sheet SS contained columns: ITEM, BR, TY, OR, RET (columns A-E, no SALE column)

**Logic Applied:**
1. Read all rows from each source sheet starting at row 2
2. For each row, extracted values from columns: ITEM (A), BR (B), TY (C), OR (D), SALE (E if present), RET (F if present, or E in SS sheet)
3. Calculated BALANCE = SALE - RET, treating missing values ("-" or absent columns) as 0
4. Grouped results by the combination of BR, TY, and OR
5. Summed SALE, RET, and BALANCE values for matching groups across all sheets

**Output Structure:**
The COLLECTION sheet was populated starting at A1 with headers: ITEM, BR, TY, OR, SALE, RET, BALANCE (columns A-G). Consolidated data was written starting at row 2 (A2:G9), with each unique BR-TY-OR combination appearing once with aggregated totals.
