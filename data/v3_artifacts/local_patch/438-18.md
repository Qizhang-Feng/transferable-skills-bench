# Solution Note: Country Name Replacement from INPUT to OUTPUT Sheet

## Task Summary
The data was located in sheet **INPUT** with headers in row 1 and data starting from row 2. The task required copying selected columns from INPUT to sheet **OUPUT** while replacing full country names in column F with abbreviated codes.

## Approach Used

**Source Data (INPUT sheet):**
- Column A: 'ITEM'
- Column C: 'BRANDS'
- Column E: 'TYPE'
- Column F: 'ORIGIN' (containing full country names)

**Target Location (OUPUT sheet):**
- Range A1:D6
- Column A: 'ITEM'
- Column B: 'BRANDS'
- Column C: 'TYPE'
- Column D: 'ORIGIN' (with abbreviated country codes)

**Implementation Logic:**
1. Read data from INPUT sheet starting at row 2 (after headers)
2. Extract values from columns A, C, E, and F for each row
3. Applied a mapping dictionary to replace country names:
   - 'UNATIDE STATES AMERICAN' → 'USA'
   - 'FRANCE' → 'FRA'
   - 'KOREA' → 'KOR'
   - 'GERMANY' → 'GR'
   - 'ITALY' → 'IT'
4. Wrote the header row to OUPUT!A1:D1
5. Wrote transformed data rows to OUPUT!A2:D6

The solution used Python with openpyxl to iterate through INPUT rows 2-6, applied the country code mapping to column F values, and wrote the results to the corresponding cells in OUPUT sheet maintaining the structure with only the four required columns.
