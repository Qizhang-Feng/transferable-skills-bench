# Solution Note: Product Sum by Category

The task required writing a formula in cell C2 of Sheet1 that sums values from Sheet2 based on a matching criterion.

**Data Structure:**
- Sheet1 contained product categories in column A (starting at row 2), with "Fully linked" in cell A2
- Sheet2 had product categories in column A (rows 2-363) and corresponding UK purchase values in column R (rows 2-363)
- Sheet1 row 1 contained headers; data started at row 2
- Sheet2 row 1 contained headers; data started at row 2

**Python/openpyxl Approach:**
1. Read the lookup value from Sheet1, cell A2 (the product category to match)
2. Read all values from Sheet2, column A, range A2:A363 (product categories)
3. Read all values from Sheet2, column R, range R2:R363 (UK purchases)
4. Iterate through the data rows, comparing each value in column A of Sheet2 with the lookup value from Sheet1 cell A2
5. Sum all corresponding values from column R of Sheet2 where the column A value matched
6. Write the summed result to Sheet1, cell C2

**Formula Equivalent:**
The Excel formula equivalent would be a SUMIF function: `=SUMIF(Sheet2!A2:A363, A2, Sheet2!R2:R363)` which sums column R values where column A matches the criterion in A2.
