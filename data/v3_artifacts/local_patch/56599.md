# Solution Note: Finding Book Number by Slip Number

The task required finding a Book Number associated with a given slip number in cell M5, with the result written to cell B2 on Sheet1.

## Data Structure

The spreadsheet contained a lookup table starting at row 5 in Sheet1:
- **Column G**: "Slip fm" (starting slip number of each book)
- **Column H**: "slip to" (ending slip number of each book)
- **Column I**: "total slips in book" (always 10 slips per book)
- **Column J**: "BookNo" (the book identifier to return)

The header row was in row 5, and data started from row 6 onwards.

## Solution Approach

The Python/openpyxl solution involved:
1. Reading the lookup slip number from cell M5 in Sheet1
2. Iterating through rows starting from row 6 in Sheet1
3. For each row, reading values from columns G (slip from) and H (slip to)
4. Checking if the lookup slip number fell within the range (inclusive) between columns G and H
5. If a match was found, reading the corresponding BookNo from column J
6. Writing the BookNo result to cell B2
7. If no match was found or M5 was blank, writing "not found - please re-enter" to cell B2

The solution handled seven-digit slip numbers and validated that the input was numeric and within valid ranges.
