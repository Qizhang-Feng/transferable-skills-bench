# Solution Note: Balancing Residuals Across Unit Columns

The data was located in **Sheet1** with headers in **row 1** and data starting from **row 2** through **row 6**.

The unit columns were:
- **Column B (unit1)** through **Column F (unit5)**
- **Column I (residuals)** contained the values to be balanced

The approach implemented:

1. **Read the data**: Extracted values from columns B through F (unit columns) and column I (residuals) for rows 2-6.

2. **Balance logic**:
   - For **positive residuals**: Subtracted from unit columns in order from **unit5 to unit1** (right to left, columns F→B)
   - For **negative residuals**: Added to unit columns in order from **unit1 to unit5** (left to right, columns B→F)
   - Skipped any unit cells that contained zero values (preserved zeros)
   - Applied constraints: minimum value of 0 and maximum value equal to the original cell value for each unit column

3. **Distribution algorithm**: For each row, distributed the residual value across available unit columns while respecting the constraints, adjusting each unit cell incrementally until the residual was fully balanced.

4. **Write results**: Updated the modified unit values back to columns B-F in their original positions, then set all **column I values to 0**.

The solution preserved the original data structure and locations while balancing the residuals according to the specified directional rules and constraints.
