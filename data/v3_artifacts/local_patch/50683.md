# Solution Note: Golf Player Last 4 Rounds Average

The data was located in **Sheet1** with headers in **Row 1**. Player identifiers were in column A (labeled 'Golfer'), and tournament scores were in columns B through N (dates from '8/1/2021' through '10/24/2021'). Column P contained season averages (labeled 'Average'), and the task required calculating the last 4 rounds average in **column Q (Q2:Q5)** labeled 'last 4 trend'.

The data rows started at **Row 2** and extended through **Row 5** for four players.

The approach involved:
1. Reading each player's row from columns B through N (tournament date columns)
2. Filtering out blank/empty cells to get only actual scores played
3. Converting valid score values to floats
4. Taking the last 4 scores from this filtered list (using Python list slicing `[-4:]`)
5. Calculating the average of these last 4 scores
6. Writing the computed average to the corresponding cell in column Q (Q2 for player 1, Q3 for player 2, etc.)

The key challenge was handling inconsistent participation where blank cells appeared throughout the score columns. The solution required iterating through each score cell, checking for non-empty values, collecting them in order, then selecting only the most recent four scores before averaging. This ensured that regardless of participation gaps, each player's last four actual tournament scores were correctly identified and averaged.
