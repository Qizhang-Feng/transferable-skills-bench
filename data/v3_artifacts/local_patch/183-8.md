# Solution Note: Weighted Average Calculation

The task required calculating weighted averages for columns J, K, and L in **Sheet1**, rows 3-6, based on generation data.

**Data Structure:**
- Header row 1 contained original OEM data columns A-F
- Header row 2 contained the summary table headers in columns H-L
- Column I (FY) listed fiscal years: 2021-2022, 2020-2021, 2019-2020, 2018-2019 in rows 3-6
- Column J contained "Sum of Gen. (kwh) DAY" values (the weights)
- Columns J, K, L needed weighted average formulas for "%PLF DAY", "M/C Avail.%", and "Grid Avail.%" respectively

**Approach:**
The solution used SUMPRODUCT formulas to calculate weighted averages. For each target cell (J3:L6):
1. Read the corresponding fiscal year from column I (current row)
2. Match this FY against column B (rows 3-6) in the source data
3. Read weights from column C ("Sum of Gen. (kwh) DAY")
4. Read values from columns D, E, or F depending on the target column (J→D, K→E, L→F)
5. Apply SUMPRODUCT to multiply matched values by their weights, divided by the sum of matched weights

The formula pattern was: `=SUMPRODUCT((condition)*(weights)*(values))/SUMPRODUCT((condition)*(weights))`

This approach ensured each metric was properly weighted by generation volume for the corresponding fiscal year.
