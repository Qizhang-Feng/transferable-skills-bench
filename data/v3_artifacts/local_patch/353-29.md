# Solution Note: Auto-Generating Lettered URNs for Blank Cells

## Task Summary
The data was located in sheet **'Sheet1'** with headers in **row 1**. Column **A** contained URN values (Unique Reference Numbers), and column **B** contained Date & Time data. Data rows started from **row 2** onwards.

## Approach Used
The solution used Python with the openpyxl library to:

1. **Read the existing data**: Loaded the workbook and accessed sheet **'Sheet1'**, then iterated through column **A** starting from **row 2** (first data row after headers).

2. **Track the last numbered URN**: Maintained a variable to store the last numeric URN encountered while scanning column **A**. This served as the base number for subsequent lettered URNs.

3. **Generate lettered suffixes**: For each blank cell in column **A**, calculated a letter suffix (a, b, c, etc.) based on how many consecutive blank cells followed the last numbered URN. The counter reset whenever a new numbered URN was encountered.

4. **Write new URNs**: Populated blank cells in column **A** with the format `{last_number}{letter}` (e.g., '5a', '5b', '5c').

5. **Preserve existing data**: Left all existing numeric URNs and column **B** data unchanged.

The solution processed the entire dataset systematically, ensuring that blank rows received appropriate lettered URNs based on their position relative to the preceding numbered URN.
