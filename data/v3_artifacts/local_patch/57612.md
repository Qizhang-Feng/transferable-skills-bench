# Solution Note: Conditional Value Mapping Based on Range

## Task Summary
The task required creating a formula in column D (cells D4:D15) of Sheet1 that maps values from column C based on specific range conditions.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Header Row**: Row 3 contained headers "values" (column C) and "expected result" (column D)
- **Data Range**: Values were located in column C starting from row 4 (C4:C15)
- **Output Range**: Results needed to be written to column D starting from row 4 (D4:D15)

## Logic Applied
The solution implemented a nested IF statement with range conditions:
- If the value in column C was between 5 and 6 (inclusive), return 5
- If the value in column C was between 3 and 4 (inclusive), return 3.75
- If the value in column C was between 1 and 2 (inclusive), return 1
- Otherwise, return 0 (for values outside these ranges)

## Python/openpyxl Approach
The implementation read values from column C (starting at C4) in Sheet1, applied the conditional logic using nested if-elif-else statements to check range boundaries with AND conditions, and wrote the calculated results to the corresponding cells in column D (D4:D15). The workbook was then saved to preserve the changes.
