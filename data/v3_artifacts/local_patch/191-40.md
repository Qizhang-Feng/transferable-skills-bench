# Solution Note: Sorting Blocks by Column D on Sheet1

The task required sorting multiple blocks of data on **Sheet1** by column D in descending order, where blocks were separated by blank rows.

## Spreadsheet Structure
- **Sheet**: Sheet1
- **Data columns**: A through H
- **Sort key**: Column D (contains formulas like =B+C)
- **Header row**: Row 1 contains the first data entry (no separate header)
- **Data range**: Started at row 1 and extended to row 85

## Python/openpyxl Approach

The solution read all data from **Sheet1, columns A through H** starting from row 1. The logic identified blocks by detecting blank rows (where all cells in columns A-H were empty or None).

For each block:
1. Extracted rows belonging to that block
2. Sorted the rows by **column D values** in descending order using Python's `sorted()` function with `key=lambda row: row[3]` (0-indexed column D) and `reverse=True`
3. Preserved formula strings in columns D, E, and F during the sort

After sorting all blocks, the data was written back to **Sheet1** starting at cell A1, maintaining exactly one blank row between consecutive blocks. The `worksheet.delete_rows()` method cleared existing data before writing the sorted results to ensure clean output.
