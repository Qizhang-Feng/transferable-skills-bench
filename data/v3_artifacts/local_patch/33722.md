# Solution Note: Dynamic Percentile Range Median Calculation with Exclusion

The task required calculating median market capitalization for companies within the 0-1% percentile range in **Sheet1**, with each subsequent calculation excluding previously selected rows.

**Data Structure:**
- Column B ('ticker'): Company ticker symbols, starting at row 2
- Column C ('marketcap'): Market capitalization values, starting at row 2
- Column D ('percentile'): Pre-calculated percentile values, starting at row 2
- Column E ('median 0-1%'): Output column for median calculations, rows 2-6

**Python/openpyxl Approach:**

1. Read all data from columns B, C, and D starting from row 2 to identify the full dataset
2. For each output row (E2 through E6):
   - Filter rows where column D (percentile) values fall between 0 and 0.01 (0-1% range)
   - Exclude any rows that were selected in previous iterations (tracking by row indices)
   - Calculate the median of column C (marketcap) values for the filtered subset
   - Write the median value to the corresponding cell in column E
   - Track the row indices used to exclude them from subsequent calculations

3. Apply formatting to column E (rows 2-6):
   - Add borders to all cells
   - Set horizontal alignment to left
   - Format as integers (no decimals, no thousand separators)

The exclusion mechanism ensured each median calculation used a progressively smaller dataset, removing previously analyzed companies from consideration.
