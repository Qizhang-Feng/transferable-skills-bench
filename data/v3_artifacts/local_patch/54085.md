# Solution Note: Conditional Item Filtering from List

## Task Summary
The task required populating column C ('Desired Result') with items from column B ('List of Items'), excluding any item that matched the value in cell A2 of column A ('Item') in Sheet1.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Header Row**: Row 1
- **Data Range**: Rows 2-5
- **Exclusion Criteria**: Cell A2 contained the item to exclude ("Pears")
- **Source List**: Column B, cells B2:B5 contained the full list of items
- **Output Range**: Column C, cells C2:C5 were designated for results

## Python/openpyxl Approach
The solution read the exclusion item from cell A2 in column A ('Item') on Sheet1. Then it iterated through column B ('List of Items') starting from row 2 (B2:B5), comparing each value against the exclusion item. For each cell in the source range:
- If the value matched the exclusion item from A2, an empty string was written to the corresponding row in column C
- If the value did not match or was empty, the original value was written to column C

The results were written to cells C2:C5 in column C ('Desired Result'), maintaining the same row order as the source data in column B. This approach preserved the sequential order while filtering out the specified exclusion item.
