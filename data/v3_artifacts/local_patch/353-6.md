# Solution Note: Filter and Delete Rows Based on Column Criteria

## Task Summary
The task required filtering and deleting rows from the 'Data' sheet where column 'I' (communication_reason) contained the value 'Hub Notes', while preserving the header row.

## Spreadsheet Structure
- **Sheet Name**: 'Data'
- **Header Row**: Row 1 (contains column names like 'job_comm_id', 'job_id', 'communication_time', etc.)
- **Data Start Row**: Row 2
- **Target Column**: Column 'I' (communication_reason) - the 9th column in the sheet
- **Data Range**: Columns A through AH, with data starting from row 2

## Python/openpyxl Approach
The solution involved reading all rows from the 'Data' sheet starting from row 2 (after the header). For each row, column 'I' was checked to determine if it matched the filter criteria ('Hub Notes'). 

Rows that did NOT match the criteria were collected into a new list, while matching rows were excluded (effectively deleted). The header row (row 1) was preserved separately.

The filtered dataset was then written back to the 'Data' sheet, starting with the header in row 1, followed by the remaining data rows from row 2 onwards.

This approach was replicable for different filter criteria by changing the target column letter and the comparison value, making it flexible for filtering other columns in the dataset.
