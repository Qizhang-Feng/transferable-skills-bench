# Solution Note: Rolling 7-Day Points Total

## Task Summary
The task required calculating a rolling 7-day sum of points for non-consecutive dates in an Excel spreadsheet.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Header Row**: Row 3 contained column headers
- **Data Start Row**: Row 4
- **Column A** ("Reporting Date"): Contained non-consecutive dates in sequential order
- **Column B** ("Points that Day"): Contained daily point values
- **Column C** ("Rolling Last 7 Day Total"): Target column for the rolling sum formula

## Solution Approach
The solution used Python with openpyxl to:

1. Read the date values from column A (starting at A4) and point values from column B (starting at B4)
2. For each row starting at row 4, calculate the sum of points from column B where the corresponding date in column A fell within the 7-day window (inclusive of the current date and going back 6 days)
3. Iterate through each date, comparing it against all previous dates to determine which rows fell within the 7-day lookback period
4. Write the calculated rolling sum values to column C (C4 onwards)

The logic accounted for the non-consecutive nature of dates by using date arithmetic rather than fixed row offsets, ensuring accurate 7-day windows regardless of gaps in the date sequence.
