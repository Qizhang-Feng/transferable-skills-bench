# Solution Note: Consolidating Data from Multiple Sheets

The task required consolidating data from sheets **Jack**, **Henry**, and **Richard** into the **Consolidate_ALL** sheet, writing to columns A through L starting at row 2.

## Source Data Structure
Each source sheet (Jack, Henry, Richard) had headers in row 1. The yellow-highlighted columns to extract were:
- Column A: Date
- Column C: Name  
- Column D: Description
- Column E: Parent Node
- Column F: Parent Description
- Column G: CC External Code
- Column N: Available in CORE
- Column O: OAL CC Type
- Column P: CC Default for LOB
- Column Q: CC Fiscal Recharge Owner
- Column R: Ancestor List

Data rows started at row 2 in each source sheet.

## Consolidation Logic
The Python/openpyxl approach involved:
1. Iterating through each source sheet (Jack, Henry, Richard)
2. Reading the specified columns (A, C, D, E, F, G, N, O, P, Q, R) from row 2 onwards
3. Writing these 11 values to columns A-K in Consolidate_ALL
4. Adding the source sheet name to column L for each row
5. Preserving cell formatting and data types from the source sheets
6. Appending rows sequentially to the first available blank row in Consolidate_ALL

The destination range was A2:L8000, accommodating all consolidated records with their sheet name identifiers in column L.
