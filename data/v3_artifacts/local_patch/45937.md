# Solution Note: Percentage Category Lookup Formula

The task required completing column E (labeled '%') in the 'Current' sheet by creating formulas that automatically determine percentage categories based on kilometer ranges.

**Data Structure:**
- The main data table was located in columns B through E, with headers in row 6
- Column D contained 'Category' values (starting at D7)
- Column E needed to be populated with corresponding '%' values (E7:E9)
- The category legend was located in cells H17:N22, with kilometer ranges specified in H20:I22
- Another category reference existed in H6:I11

**Solution Approach:**
The task was solved using Python with openpyxl by:
1. Reading the category value from column D (cells D7:D9) in the 'Current' sheet
2. Reading the legend data from the range H17:N22 to establish the mapping between categories and percentages
3. Implementing a lookup logic that matched the category from column D against the legend
4. For kilometer-based categories, the ranges in H20:I22 were used to determine which category applied
5. Writing the resulting percentage values to column E (E7:E9)

The solution involved VLOOKUP-style logic implemented in Python, matching category identifiers to their corresponding percentage values from the legend table, then populating the target cells with the appropriate formulas or values.
