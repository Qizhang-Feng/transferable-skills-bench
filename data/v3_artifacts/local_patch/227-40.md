# Solution Note: Remove Row-wise Duplicates in Excel

The task required removing duplicate values within each row of worker data from sheet 'Sheet1'. The source data was located in cells A1:E4, with headers in row 1 ('Site Name', 'Worker1', 'Worker2', 'Worker3', 'Worker4') and data starting from row 2.

The Python/openpyxl approach involved:

1. **Reading the source data**: The entire range A1:E4 was read from sheet 'Sheet1', capturing both headers and the three data rows (New York, Los Angeles, Las Vegas sites).

2. **Processing logic**: For each data row (rows 2-4), the site name in column A was preserved. For columns B through E (Worker1-Worker4), duplicates were identified by comparing each worker name against all previous worker names in that same row. When a duplicate was found, it was replaced with an empty string.

3. **Writing the output**: The processed results (headers plus cleaned data) were written to the range A14:E17 in sheet 'Sheet1'. Row 14 contained the headers, and rows 15-17 contained the deduplicated worker data.

The solution maintained the original structure while ensuring each worker name appeared only once per row, reading from left to right. Empty strings replaced subsequent occurrences of duplicate names within the same row.
