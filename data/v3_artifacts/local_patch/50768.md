# Solution Note: Risk Assessment Matrix Auto-Population

## Task Summary
The task required automatically plotting Risk IDs from a Risk Assessment table into a 3x3 matrix based on their Impact and Likelihood values, with multiple IDs in the same cell separated by commas.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Source Data Location**: Columns A-C, starting from row 3 (header row)
- **Column Names**: 
  - Column A: 'ID'
  - Column B: 'Impact'
  - Column C: 'Liklihood' (as spelled in the sheet)
- **Data Rows**: Started at row 4 and continued downward
- **Target Matrix Location**: Cells F12:H14 (3x3 grid)

## Solution Approach
The solution used Python with openpyxl to:

1. Read the Risk Assessment data from columns A, B, and C starting at row 4
2. Parse each row to extract the ID (column A), Impact value (column B), and Likelihood value (column C)
3. Group Risk IDs by their Impact-Likelihood combinations using a dictionary structure
4. Map the Impact values (1-3) to matrix rows (14, 13, 12 from bottom to top) and Likelihood values (1-3) to matrix columns (F, G, H from left to right)
5. Write the grouped Risk IDs into the corresponding matrix cells at F12:H14, joining multiple IDs with commas
6. The matrix was oriented with Impact increasing upward and Likelihood increasing rightward
