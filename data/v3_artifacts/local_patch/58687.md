# Solution Note: Apple Index Assignment

The task involved automatically assigning index numbers (1-92) to apples across months in **Sheet1**.

## Data Structure

**Table 1 (Horizontal):**
- Row 4: Month numbers (1-20) in columns C through V
- Row 5: "Apples sold" per month in columns C through V
- Row 6: "Apples Accum" (cumulative totals) in columns C through V

**Table 2 (Vertical):**
- Column A: "Apple No." (index 1-92)
- Column B: "Month No." (B9:B100) - the target output column

## Solution Approach

The solution read the horizontal data from **Row 5** (columns C through V) to get the count of apples sold per month. 

For each apple index (1-92), the logic determined which month it belonged to by:
1. Iterating through each month's apple count
2. Maintaining a running cumulative total
3. When the cumulative total reached or exceeded the current apple index, that month number was assigned

The formula logic essentially reversed the cumulative sum: for each apple index in column A (rows 9-100), it found which month range the index fell into based on the "Apples sold" values.

The results were written to **column B, rows 9 through 100** (B9:B100), matching the 92 total apples with appropriate month numbers based on the distribution shown in Row 5.
