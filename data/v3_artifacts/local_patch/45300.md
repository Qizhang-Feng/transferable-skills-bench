# Solution Note: Removing Text from 'FT' Onwards in Excel Strings

## Task Summary
The data was located in **Sheet1**, with source strings in **column A** (rows 1-4). The task required removing all text starting from the word 'FT' (inclusive) to the end of each string, while preserving the text before 'FT'. Results were written to **column B** (B1:B4).

## Approach Used
The solution used Python with the openpyxl library to:

1. **Load the workbook** and access the worksheet named **Sheet1**
2. **Read data** from **column A, rows 1-4** (no header row; data started at row 1)
3. **Apply string processing logic**: For each cell value in column A:
   - Searched for the substring 'FT' within the string
   - If 'FT' was found, extracted only the portion of text before 'FT' using string slicing
   - If 'FT' was not found, kept the original string unchanged
   - Stripped any trailing whitespace from the result
4. **Write results** to **column B, rows 1-4** (B1:B4), aligning each processed string with its corresponding source row
5. **Save the workbook** to persist the changes

The solution handled all four rows containing transfer transaction strings, successfully removing the FT reference codes and subsequent data while preserving the transfer numbers and names that preceded them.
