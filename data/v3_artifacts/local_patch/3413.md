# Solution Note: Conditional Department/RU Sum Formula

## Task Summary
The task required creating formulas in cells G3:G6 on Sheet1 that conditionally sum values from column C based on matching criteria in columns A-B against columns E-F.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Headers**: Row 2 contained headers: 'Dept' (A), 'RU' (B), 'Total' (C), 'Dept' (E), 'RU' (F), 'Total' (G)
- **Source Data**: Rows 3-6 in columns A:C contained department names, regional units, and totals
- **Lookup Criteria**: Rows 3-6 in columns E:F contained department and RU values to match against
- **Output Range**: G3:G6 required formulas

## Logic Applied
The formula logic implemented a two-tier conditional sum:
1. **Primary condition**: Sum values from column C where column A matched the department in column E of the same row AND column B matched the RU in column F of the same row
2. **Fallback condition**: If no exact Dept+RU match existed, sum all values from column C where only column A matched the department in column E (ignoring RU)

## Python/openpyxl Approach
The solution read data from columns A, B, and C (rows 3-6) as the source dataset. For each row in the output range (G3:G6), it extracted the lookup criteria from columns E and F of that row, applied the conditional summing logic against the source data, and wrote the calculated result to the corresponding cell in column G.
