# Solution Note: Currency Note Distribution Based on Salary

The task involved distributing 120 different currency notes among employees based on their net salaries in the **denomination** sheet.

## Data Structure
- **Input columns**: 
  - Column A: CompanyID
  - Column B: EmpCode
  - Column C: EmpName
  - Column D: Designation
  - Column E: NetSalary
- **Output columns**: H through P (denominations: 1000, 500, 200, 100, 50, 20, 10, 5, 1)
- **Header row**: Row 1
- **Data rows**: Rows 2-6 (5 employees)
- **Answer range**: H2:P8

## Solution Approach
The solution read NetSalary values from column E (rows 2-6) and calculated the optimal distribution of currency notes for each employee. The algorithm worked by:

1. Reading each employee's NetSalary from column E
2. Applying a greedy algorithm to break down each salary into available denominations (starting from highest: 1000, 500, 200, 100, 50, 20, 10, 5, 1)
3. Ensuring the total count of each denomination across all employees did not exceed the constraint of 120 total notes
4. Writing the calculated note counts to columns H through P for each employee row

The Python/openpyxl implementation iterated through each employee, calculated denomination breakdowns, and wrote the results to the corresponding cells in the output range while tracking the total note count constraint.
