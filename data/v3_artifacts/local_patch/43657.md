# Solution Note: Counting Green-Highlighted Name Occurrences

The task was to count how many times each person's name appears in green-highlighted cells within a specific row range on the "Jul-22 (2)" sheet.

**Data Structure:**
- Sheet name: "Jul-22 (2)"
- Names list: Column L, cells L2:L8 (Alicia, Ally, Hannah, Jenny, Karen, etc.)
- Search range: Row 3, columns C through G (C3:G3) containing comma-separated names
- Output location: Column K, cells K2:K8

**Approach:**
The solution required reading the fill color (background color) of cells C3:G3 to identify which cells were highlighted in green. For each name in column L (L2:L8), the logic was:
1. Iterate through cells C3:G3
2. Check if the cell's fill color matched green (RGB or theme color)
3. If green, search for the name from column L within that cell's text content (case-insensitive, partial match)
4. Count all matches across the green-highlighted cells
5. Write the count to the corresponding row in column K

The Python/openpyxl approach involved accessing `cell.fill.fgColor` to determine highlighting, using string search operations (similar to Excel's SEARCH function) to find name occurrences within cell text, and writing the summed count back to column K for each name.
