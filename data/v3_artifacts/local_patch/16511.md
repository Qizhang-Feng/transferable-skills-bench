# Solution Note: Track Number Increment by Album

The task required creating a formula to automatically increment track numbers for songs grouped by album name in **Sheet1**.

## Spreadsheet Structure
- **Header row**: Row 1
- **Data range**: Rows 2-10
- **Column D**: 'Album Name' - contains the album grouping identifier
- **Column E**: 'Song Title' - contains song names
- **Column F**: 'Track No' - target column for track number formulas (F2:F10)

## Python/openpyxl Approach

The solution involved reading data from **Sheet1** and writing formulas to column F (Track No) starting at cell **F2**.

The logic implemented was:
1. Read the 'Album Name' values from **column D** (D2:D10)
2. For each row starting at **F2**, create a COUNTIF formula that counts how many times the current album name appears in the range from the first data row up to and including the current row
3. The formula pattern used was: `=COUNTIF($D$2:D2,D2)` for F2, which creates an expanding range as it's copied down
4. Write these formulas to cells **F2 through F10** in column F

This approach ensured that each time an album name appeared, the track number incremented sequentially (1, 2, 3, etc.), and reset to 1 when a new album name was encountered in column D.
