# Solution Note: Conditional Lookup for Branch Managers

## Task Summary
The task required populating the 'result' sheet (cells B2:D12) with designation, contact number, and email from the 'main sheet' based on location matching and specific designation criteria.

## Data Structure
**Source Sheet: 'main sheet'**
- Header row: Row 1
- Data rows: Starting from Row 2
- Columns used:
  - Column A: 'Location'
  - Column B: 'Name of employee'
  - Column C: 'Contact no'
  - Column D: 'Mail id'
  - Column E: 'Designation'

**Target Sheet: 'result'**
- Header row: Row 1
- Data rows: Starting from Row 2
- Columns to populate:
  - Column A: 'Location' (already filled)
  - Column B: 'Designation'
  - Column C: 'Contact no'
  - Column D: 'Mail id'

## Python/openpyxl Approach
The solution iterated through each row in the 'result' sheet starting from row 2. For each location in column A, it searched the 'main sheet' for a matching location in column A. When a match was found, it checked if the designation in column E was either 'Branch Manager' or 'Branch Credit Manager'. If the condition was met, the designation (column E), contact number (column C), and email (column D) were written to columns B, C, and D respectively in the 'result' sheet. If the designation didn't match the criteria, 'N/A' was written to all three columns.
