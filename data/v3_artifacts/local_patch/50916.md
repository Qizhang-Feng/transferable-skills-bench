# Solution Note: School Calendar Cycle Day Auto-Population

The task required creating a formula system to automatically populate class schedules based on cycle day numbers in a school calendar that operates on a 7-day rotating cycle.

## Spreadsheet Structure

The reference data was located in sheet **'21-22 Schedule'** with headers in **row 1**. The cycle day numbers were in **column B** ('Day'), ranging from days 1-5 visible in the data. The class schedule periods were organized in **columns C through H** (H1, P1, P2, P3, P4, P5), with data starting in **row 2** and continuing through **row 6**.

## Solution Approach

The solution involved using INDEX-MATCH formulas in the answer range **C12:H14** that would:

1. Read the cycle day number entered by the user (presumably in column B of rows 12-14)
2. Match that cycle day number against the reference cycle days in **column B, rows 2-6** of sheet '21-22 Schedule'
3. Return the corresponding class name from the appropriate period column (**columns C-H, rows 2-6**)

The formula needed to use absolute references for the lookup table range while allowing relative references for the period columns, enabling it to be copied horizontally across all six period columns and vertically across multiple date rows. This eliminated manual copy-paste work when irregular "off days" required cycle day adjustments.
