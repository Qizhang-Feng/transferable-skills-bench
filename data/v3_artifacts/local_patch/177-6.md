# Solution Note: Merging Rows Based on Column H Match

The task required consolidating rows from the 'DATA' sheet where Column H (ComboKey) values matched, then writing results to the 'combined' sheet starting at A1.

**Data Structure:**
- Source: 'DATA' sheet with headers in Row 1
- Key column: Column H ('ComboKey')
- Data range: Columns A through R
- Data started at Row 2

**Approach:**
The solution read all rows from 'DATA' sheet starting at Row 2. For each row, Column H was used as the grouping key. Rows with identical Column H values were merged by:

1. Preserving columns A-H from the first occurrence (Material, Description, Number, Key, Product#, WS, UM, ComboKey)
2. For columns I-R (date columns from 2022-11-01 through 2023-08-01), summing all numeric values across matching rows, treating non-breaking spaces ('\xa0') and empty cells as zeros

**Output Formatting:**
- Results were written to 'combined' sheet starting at Row 1 (with headers)
- Columns A-H maintained exact original formatting
- Columns I-R were formatted as numbers with exactly 2 decimal places, no thousand separators
- Zero values were displayed as "0" (not blank)

The Python/openpyxl implementation iterated through all DATA rows, grouped by Column H into a dictionary, aggregated numeric values for date columns, then wrote consolidated rows to the combined sheet with proper number formatting applied.
