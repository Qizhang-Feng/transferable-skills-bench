# Solution Note: WHP Data Collection Task

The task required extracting specific referral type data from the "WHP DATA" sheet and placing it in the "WHP" sheet at range E7:G8.

## Data Source
The source data was located in the "WHP DATA" sheet:
- Header row was at row 3, containing: "Local London Partnership", "EEG", "Dis", "LTU"
- Data started at row 4
- Column B contained site names ("Barking", "Bexleyheath")
- Columns C, D, and E contained the referral type values for "EEG", "Dis", and "LTU" respectively

## Target Location
The destination was in the "WHP" sheet:
- Range E7:G8 was the target area
- Row 6 contained headers: "EEG", "Dis", "LTU" in columns E, F, G

## Python/openpyxl Approach
The solution read data from the "WHP DATA" sheet starting at row 4, extracting values from columns C, D, and E (the three referral type columns). The data was then written to the "WHP" sheet starting at cell E7, filling a 2-row by 3-column range (E7:G8). This involved iterating through the first two data rows from "WHP DATA" and mapping each row's three values to the corresponding cells in the target range, maintaining the column order of EEG, Dis, and LTU.
