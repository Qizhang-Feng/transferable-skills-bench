# Solution Note: Remove Matching Values and Calculate Difference

The task was completed using Python with openpyxl on **Sheet1**.

## Data Structure
- **Value column**: Column A, starting from row 3 (header in row 2)
- **Binaries column**: Column B, starting from row 3 (header in row 2)
- **Result values column**: Column D, starting from row 5 (header in row 4)
- **Target value**: Cell E2
- **Solver result**: Cell G2
- **Difference**: Cell J2

## Approach
The solution read the **Result values** from column D (rows 5 onwards) and the **Value** and **Binaries** data from columns A and B (rows 3 onwards). For each value in the Result values column, the code searched for the first matching occurrence in the Value column and removed that single instance along with its corresponding Binaries value.

After removal, the remaining values in both columns A and B were shifted up to eliminate blank cells, maintaining the paired relationship between Value and Binaries entries.

The **Solver result** (cell G2) was calculated as the sum of the remaining values in column A after all removals. The **Difference** (cell J2) was computed as the Target value (E2) minus the Solver result (G2).

The solution preserved the original header structure and only modified the data rows, ensuring that multiple occurrences of the same value in the Value column were handled correctly by removing only one instance per match.
