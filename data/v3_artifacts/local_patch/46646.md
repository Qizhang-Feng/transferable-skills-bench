# Solution Note: Night Shift Percentage Calculation

The task required calculating the percentage of night shifts for each month (January through December) in row 11 of **Sheet1**, with outputs in cells **B11:M11**.

The year value was read from cell **B3** (value: 2016). This year was used to determine the total number of days in each month, accounting for leap year logic (2016 is a leap year, so February has 29 days).

The solution involved:
1. Reading the year from cell **B3** in **Sheet1**
2. Determining if the year was a leap year using the standard leap year formula (divisible by 4, except centuries unless divisible by 400)
3. Calculating the number of days for each month (January=31, February=28/29, March=31, etc.)
4. Computing night shift percentages based on the specific days in each month
5. Writing the calculated percentages to cells **B11** through **M11** (columns B-M representing months January-December)

The output formatting rules applied were:
- Zero values: no decimal places (e.g., "0")
- Single non-zero decimal place: one decimal place (e.g., "0.5")
- All other values: nine decimal places (e.g., "0.123456789")

The Python/openpyxl approach read the year value, applied month-day calculations with leap year logic, computed the night shift percentages, and wrote formatted decimal values to the target range in row 11.
