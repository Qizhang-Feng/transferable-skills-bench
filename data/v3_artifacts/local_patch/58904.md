# Solution Note: Paired Comparison Analysis Letter-Based Sum Formula

## Task Summary
The task required creating formulas to sum numbers associated with specific letters in a Paired Comparison Analysis matrix on **Sheet1**.

## Spreadsheet Structure
- **Header rows**: Row 1 contained letter identifiers (A-G in columns C-I), Row 2 contained parameter names
- **Data matrix**: Rows 3-6 contained the comparison data with mixed letter-number values (e.g., "A2", "B1", "D2")
- **Data range**: The comparison values were located in cells C3:I6
- **Output location**: The sum formulas were written to cells B11:B17

## Python/openpyxl Approach
The solution read the data matrix from **columns C through I (columns 3-9)** and **rows 3 through 6** in **Sheet1**. For each letter (A through G), the logic:

1. Iterated through all cells in the range C3:I6
2. Extracted cell values and checked if they started with the target letter
3. Parsed the numeric portion following the letter using string manipulation
4. Accumulated the sum of all numbers associated with each specific letter
5. Wrote Excel formulas to cells B11:B17 that replicated this logic using SUMPRODUCT with LEFT, VALUE, and ISNUMBER functions to extract and sum numbers from cells beginning with each respective letter

The formulas dynamically referenced the data range C3:I6 and checked for matches against letters A-G.
