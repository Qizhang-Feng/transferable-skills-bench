# Solution Note: Summing Column Values Based on Dynamic Row Range

The task required calculating a sum from a specific column in **Sheet1**, where the column and row range were determined by values in predefined cells.

The data structure consisted of:
- **Row 1**: Building abbreviations (BA, EMA, LMA, PE, PME, CE, Fut, AF, OF, VF, etc.)
- **Row 2**: Building full names (Zues, CoA, GT, Cdm, CF, Traz, Cape, Inno, Arc, Rain, AO, BG, TA, HC)
- **Rows 3-6**: Numeric contribution values for different levels (1-4)

The reference values were located at:
- **Cell S4**: Contains "GBG" (the building identifier to search for)
- **Cell T4**: Contains "ToR" (likely unused in this calculation)
- **Cell T6**: Contains the starting level value (40)

The solution approach involved:
1. Reading the building identifier from **cell S4** ("GBG")
2. Locating which column in Row 2 matched this identifier
3. Reading the starting level from **cell T6** (40)
4. Determining the appropriate row range based on the level thresholds in the identified column
5. Summing the values from that column within the calculated row range (rows 3-6)
6. Writing the result to **cell S10**

The Python/openpyxl implementation read Row 2 to find the column index, then summed values from rows 3-6 of that column, writing the final result to the answer position.
