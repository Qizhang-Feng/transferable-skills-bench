# Solution Note: Pricing Lookup Formula Task

## Task Summary
The task required creating a lookup formula in column E ('Post Cost') of the 'Pricing' sheet that references data from columns C ('Package Type') and D ('Weight (g)') to find matching costs from the 'Package & Weight Data' sheet.

## Spreadsheet Structure
- **Source Sheet**: 'Pricing' with headers in row 1, data starting in row 2
- **Lookup Sheet**: 'Package & Weight Data' with headers in row 1, data starting in row 2
- **Input Columns**: Column C (Package Type) and Column D (Weight in grams) from 'Pricing' sheet
- **Output Column**: Column E (Post Cost) in 'Pricing' sheet, cells E2:E5
- **Lookup Table**: Columns A-D in 'Package & Weight Data' sheet containing Package Type, Weight From (g), Weight To (g), and Price

## Python/openpyxl Approach
The solution read the lookup table from 'Package & Weight Data' sheet (columns A through D, rows 2 onwards) into a data structure. For each row in the 'Pricing' sheet (rows 2-5), the code extracted the Package Type from column C and Weight from column D. It then iterated through the lookup table to find a matching row where the Package Type matched column A AND the Weight fell between the Weight From (column B) and Weight To (column C) values inclusively. The corresponding Price from column D of the lookup table was written to column E of the 'Pricing' sheet.
