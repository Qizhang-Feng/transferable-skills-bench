# Solution Note: Employee Assignment Formula with Multiple Offices

## Spreadsheet Structure
The data was in sheet **Sheet1** with headers in row 1 and data starting from row 2 through row 15.

## Column Layout
- **Column A**: Client ID
- **Column B**: Assigned employee 2023 (current employee)
- **Column C**: Office 1 (potential new employee)
- **Column D**: Office 2 (potential new employee)
- **Column E**: Office 3 (potential new employee)
- **Column F**: Assigned employee 2024 (target output column, F2:F15)

## Logic Applied
The solution needed to determine the 2024 assigned employee by:
1. Checking if any new employee names existed in columns C, D, or E
2. If exactly one new employee was found across C:E, that name was used
3. If no new employees were found (all three columns empty), the current employee from column B was retained
4. If multiple new employees were found across C:E, an "ERROR" message was returned to flag conflicting assignments

## Python/openpyxl Approach
The script read columns B through E for each row starting at row 2. For each row, it counted non-empty cells in the C:E range. Based on the count (0, 1, or 2+), it applied the appropriate logic and wrote the result to column F of the same row.
