# Solution Note: Delete Rows Above First 'Invoice No.' in Column A

## Task Summary
The task required locating the first instance of 'Invoice No.' in column A of the 'Statement' sheet and deleting all rows above it, so that 'Invoice No.' becomes the new row 1.

## Spreadsheet Structure
The data was in the 'Statement' sheet. The first instance of 'Invoice No.' appeared in cell A7 (based on the answer position A2:E12 indicating the final data range after deletion). Rows 1-6 contained header information including company details, account numbers, and blank rows that needed to be removed.

## Python/openpyxl Approach
The solution involved:
1. Loading the workbook and accessing the 'Statement' sheet
2. Iterating through column A (column index 1) starting from row 1
3. Searching for the first cell containing the exact text 'Invoice No.'
4. Once found, determining the number of rows to delete (all rows before the 'Invoice No.' row)
5. Using the `delete_rows()` method to remove rows 1 through (found_row - 1)
6. After deletion, 'Invoice No.' automatically shifted to cell A1
7. Saving the workbook with the modified structure

The final result placed the header row ('Invoice No.', 'Transaction Date', 'Due Date', 'Description', 'Amount') in row 1, with data starting from row 2 onwards in the range A1:E12.
