# Solution Note: Item Quantity Count with Custom Sort

The task required counting occurrences of each item from sheet **'工作表1'** and displaying results in a specific sort order at cells **I2:J7**.

## Data Structure
- **Source data**: Sheet '工作表1', column B ('ITEM'), starting from row 2
- **Header row**: Row 1 contained 'PO', 'ITEM', 'DEC', 'Price'
- **Data rows**: Started at row 2 and could extend to thousands of rows

## Solution Approach
The Python/openpyxl solution involved:

1. **Reading data**: Extracted all values from column B ('ITEM') in sheet '工作表1', starting from row 2 downward until encountering empty cells
2. **Counting logic**: Used Python's `Counter` from the collections module to efficiently count occurrences of each unique item value
3. **Custom sorting**: Applied a specific sort order (A006, A004, A005) using a custom key function to arrange items in the required sequence
4. **Writing results**: 
   - Wrote headers "ITEM" and "Qty" to cells I2 and J2
   - Populated sorted item names in column I (starting I3)
   - Populated corresponding counts in column J (starting J3)
   - Applied left alignment to the entire output range I2:J7

This approach efficiently handled large datasets (10,000+ rows) without the manual overhead of SUBTOTAL formulas, providing instant results regardless of data volume.
