# Rolling 365-Day Average Solution Note

The data was located in **Sheet1** with headers in **row 3**. The date column was **column A** ('Date'), the values to average were in **column B** ('Beans Counted'), and the rolling average results were written to **column C** ('12 month rolling average').

Data rows started at **row 4** and extended to **row 200** (or the last row with data).

## Python/openpyxl Approach

The solution read dates from **column A** and values from **column B** starting at **row 4**. For each row, the logic calculated a 365-day rolling average by:

1. Identifying the current row's date
2. Filtering all previous rows (including the current row) where the date fell within 365 days prior to the current date
3. Computing the average of the 'Beans Counted' values from those filtered rows
4. Writing the calculated average to **column C** of the current row

This approach handled sparse data (non-daily entries) by dynamically determining which rows fell within the 365-day window for each calculation point. The formula was applied to each row individually, ensuring the range expanded and contracted appropriately based on date differences rather than fixed row counts.

The results were written directly to cells **C4:C200** using openpyxl, allowing the spreadsheet to be converted to a Table format afterward for automatic extension with new data.
