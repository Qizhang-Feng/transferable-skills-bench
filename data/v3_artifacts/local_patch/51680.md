# Solution Note: Identifying Business Areas Using Stock Items

The task involved analyzing data in **Sheet1** where columns **A through F** contained 'Y' or 'N' indicators showing which business areas use specific items. The header row (row 1) contained area names: 'Blue', 'Black White', 'Green ', 'Red Purple', 'Orange', and 'Brown' in columns A1 through F1 respectively.

The solution required reading each data row starting from **row 2** through **row 14**, checking columns A through F for 'Y' values, and compiling the corresponding header names into a comma-separated list in column G.

**Python/openpyxl approach:**
1. Read the header row (A1:F1) to extract the business area names
2. For each data row (rows 2-14), iterate through columns A-F
3. Check if the cell value equals 'Y'
4. If 'Y' is found, append the corresponding header name from that column to a list
5. Join the collected header names with ", " (comma and space) as separator
6. Write the resulting string to the corresponding cell in column G (G2:G14)

The logic maintained the left-to-right order of columns, ensuring the output followed the sequence of headers. No trailing comma was added. The existing formatting in column G ('Joined Colours') was preserved, and the formula/logic was applied consistently across all data rows.
