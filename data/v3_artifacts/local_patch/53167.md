# Solution Note: Product Usage Calculation

## Task Summary
The task required calculating the amount of product used since its last usage for each serial number entry in **Sheet1**.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Header Row**: Row 3
- **Data Range**: Rows 4-16
- **Column B**: Serial Number (tracking individual products)
- **Column C**: Date (timestamp of each measurement)
- **Column D**: Weight (current weight of product)
- **Column E**: Amount Used (target column for formulas, range E4:E16)

## Solution Approach
The solution used Python with openpyxl to:

1. Read data from **columns B (Serial Number), C (Date), and D (Weight)** starting from row 4
2. For each row, identify the previous occurrence of the same serial number by searching backwards through earlier rows
3. Calculate the amount used as: previous weight minus current weight (from column D)
4. For the first occurrence of each serial number, the amount used was set to 0 (no previous measurement exists)
5. Write the calculated values to **column E (Amount Used)** for rows 4-16

## Key Logic
The calculation tracked each serial number independently. When a serial number appeared multiple times, the formula compared the current weight against the most recent previous weight for that same serial number. This required maintaining state or looking back through the dataset to find matching serial numbers and their corresponding weights.
