# Solution Note: Contract Cost Savings Calculation

The task required calculating total cost savings over an 81-month contract period for multiple savings occurring at different months.

**Spreadsheet Structure:**
- Sheet name: `Sheet1`
- Header row: Row 2
- Data rows: Rows 3-38
- Month columns: `Month 12` (column D), `Month 13` (column E), `Month 18` (column F), `Month 21` (column G), `Month 24` (column H), `Month 25` (column I), `Month 30` (column J), `Month 36` (column K), `Month 37` (column L), `Month 48` (column M), `Month 60` (column N), `Month 72` (column O), `Month 78` (column P)
- Output column: `Priced in productivity` (column Q, cells Q3:Q38)

**Logic Applied:**
For each row (TI number), the calculation iterated through each month column. When a non-zero value was found in a month column, the formula applied was: `(81 - (month_number - 1)) × savings_amount`. The month number was extracted from the column header (e.g., "Month 21" → 21). The subtraction of 1 accounts for the saving being included within the given month.

All individual savings for each row were summed to produce the total cost savings, which was written to the corresponding cell in column Q.

**Python/openpyxl Approach:**
The solution read month column headers from row 2, extracted month numbers using string parsing, iterated through data rows 3-38, applied the calculation formula for each non-zero cell value, and wrote the summed result to column Q.
