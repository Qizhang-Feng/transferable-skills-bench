# Solution Note: Time-Based Forecast Value Calculation

The data was located in **Sheet1** with headers in row 1. The task required populating column E (**Forecast VALUE**) in the range **E2:E15** based on time-dependent logic.

## Data Structure
- **Column A (Time)**: Contains time values starting from row 2
- **Column B (Actual Value)**: Contains actual values to use before 09:45
- **Column C (Forecast Value)**: Contains forecast values to use after 09:45
- **Column E (Forecast VALUE)**: Target column for calculated results

## Logic Applied
The solution read time values from column A (cells A2:A15). For each row:
- If the time was before 09:45:00, the value from column B was used
- If the time was 09:45:00 or later, the value from column C was used
- If the selected value was 0 or empty, the cell was left blank
- Otherwise, the numeric value was written to the corresponding cell in column E

## Formatting
After populating the values, the fill color **#FFC000** (orange) was applied to the range **E2:E15** using openpyxl's PatternFill with fill_type='solid'. All other existing formatting in the workbook remained unchanged.

The Python approach used openpyxl to load the workbook, iterate through rows 2-15, apply the conditional logic based on time comparison, write results to column E, and apply the specified background color.
