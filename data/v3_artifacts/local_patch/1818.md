# Solution Note: Filtering Lowest Performing Students

The task required identifying students with "Lowest Performing" status from the 'Data' sheet and copying their information to the 'Summary' sheet.

## Data Structure
- **Source Sheet**: 'Data' with headers in row 1
  - Column B: 'Student Number' (e.g., MR21_001, MR21_002, etc.)
  - Column C: 'Student Name' (e.g., AA1, AA2, etc.)
  - Column D: 'Performance' (values: "Best Performing", "Average Performing", "Lowest Performing")
  - Data started from row 2

- **Target Sheet**: 'Summary' with headers in row 2
  - Column B: 'Student Number'
  - Column C: 'Student Name'
  - Target range: B3:C68 (66 rows available for results)

## Python/openpyxl Approach
The solution iterated through all rows in the 'Data' sheet starting from row 2. For each row, the value in column D ('Performance') was checked. When a row contained exactly "Lowest Performing", the corresponding values from column B ('Student Number') and column C ('Student Name') were extracted and written to the 'Summary' sheet starting at row 3 (B3:C3 for the first match, B4:C4 for the second, etc.). The write position incremented with each matching student found, filling the range B3:C68 sequentially until all lowest performing students were copied.
