# Solution Note: Word Pattern Matching with Letter Rotation

The task required finding word pairs in column A of sheet 'Sheet1' where one word transforms into another by moving its 5th letter to the first position, while both words share the same last three letters.

**Data Location:**
- Input data was in column A of sheet 'Sheet1', starting from row 1 (no header row)
- The example showed words like 'AROINTED', 'DERATION', 'ORDINATE', 'RATIONED', 'ORIENTAL', 'RELATION'
- Results needed to be written to columns C and D, starting at row 2 (C2:D5000)
- Row 1 contained an example: 'EARTHPEA' in C1 and 'HEARTPEA' in D1

**Python/openpyxl Approach:**
1. Read all words from column A of sheet 'Sheet1' (rows 1 through 5000 or until empty)
2. For each word, extract the last three letters as a suffix pattern
3. Generate a transformed version by moving the 5th character to position 1 (e.g., 'EARTHPEA' → 'HEARTPEA')
4. Check if the transformed word exists in the word list
5. If a match is found, write the original word to column C and the transformed word to column D
6. Continue writing matches sequentially starting from C2:D2 downward

The algorithm grouped words by their last three letters for efficient matching and validated that both the original and rotated versions existed in the dataset.
