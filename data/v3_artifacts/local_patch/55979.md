# Solution Note: Supplier Lookup Formula

## Task Summary
The task required creating a formula in cell B7 of the 'Calc' sheet that identifies which supplier provides a specific item number entered in cell A10.

## Spreadsheet Structure
- **Calc sheet**: Input cell A10 (item number), output cell B7 (supplier name)
- **Supplier sheets**: Three sheets named 'Supplier_1', 'Supplier_2', and 'Supplier_3'
  - Each contained headers in Row 1: 'Item ', 'Name', 'PL-code', 'Price'
  - Item numbers were in column A, starting from Row 2
  - Supplier_1 items: A1111101-A1111105
  - Supplier_2 items: B2222201-B2222205
  - Supplier_3 items: C3333301-C3333305

## Python/openpyxl Approach
The solution involved:
1. Reading the item number from cell A10 in the 'Calc' sheet
2. Iterating through each supplier sheet ('Supplier_1', 'Supplier_2', 'Supplier_3')
3. Searching column A (starting from Row 2) in each supplier sheet for a match with the input item number
4. When a match was found, extracting the supplier name from the sheet name
5. Writing the supplier name to cell B7 in the 'Calc' sheet

The logic checked each supplier's item list sequentially until finding the matching item number, then returned the corresponding supplier identifier.
