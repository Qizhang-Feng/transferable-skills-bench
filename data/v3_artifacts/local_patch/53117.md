# Solution Note: Conditional Country Output with Maximum Invoice Lookup

The data was located in **Sheet1** with headers in row 1 and data starting from row 2 through row 27.

**Input columns:**
- Column A: 'Customer' - customer identifiers
- Column B: 'Total' - invoice amounts
- Column C: 'Country' - country codes

**Output column:**
- Column D: 'Output Need' - the target column for results (D2:D27)

**Logic implemented:**

The solution read all data from columns A, B, and C starting at row 2. For each row, the logic checked the value in column C:

1. If the country in column C was NOT 'US', the formula wrote the same country value to column D
2. If the country in column C was 'US', the solution performed a lookup:
   - Filtered all rows where column A matched the current customer
   - Excluded rows where column C was 'US'
   - Found the maximum value in column B among these filtered rows
   - Identified which country (column C) corresponded to this maximum invoice amount
   - Wrote that country name to column D

The Python/openpyxl approach iterated through each data row, applied conditional logic based on the country value, and performed aggregation operations across the entire dataset to find customer-specific maximum non-US invoice amounts when needed. Results were written directly to column D cells.
