# Solution Note: Comparing Two Data Sources with Specific Logic

The task required comparing values between two source tables and populating a result table based on specific conditions.

**Data Structure:**
- Sheet 'Source' contained columns 'Item' (A) and 'Value' (B), with headers in row 1 and data starting in row 2
- Sheet 'Source2' contained column 'Value' (A), with header in row 1 and data starting in row 2
- Sheet 'Result' had column 'Value' (A) with header in row 1, requiring output starting at cell A2

**Logic Applied:**
1. Read all values from 'Source' sheet, column B (starting row 2)
2. Read all values from 'Source2' sheet, column A (starting row 2)
3. Identify values present in 'Source' but not in 'Source2'
4. Apply special rule: exclude value 2 from the differences UNLESS it appears in both sources
5. If value 2 exists in both 'Source' and 'Source2', include it exactly once in the result
6. Remove duplicate values from the final result set

**Implementation:**
Using openpyxl, the solution read both value columns into Python sets/lists, performed set difference operations with the exception handling for value 2, checked for its presence in both sources, and wrote the unique results to 'Result' sheet starting at A2. The output was sorted and deduplicated before writing.
