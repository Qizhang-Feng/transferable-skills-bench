# Solution Note: Filter Rows by Multiple OR Conditions

The task required filtering data from **Sheet1** based on multiple OR conditions and writing results to **Sheet2**.

## Source Data
The source data was in **Sheet1** with headers in row 1:
- Column A: 'SL'
- Column B: 'ITEM' 
- Column C: 'CLASS'

Data rows started from row 2 onwards.

## Filter Logic
The filtering criteria applied was:
- Column B ('ITEM') equals 'TELIVISION' **OR**
- Column C ('CLASS') equals 'CLASS III' **OR**
- Column C ('CLASS') equals 'CLASS IV'

## Python/openpyxl Approach
1. Loaded the workbook and accessed **Sheet1**
2. Read the header row from row 1 (columns A through C)
3. Iterated through data rows starting from row 2
4. For each row, checked if column B value was 'TELIVISION' OR column C value was 'CLASS III' OR column C value was 'CLASS IV'
5. Collected all matching rows that satisfied any of these conditions
6. Wrote headers to **Sheet2** starting at cell **A2**
7. Wrote filtered data rows to **Sheet2** starting at cell **A3**

The output range in **Sheet2** spanned from A1 to C18, which included the headers at A2:C2 and the filtered data rows below starting at A3.
