# Solution Note: Conditional Addition/Subtraction Based on Column Value

## Task Summary
The task required creating formulas in the 'Total' column (column D) of 'Sheet1' that conditionally add or subtract 10 based on the value in 'Column1' (column A).

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Header Row**: Row 1
- **Data Range**: Rows 2-5
- **Key Columns**:
  - Column A: 'Column1' (contains 'A' or 'B' values)
  - Column B: 'Column2' (numeric values)
  - Column C: 'Column3' (numeric values)
  - Column D: 'Total' (target column for formulas)

## Python/openpyxl Approach
The solution read data from 'Column1' (column A) starting at row 2 through row 5. For each row, the logic checked if the value in column A was 'A' or 'B'. 

The formula logic was:
- If Column1 = 'A': Sum Column2 + Column3 + 10
- If Column1 = 'B': Sum Column2 + Column3 - 10

The formulas were written to cells D2 through D5 using Excel formula syntax (e.g., `=IF(A2="A", B2+C2+10, B2+C2-10)`). This approach used openpyxl to iterate through rows 2-5, construct the appropriate IF formula for each row referencing the corresponding cells in columns A, B, and C, and assign these formulas to the 'Total' column (column D).
