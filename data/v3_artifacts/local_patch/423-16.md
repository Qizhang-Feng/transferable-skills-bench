# Solution Note: Transpose Yellow Cells from Test Sheet to Master Sheet

The task required transposing yellow-highlighted (#FFFF00) cells from column A of the 'Test' sheet into row 1 of the 'master' sheet, starting at cell A1.

## Approach Used

**Source Data:**
- Sheet name: 'Test'
- Column: A (column index 1)
- The cells in column A contained values like 'Working Paper', 'Satker xxx', 'Purpose ;……', 'Jalan xxxx', 'BM 2850 AP', 'Data Transaksi Penjualan', 'Objek Penilaian'
- Only cells with yellow background color (#FFFF00) were selected for transposition

**Target Location:**
- Sheet name: 'master'
- Row: 1 (starting at A1, extending to AH1)
- Existing headers in row 1 were: 'Photo', 'Information', 'Linked', 'Price', 'Jenis Transaksi', 'Waktu Transaksi', 'etc….'

**Python/openpyxl Logic:**
1. Iterated through all cells in column A of the 'Test' sheet
2. Checked each cell's fill color to identify yellow (#FFFF00) background
3. Collected non-blank yellow cells into a list
4. Wrote the collected values horizontally into row 1 of the 'master' sheet, starting at column A
5. Applied yellow fill (#FFFF00) and borders to all transposed cells
6. Skipped any blank cells during the transposition process

The solution preserved only yellow-highlighted, non-empty values and formatted them consistently in the destination row.
