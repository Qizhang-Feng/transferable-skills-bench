# Solution Note: Delete Rows Where Column I Has Value But Column H Is Blank

## Task Summary
The task required identifying and deleting entire rows from **Sheet1** where column I contained a value but the corresponding cell in column H was empty.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Data Range**: Rows 1-6 with data in columns A through I
- **Header Row**: Row 1 contained data (not traditional headers)
- **Key Columns**: 
  - Column H (8th column) - Date field that could be blank
  - Column I (9th column) - Numeric values

## Python/openpyxl Approach
The solution iterated through **Sheet1** starting from **row 1** through the last row with data. For each row, the logic checked:
1. Read the value in **column I** (9th column)
2. Read the value in **column H** (8th column)
3. If column I had a value (not None/empty) AND column H was blank (None or empty string), mark that row for deletion

Rows were deleted in reverse order (from bottom to top) to avoid index shifting issues that occur when deleting rows sequentially from top to bottom.

## Result
Row 4 met the deletion criteria (column I had value '49618.31' while column H was empty), so it was removed. The final dataset contained rows 1-3 and 5-6 (renumbered as 1-5 after deletion), preserving all data in the range **A1:I5**.
