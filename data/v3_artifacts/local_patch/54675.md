# Solution Note: Counting Partial Text Matches with Conditions in Excel

The task required modifying a COUNTIFS formula in cell F3 of Sheet1 to count occurrences where column B contained "Gary Stoddard" (including partial matches within cells with multiple names) AND column A contained the value "4".

The data structure was:
- **Sheet Name**: Sheet1
- **Column A**: "Month" (header in row 1, data starting row 2)
- **Column B**: "Investigating Officer (s)" (header in row 1, data starting row 2)
- **Target Cell**: F3

The original formula `COUNTIFS(B:B,"Gary Stoddard",A:A,"4")` only matched exact cell contents. To include partial matches (e.g., "Gary Stoddard, Gary Stoddart, Gary Stoddart, Gary Stoddart" in row 4), the formula needed wildcard characters.

**Python/openpyxl approach:**
1. Read column A (Month) and column B (Investigating Officer(s)) from Sheet1, starting from row 2
2. Iterate through each row and check if:
   - Column A value equals "4" (or the string "4")
   - Column B value contains the substring "Gary Stoddard" (case-sensitive or insensitive as needed)
3. Count matching rows
4. Write the count result to cell F3

The modified Excel formula used wildcards: `=COUNTIFS(B:B,"*Gary Stoddard*",A:A,"4")` where asterisks enabled partial text matching within cells.
