# Bond Cashflow Automation Solution Note

The task automated cashflow generation for bonds in **Sheet1**. 

## Data Structure
- **Column A (Principal)**: Bond principal amounts
- **Column B (Interest)**: Interest payment amounts
- **Column C (Frequency)**: Payment frequency (1=annual, 2=semi-annual, 4=quarterly)
- **Column D (Maturity)**: Maturity dates for each bond
- **Row 4 (E4:AB4)**: Monthly date headers from 2022-01-31 to 2023-12-31
- **Row 5**: Display "-" for all columns E5:AB5
- **Rows 6-25**: Bond data and calculated cashflows

## Python/openpyxl Approach

The solution read bond parameters from columns A through D starting at row 6. For each bond row (6-25), the logic:

1. Retrieved the maturity date from column D
2. Read the payment frequency from column C to determine payment intervals (12/frequency months)
3. Calculated payment dates by working backwards from the maturity date using the frequency
4. For each date column (E4:AB4), checked if the column header matched a calculated payment date
5. If matched, wrote the interest amount from column B; otherwise wrote an empty string
6. For row 5 specifically, wrote "-" across all date columns (E5:AB5)

The formula logic determined payment schedules dynamically based on each bond's specific maturity date and frequency, populating the cashflow grid in range E6:AB25.
