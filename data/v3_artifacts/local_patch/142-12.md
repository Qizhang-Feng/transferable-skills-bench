# Solution Note: Update Marble Slab Creamery Entries with Whole Dollar Amounts

## Task Summary
The task required identifying and updating specific entries in **Sheet1** where Column F contained "Marble Slab Creamery" and Column J contained whole number amounts (no cents).

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Target Columns**: 
  - Column F: Contained merchant/vendor names
  - Column J: Contained transaction amounts (as strings or numbers)
- **Data Range**: Rows 1-15 (F1:J15)
- **Header Row**: Row 1 appeared to be the first data row (no explicit headers)

## Solution Approach
The solution iterated through Column F and Column J simultaneously in Sheet1. For each row:

1. **Read** the value from Column F to check if it exactly matched "Marble Slab Creamery"
2. **Read** the corresponding value from Column J and converted it to a float
3. **Applied logic** to determine if the amount was a whole number by checking if `amount % 1 == 0` (no decimal/cents portion)
4. **Updated** Column F to "Georgia State WH" when both conditions were met

The code used openpyxl to load the workbook, access Sheet1, and iterate through the specified range. After making changes, the workbook was saved to preserve the updates. The specific rows mentioned (F10 and F13) were examples where both conditions were satisfied.
