# Solution Note: Maximum Turnover Display by ID

## Task Summary
The data was in sheet **'Before'** with headers in row 1. Column A contained 'UNIQUE ID', column B contained 'AMOUNT', and column C contained 'DATE'. The task required populating column D (rows 2-13) with the maximum turnover amount for each ID, displaying the value only in the row where that maximum occurred for that ID.

## Python/openpyxl Approach

1. **Read the data**: The UNIQUE ID values were read from column A (cells A2:A13) and AMOUNT values from column B (cells B2:B13) starting from row 2 (first data row after header).

2. **Group by ID**: The data was grouped by UNIQUE ID to identify all rows belonging to each ID and determine the maximum AMOUNT value for each group.

3. **Find maximum per ID**: For each unique ID, the MAX function logic was applied to find the highest AMOUNT value among all rows with that ID.

4. **Conditional write**: For each row in the range D2:D13, the logic checked if the current row's AMOUNT (column B) matched the maximum AMOUNT for that row's ID. If true, the maximum value was written to column D; otherwise, an empty string was written.

5. **Output location**: Results were written to column D, cells D2:D13, with no header in D1 and no formatting applied.
