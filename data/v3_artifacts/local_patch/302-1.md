# Solution Note: Matching and Replacing Item Values Across Sheets

## Task Summary
The task required replacing mismatched item values in SHEET1 and SHEET2 with correct values from the ITEMS sheet, based on row-by-row comparison using the S.N column as the key.

## Spreadsheet Structure
- **ITEMS sheet**: Master reference data with columns A (S.N) and B (ITEM), header in row 1, data starting from row 2 through row 6
- **SHEET1**: Data to be corrected with columns A (S.N) and B (ITEM), header in row 1, data in rows 2-6
- **SHEET2**: Data to be corrected with columns A (S.N) and B (ITEM), header in row 1, data in rows 2-6

## Python/openpyxl Approach
1. Loaded the workbook and accessed sheets: ITEMS, SHEET1, and SHEET2
2. Read the master data from ITEMS sheet, column B (ITEM), rows 2-6, creating a dictionary mapping S.N values from column A to their corresponding ITEM values
3. For SHEET1, iterated through column B cells from B2 to B6, comparing each value against the corresponding ITEM value from the ITEMS dictionary (matched by S.N in column A)
4. Replaced mismatched values in SHEET1 column B with correct values from the ITEMS dictionary
5. Repeated the same comparison and replacement process for SHEET2, column B, rows 2-6
6. Saved the workbook with corrected values in both sheets

The answer ranges were SHEET1!B2:B8 and SHEET2!B2:B7 (extended to include potential additional rows).
