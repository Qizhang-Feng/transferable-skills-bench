# Solution Note: Volume Lookup Across Sheets

## Task Summary
The task required retrieving volume values from the 'Volume' sheet and populating column H ('Volume') in the 'Sea Freight Rates' sheet (rows 2-191) based on matching origin port, equipment type, and destination port.

## Spreadsheet Structure
**Sea Freight Rates sheet:**
- Headers in row 1: 'Origin Port (portcode)' (column B), 'Destination Port (portcode)' (column D), 'Equipment type' (column E), 'Volume' (column H)
- Data starts at row 2, extends to row 191

**Volume sheet:**
- Headers in row 2: 'Origin Port' (column A), 'Equipment type' (column B), destination port codes ('NLRTM PT', 'DEBRV PT', 'DEHAM PT') in columns D, E, F respectively
- Row 1 contains 'Destination Port' label
- Data starts at row 3

## Python/openpyxl Approach
The solution read the 'Volume' sheet starting from row 3, extracting origin port codes (column A) and equipment types (column B). The destination port headers were read from row 2 (columns D onwards) to create a mapping structure.

For each row in 'Sea Freight Rates' (rows 2-191), the code:
1. Read origin port code (column B), destination port code (column D), and equipment type (column E)
2. Located the matching row in the Volume data by origin port and equipment type
3. Found the appropriate destination port column in the Volume sheet
4. Retrieved the volume value and wrote it to column H in 'Sea Freight Rates'
5. Wrote "#N/A" when no match was found
