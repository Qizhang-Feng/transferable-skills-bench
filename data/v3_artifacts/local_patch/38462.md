# Solution Note: Rounding Time Values to Nearest Whole Hour

The task involved rounding time values formatted as `[h]:mm` to the nearest whole hour in Excel format. The data was located in **Sheet1**.

## Data Structure
- **Column B** contained time values formatted as strings (e.g., "2046 days, 17:30:00")
- **Column C** was the target output location for rounded hour values
- Data started at **row 1** (no header row present)
- The answer was required in cells **C1:C2**

## Python/openpyxl Approach

The solution read values from **column B (cells B1, B2)** in **Sheet1**. The logic involved:

1. Parsing the string format "X days, HH:MM:SS" to extract total hours
2. Converting days to hours (days × 24) and adding the hour component
3. Rounding minutes up to the next hour if any minutes were present (using ceiling function)
4. Writing the rounded integer hour values to **column C (cells C1, C2)**

The key challenge was handling the custom time format exported from AMOS software, which represented cumulative hours as days plus time. The solution converted the entire duration to hours, then applied ceiling rounding to ensure any partial hour rounded up to the next whole hour. The output format was set to General (integer) rather than time formatting.
