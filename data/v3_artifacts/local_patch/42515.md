# Solution Note: Dynamic Foreign Exchange Rate Formula

## Problem Context
The task involved modifying formulas in **Sheet1**, specifically in **TABLE 2** (columns E-G), to dynamically reference exchange rates from **TABLE 1** (columns A-C) without manual adjustment of absolute cell references.

## Spreadsheet Structure
- **TABLE 1** started at row 4 with headers: 'CROSS-PARITY IN GBP' (column A), 'MIDRATE' (column B), 'PRICEDATE' (column C)
- **TABLE 2** started at row 4 with headers: 'CROSS-PARITY IN EUR' (column E), 'EUR-XXX CROSS-RATE' (column F), 'RATE IN EUR' (column G)
- Data rows began at row 5 and extended through row 19
- The original formula `SUM(1/((1/B5)*$B$8))` used a fixed absolute reference ($B$8) that needed to increment as the formula was dragged down

## Python/openpyxl Approach
The solution read data from **column B (MIDRATE)** in TABLE 1, starting from row 5. For each row in the target range **F5:F19**, the formula was constructed to reference the corresponding row offset in column B. The logic calculated: `1 / ((1 / B[current_row]) * B[current_row + offset])` where the offset incremented with each row. The computed formulas were written to **column F (EUR-XXX CROSS-RATE)** in TABLE 2, creating a dynamic reference pattern that automatically adjusted when dragged down.
