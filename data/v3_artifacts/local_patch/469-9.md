# Solution Note: Separating Debits and Credits

The task required separating positive and negative values from the 'Amount' column (Column C) in sheet 'Sheet1' into two distinct columns: 'Debits' (Column H) and 'Credits' (Column I).

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Source Column**: 'Amount' (Column C)
- **Target Columns**: 'Debits' (Column H) and 'Credits' (Column I)
- **Header Row**: Row 1
- **Data Start Row**: Row 2

## Python/openpyxl Approach

The solution read values from Column C starting at row 2 (after the header). For each value in the 'Amount' column:

1. **Headers were written** to cells H1 ('Debits') and I1 ('Credits')
2. **Logic applied**: 
   - If the amount was positive, the absolute value was written to Column H (Debits) and Column I (Credits) was left empty
   - If the amount was negative, the absolute value was written to Column I (Credits) and Column H (Debits) was left empty
   - Empty cells in Column C resulted in empty cells in both H and I

3. **Data was written** row-by-row from H2:I10, matching the corresponding row numbers from the source data

The workbook was then saved with the new columns populated according to the debit/credit classification based on the sign of the original amount values.
