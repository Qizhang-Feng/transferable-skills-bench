# Solution Note: Finding Sellers with Minimum Price Above Zero

The task was to identify which seller(s) offered the minimum price greater than zero for each product in Sheet1.

**Data Structure:**
- Header row: Row 1
- Data rows: Rows 2-6
- Product names: Column A ('Product name')
- Price columns: Columns B-N ('Amazon', 'HN', 'Catch', 'Costco', 'DJ', 'THX', 'H', 'KWH', 'MBX', 'MMX', 'MY', 'POK', 'VB')
- Output location: Column P ('Seller'), cells P2:P7

**Python/openpyxl Approach:**

The solution read data from Sheet1 starting at row 2. For each product row, it iterated through columns B through N to extract all price values. The logic filtered out empty cells and zero values, then identified the minimum value among remaining positive prices.

After finding the minimum price greater than zero, the code searched through the same price columns again to collect all column headers where the price matched this minimum value. When multiple sellers offered the same minimum price, their column headers were concatenated with commas (no spaces) as the separator.

The resulting seller name(s) were written to column P in the corresponding row. Empty strings were written when no valid prices existed for a product.

This approach handled missing data (empty cells), explicit zeros, and multiple sellers with identical minimum prices.
