# Solution Note: Combined SUMIFS and Forecast Formula

The task required populating cells E6:P9 on the 'Summary' sheet with a hybrid formula that uses actual values when available, otherwise calculates forecast values.

**Data Structure:**
- Sheet 'Data' contains columns: 'GL Acct' (A), 'Desc' (B), 'Amount' (C), 'Period' (D), with data starting at row 2
- Sheet 'Summary' has GL accounts in column C (starting row 6), period headers in row 5 (columns E through P representing periods 1-12), and cell D3 contains the number of actual months completed (3)

**Logic Applied:**
For each cell in the range E6:P9:
1. The period number was determined from the column header in row 5
2. A SUMIFS lookup was performed on sheet 'Data' to sum 'Amount' (column C) where 'GL Acct' (column A) matched the account in column C of the current row, and 'Period' (column D) matched the current column's period
3. If the SUMIFS returned a value > 0, that actual value was used
4. Otherwise, a forecast was calculated as: SUM(actuals from columns E:G for that row) / (12 - D3), where D3 contained the actual months count
5. The formula was written to each cell in the range E6:P9

The approach used openpyxl to read data from both sheets and write the combined formula logic to the target range.
