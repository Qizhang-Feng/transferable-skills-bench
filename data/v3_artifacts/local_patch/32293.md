# Solution Note: Conditional Formula Population Based on Lookup Values

## Task Summary
The task required populating cells B14 through B17 in Sheet1 based on the value in cell A14 using conditional logic that matched against a lookup table.

## Spreadsheet Structure
The lookup table was located in Sheet1, rows 1-4:
- Column A (rows 1-4) contained the lookup keys: MNQ, MES, NQ, ES
- Column B (rows 1-4) contained the corresponding values: 0.56, 0.56, 2.09, 2.04

The target range was B14:B17 in Sheet1, with the condition value in cell A14.

## Python/openpyxl Approach
The solution read the value from cell A14 in Sheet1 to determine which lookup key to match. The logic implemented a conditional check:
- If A14 equals "MNQ", the value from B1 (0.56) was written to B14
- If A14 equals "MES", the value from B2 (0.56) was written to B14
- If A14 equals "NQ", the value from B3 (2.09) was written to B14
- If A14 equals "ES", the value from B4 (2.04) was written to B14

The implementation used openpyxl to read cell A14, performed the conditional matching against the lookup table in A1:B4, retrieved the corresponding value from column B, and wrote it to cell B14. Cells B15:B17 were left empty as they were not part of the conditional logic requirement.
