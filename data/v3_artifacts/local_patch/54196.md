# VLOOKUP Lead Time Retrieval Solution Note

## Problem
The task required populating the "Estimated Delivery time" column in 'Sheet two' (column E, cells E2:E279) by looking up supplier information from 'Sheet one'.

## Data Structure
- **Sheet one** contained the lookup table with:
  - Column A: 'Supplier #' (raw supplier numbers: 11, 12, 13, etc.)
  - Column B: 'Estimated Delivery time' (values like '6D', '1D', '4D')
  - Column C: 'Standardized supplier nr' (zero-padded: '0000000011', etc.)
  - Headers in Row 1, data starting Row 2

- **Sheet two** contained the main data with:
  - Column B: 'Supplier #' (matching format to Sheet one, column A)
  - Column E: 'Column1' (target column for results)
  - Headers in Row 1, data starting Row 2

## Python/openpyxl Solution
The solution read the 'Supplier #' values from 'Sheet two' column B (B2:B279) and matched them against 'Sheet one' column A (Supplier #). For each match found, the corresponding 'Estimated Delivery time' from 'Sheet one' column B was retrieved and written to 'Sheet two' column E. A dictionary was created mapping supplier numbers to delivery times from 'Sheet one', then iterated through 'Sheet two' rows to populate the lookup results. Rows without matches remained empty or retained existing values.
