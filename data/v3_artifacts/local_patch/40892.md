# Solution Note: Color Extraction from Product Names

## Task Summary
The task required extracting color names from product descriptions in column A and displaying the matched color in column B, but only if the color exists in a reference list in column D.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Header Row**: Row 1
- **Data Range**: Rows 2-17
- **Input Column**: Column A ('Product Name') - contains product descriptions with embedded color names
- **Output Column**: Column B ('Needs to return value from list when A2 etc matches') - where extracted colors were written
- **Reference List**: Column D ('List') starting from row 2 - contains valid color names (Red, Gold, Blue, etc.)

## Solution Approach
The solution used Python with openpyxl to:

1. Read the reference color list from column D (D2 onwards) into a set for efficient lookup
2. Iterate through each cell in column A from row 2 to row 17
3. For each product name, search for any color from the reference list (case-insensitive matching)
4. When a color match was found, write the properly capitalized color name to the corresponding cell in column B
5. If no color match was found, write an empty string to column B

The matching logic performed case-insensitive substring searches to handle colors appearing anywhere within the product name text.
