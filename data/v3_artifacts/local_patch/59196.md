# Solution Note: Finding Column with Maximum Value and Returning Header

The task required identifying which column (A, B, C, or D) contained the highest value in each row and returning that column's header.

**Spreadsheet Structure:**
- Sheet name: `Sheet1`
- Header row: Row 2
- Column headers were located in cells D2:G2 with values 'A', 'B', 'C', 'D'
- Data values were in the range D3:G5 (3 rows of data)
- Results needed to be written to column H, cells H3:H5

**Python/openpyxl Approach:**

1. Read the header row from cells D2:G2 to capture the column names ('A', 'B', 'C', 'D')
2. For each data row (rows 3-5):
   - Read the numeric values from columns D through G
   - Identify the maximum value in that row
   - Determine which column position contained the maximum value
   - Map that position back to the corresponding header from D2:G2
3. Write the resulting header letter to column H (cells H3:H5)

The logic involved iterating through each row, using Python's `max()` function combined with `index()` to find the position of the maximum value, then retrieving the corresponding header from the stored header list. This approach handled ties by returning the first occurrence of the maximum value.
