# Solution Note: Conditional Sum with Date Range and Text Search

The task required summing values from **Sheet1** based on date range criteria and text search terms.

**Data Structure:**
- Date values were in column A (starting row 3)
- Text values to search were in column B (starting row 3)
- Values to sum were in column C (starting row 3)
- Date range boundaries were in cells E4 (start date) and F4 (end date)
- Search terms were listed in column H (rows 4-7)
- Output formulas were written to column I (rows 4-7)

**Logic Applied:**
For each search term in column H, the solution needed to:
1. Check if the date in column A fell between the start date (E4) and end date (F4)
2. Check if the corresponding text in column B contained the search term from column H
3. Sum the corresponding values from column C where both conditions were true

**Python/openpyxl Approach:**
The solution iterated through rows 4-7 in column H to read each search term. For each term, it constructed a SUMIFS formula that:
- Summed column C (Units Sold)
- Applied date range criteria using column A against E4 and F4
- Applied text search criteria using SEARCH or FIND function on column B to match the term in column H

The formulas were written to cells I4 through I7, with appropriate absolute and relative cell references to maintain correct ranges when copied.
