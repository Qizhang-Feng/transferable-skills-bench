# Solution Note: Calculating Win Counts from Game Scores

The data was located in **Sheet1** with headers in **row 1** and player data starting from **row 2**.

**Structure:**
- Column A contained player names (Bill, Bob, Tim, Max, Chris)
- Columns B through G contained game scores (Game 1 through Game 6)
- Column H ("# of times scored highest") needed to be populated with win counts

**Approach:**

The solution required reading the score data from **columns B:G (rows 2-6)** for each player. For each game (each column from B to G), the logic identified which player had the maximum score in that column.

The Python/openpyxl approach involved:
1. Loading the workbook and accessing **Sheet1**
2. Reading the score matrix from **cells B2:G6** (5 players × 6 games)
3. For each game column (B through G), determining the row index with the highest score
4. Counting how many times each player (row) achieved the maximum score across all six games
5. Writing the calculated win counts to **column H, cells H2:H8**

The formula logic iterated through each game column, compared all player scores for that game, and incremented a counter for the winning player. This eliminated manual counting and provided an automated solution that could be recalculated if scores changed.
