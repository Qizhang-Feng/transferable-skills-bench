# Solution Note: Accumulated Wealth Calculation with Variable Deposits and Efficiency Rates

## Spreadsheet Structure
The data was located in **Sheet1** with headers in **row 1** and data starting from **row 2**.

## Column Layout
- **Column A (Year)**: Year identifiers (2020-2024)
- **Column B (Deposit)**: Annual deposit amounts in euros
- **Column C (Efficiency)**: Annual efficiency/interest rates as decimals
- **Column D (Inflation)**: Inflation rates (not used in initial calculation)
- **Column E (Efficiency amount)**: Interest earned per year
- **Column F (End amount)**: Deposit + efficiency amount for that year
- **Column H (Total ability)**: Cumulative wealth tracker

## Solution Approach
The task required calculating accumulated wealth in **column H (H2:H40)** where each year's total wealth equals the previous year's total wealth grown by the current efficiency rate, plus the current year's deposit grown by the current efficiency rate.

The Python/openpyxl solution:
1. Read values from columns B (Deposit), C (Efficiency), and F (End amount) starting at row 2
2. For H2: Set initial value equal to F2 (first year's end amount)
3. For H3 onwards: Apply formula `previous_total * (1 + current_efficiency) + current_deposit * (1 + current_efficiency)`
4. Handle empty rows by checking if Deposit (column B) is empty; if so, carry forward the previous total grown by efficiency
5. Write calculated values to column H (H2:H40)
