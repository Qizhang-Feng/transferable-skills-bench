# Solution Note: Identifying Latest Entry Time by Name

## Spreadsheet Details
The data was in sheet **'Лист1'** with headers in row 1 and data starting from row 2 through row 41.

## Task Requirements
The task required identifying the latest Entry Time (column I) for each unique Name (column C), then labeling records in column M ('Max') as either 'Latest' or 'Not Latest'.

## Python/openpyxl Approach

1. **Read the data**: Extracted values from column C ('Name', cells C2:C41) and column I ('Entry Time', cells I2:I41). Both columns contained the key information needed for comparison.

2. **Logic applied**: 
   - Grouped all Entry Time values by their corresponding Name
   - For each Name, determined the maximum (latest) Entry Time using datetime comparison
   - For each row, compared the current row's Entry Time against the maximum Entry Time for that Name
   - If the Entry Time matched the maximum, assigned 'Latest'; otherwise assigned 'Not Latest'

3. **Write results**: Wrote the computed labels to column M ('Max'), specifically cells M2:M41, matching each data row.

## Key Considerations
The Entry Time values in column I were datetime objects that required proper parsing for accurate comparison. Multiple rows could share the same Name but have different Entry Times, and all rows with the maximum Entry Time for a given Name were marked as 'Latest'.
