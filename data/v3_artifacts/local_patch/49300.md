# Solution Note: Two-Criteria Lookup with INDEX/MATCH

## Problem Summary
The task required retrieving salary values from a lookup table in the 'Data' sheet based on two criteria: Job Title (from column A) and Months (from column B) in 'Sheet1', with results written to column C.

## Spreadsheet Structure
- **Sheet1**: Headers in row 1 ('Job Title', 'Months', 'Salary'). Data started in row 2, with lookup criteria in columns A and B, and results needed in column C (cells C2:C3).
- **Data sheet**: A two-dimensional lookup table with months as column headers in row 2 (columns C through AY, representing months 1-48 and ">48"), job titles in column B (rows 3-4: 'BM' and 'DM'), and corresponding salary values in the table body (C3:AY4).

## Python/openpyxl Approach
The solution read values from 'Sheet1' columns A and B (starting row 2) to identify the job title and month criteria. For each row, the code:
1. Located the matching job title in 'Data'!B3:B4
2. Found the matching month value in 'Data'!C2:AY2
3. Retrieved the intersection value from the salary table (C3:AY4)
4. Wrote the numeric result to the corresponding cell in 'Sheet1' column C

The logic implemented a two-way lookup by matching row (job title) and column (months) indices, then extracting the value at their intersection point.
