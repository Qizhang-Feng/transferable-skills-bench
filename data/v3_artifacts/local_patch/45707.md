# Solution Note: Rolling Month Sum Formula Task

## Task Summary
The data was located in **Sheet1** with headers in row 1: 'Date' (column A), 'Occurence' (column C), and 'Rolling month sum' (column E). Data rows spanned from row 2 to row 69.

## Approach Used
The task required creating formulas in column D (D2:D69) that:
1. Checked if the next day (row+1) in column A was the 1st of a new month
2. If true, counted all occurrences of the value 1 in column C for that entire month/year
3. If false, left the cell empty

## Python/openpyxl Implementation
The solution read date values from **column A (cells A2:A69)** and occurrence values from **column C (cells C2:C69)** in **Sheet1**. 

For each row from 2 to 69:
- Extracted the current date from column A
- Calculated the next day's date
- Checked if the next day was the 1st of a month (day == 1)
- If true, filtered all dates in column A matching that month/year
- Counted how many times the value 1 appeared in column C for those matching dates
- Wrote the count to the corresponding cell in column D
- If false, left column D empty (no formula, no value)

The headers in row 1 (A1, C1, E1) were preserved unchanged. Column B and column D had no headers initially and remained that way.
