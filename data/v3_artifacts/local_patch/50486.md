# Solution Note: Conditional Pallet Type Display

The task required populating column A (PALLET TYPE) in Sheet1 based on which of two columns contained numeric values.

**Spreadsheet Structure:**
- Sheet name: Sheet1
- Header row: Row 1
- Data rows: Rows 2-5
- Column C header: 'CHEP'
- Column D header: 'LOSCAM'
- Target column: Column A (PALLET TYPE)

**Logic Applied:**
The solution checked columns C and D for each data row (rows 2-5). If column C contained a numeric value, "CHEP" was written to column A. If column D contained a numeric value, "LOSCAM" was written to column A. If neither column contained a numeric value, column A was left empty.

**Python/openpyxl Approach:**
1. Loaded the workbook and accessed Sheet1
2. Iterated through rows 2-5 (data rows after the header)
3. For each row, read the values from column C (CHEP) and column D (LOSCAM)
4. Applied conditional logic: checked if column C value was numeric (not empty string), then checked column D
5. Wrote the appropriate text ("CHEP" or "LOSCAM") to column A of the same row
6. Saved the workbook with the updated values in cells A2:A5

The solution handled empty cells and ensured only rows with numeric values in either column C or D received a pallet type designation.
