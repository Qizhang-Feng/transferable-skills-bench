# Solution Note: Grouping Sales by Product First Letter

The data was located in **Sheet1** with headers in Row 1. The product SKUs were in column A (**'Product'**) starting from cell A2, and their corresponding quantities were in column B (**'Quantity'**) starting from B2.

The task required grouping products by the first letter of their SKU (e.g., 'A' from 'A1', 'B' from 'B2') and summing the quantities for each group. The output headers ('A', 'B', 'C', 'D') were already present in cells D2:G2, and the summed results needed to be written to cells **D3:G3**.

**Python/openpyxl approach:**
1. Read product names from column A (cells A2:A6) and quantities from column B (cells B2:B6)
2. Extract the first character from each product SKU using string indexing
3. Create a dictionary to accumulate quantities by first letter (e.g., {'A': sum_of_A_products, 'B': sum_of_B_products, ...})
4. Iterate through the data rows, extracting the first letter and adding the corresponding quantity to the appropriate dictionary key
5. Read the header letters from cells D2:G2 to determine the output order
6. Write the aggregated sums to cells D3:G3, matching each sum to its corresponding letter header

The solution involved grouping and aggregation logic with dictionary-based accumulation, followed by writing results in the specified output range.
