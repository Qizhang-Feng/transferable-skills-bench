# Solution Note: Excel Macro Task - Sum and Delete Rows Based on Conditions

## Task Summary
The data was located in **Sheet1** with headers in **Row 1** starting at **A1:E1**: 'Date', 'Branch', 'Reference', 'Amount', 'Match Ref and Val'.

Data rows started from **Row 2** onwards, spanning columns **A through E**.

## Approach Used

**Step 1: Calculate Sum**
- Read **Column E ('Match Ref and Val')** to identify all rows containing zero values
- Sum the corresponding values from **Column D ('Amount')** for these zero-matching rows
- Locate the row where **Column C ('Reference')** contains the text 'BR1 Sales'
- Insert the calculated sum into **Column D** of that specific row

**Step 2: Delete Rows**
- Iterate through the dataset from bottom to top (to avoid index shifting issues)
- Delete rows where **Column E** equals zero, EXCEPT rows where **Column C** contains 'BR1 Sales'
- Remove any blank rows encountered during iteration

## Implementation Details
Using Python with openpyxl:
- Loaded the workbook and accessed **Sheet1**
- Iterated through rows starting from **Row 2** (first data row after headers)
- Applied conditional logic to check Column E values and Column C text content
- Used `worksheet.delete_rows()` for row removal operations
- Saved the modified workbook after all operations completed

The final result preserved the header row and the 'BR1 Sales' row with the calculated sum.
