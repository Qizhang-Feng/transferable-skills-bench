# Solution Note: Date Lookup with Multiple Conditions

The task required finding a specific date from sheet 'A' based on multiple conditions and pasting it into sheet 'B'.

**Sheet 'A' Structure:**
- Row 1 contained the header "Silver" in column F
- Row 2 contained column headers: "Date" in column F, "TOP" in column L, and "Record" in column P
- Data started from row 3
- Column F contained dates (e.g., 2014-02-03)
- Column L contained values including "100"
- Column P contained status values ("Yes", "NA", or "No")

**Logic Applied:**
1. First, scanned column P (starting from row 3) in sheet 'A' to find the first cell containing either "Yes" or "NA"
2. From that identified row, searched upward in column L to find the first occurrence of the value 100
3. Once found, retrieved the corresponding date from column F in that same row
4. Read the text from cell F1 in sheet 'A' (which was "Silver")
5. Searched column C in sheet 'B' (starting from row 2) to find a matching name
6. When the match was found in sheet 'B', pasted the identified date into column F of that matching row

**Sheet 'B' Structure:**
- Row 1 contained headers including "Name" in column C and "Last 100 Date" in column F
- Data started from row 2 with names in column C
