# Solution Note: VBA Dataset Management Task

## Task Overview
The task required processing data in the **Main** sheet with headers in row 1 and data starting from row 2. The dataset contained 13 columns (A through M) with company information, locations, employee details, and billing data.

## Specific Operations Performed

### 1. Row Deletions
- **Delete by Company**: Removed all rows where column A ('Company') contained 'Motorcycle'
- **Conditional Delete**: Removed rows where column A ('Company') was 'Ahmed Sons' AND column B ('Location') was 'Canada'

### 2. Data Updates
- **Bill Rate Modification**: Updated column G ('Bill Rate of Resource') to 180 for rows where column A ('Company') was 'National TV' AND column B ('Location') was 'India'

### 3. Row Insertions
- Inserted two new blank rows immediately after the header row (after row 1)

## Implementation Approach
The solution used openpyxl to:
1. Read the Main sheet starting from row 2
2. Iterate through rows in reverse order (to safely delete rows without index shifting)
3. Check conditions in columns A and B for deletion criteria
4. Apply updates to column G based on compound conditions in columns A and B
5. Insert new rows at position 2 using `insert_rows()` method

The **Lookup** sheet remained unchanged and was not part of the processing logic.
