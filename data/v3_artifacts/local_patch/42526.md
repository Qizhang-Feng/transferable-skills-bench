# Solution Note: Proportional Point Assignment with Zero Anchor

## Task Summary
The data was located in **Sheet1**, with values in **column A ('values')** starting from **row 2** (header in row 1). The scoring results needed to be written to **column B ('Scoring between two numbers(-5,+5)')** in the range **B2:B16**.

## Approach
The task required assigning points proportionately between -5 and +5, with a critical constraint: **zero must map to zero points**. This meant the positive and negative ranges needed separate scaling factors.

**Logic Applied:**
1. Read all numeric values from column A (rows 2-16)
2. Identified the maximum positive value (41) and minimum negative value (-18)
3. For **positive values**: Applied the formula `(value / 41) * 5` to scale proportionately from 0 to +5
4. For **negative values**: Applied the formula `(value / 18) * 5` to scale proportionately from 0 to -5
5. For **zero**: Assigned exactly 0 points

**Implementation:**
- Used openpyxl to load the workbook and access Sheet1
- Iterated through cells A2:A16 to read input values
- Applied the conditional scaling logic based on whether each value was positive, negative, or zero
- Wrote the calculated scores to corresponding cells in column B (B2:B16)
- Saved the workbook with the updated scores

This approach ensured that the point distribution was proportional within each range while maintaining zero as the anchor point.
