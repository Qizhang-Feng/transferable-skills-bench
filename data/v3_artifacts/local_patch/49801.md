# Solution Note: Concatenating LEFT and RIGHT Functions in Excel

## Task Summary
The task required combining the LEFT 2 characters and RIGHT 5 characters from a single cell value in Sheet1, cell A1, which contained 'SL-BK56-OWWFST-0035'. The result needed to be placed in cell C1.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Source Data**: Column A (cell A1 contained the full string)
- **Intermediate Results**: Column B showed LEFT result, Column C showed RIGHT result
- **Target Cell**: C1

## Python/openpyxl Approach
The solution read the value from cell A1 in Sheet1. The logic applied was:
1. Extract the leftmost 2 characters from the A1 value using Python string slicing `[:2]`
2. Extract the rightmost 5 characters from the A1 value using Python string slicing `[-5:]`
3. Concatenate these two extracted strings together
4. Write the concatenated result directly to cell C1

No header row was present in this spreadsheet; the data started at row 1. The operation was performed on a single cell rather than a range. The Python approach replicated the Excel formula `=LEFT(A1,2)&RIGHT(A1,5)` by using native string manipulation methods instead of Excel formulas, writing the final computed string value to the target cell.
