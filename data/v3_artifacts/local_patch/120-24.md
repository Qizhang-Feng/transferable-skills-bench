# Solution Note: Conditional Data Validation and Replacement in Excel

## Task Summary
The task required implementing conditional data replacement logic in Excel using multiple criteria across different columns.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Header Row**: Row 1
- **Data Range**: Rows 2-45

## Column Specifications
- **Column AY** (index 50): Contains FAH Parent Desc values to check for 'BOA_033E' or 'BOA_011G'
- **Column BG** (index 58): Contains Expected Result in BG - check for 'OCOGS - Spares - Transfer Price Overhead' or 'OFSS - ORCL Consulting Prod Cost I/C'
- **Column BL** (index 63): Contains GCOA Ledger - check if value equals 'LAG'
- **Column BN** (index 65): Contains COA_Child - target column for writing results

## Python/openpyxl Approach
The solution read data from Sheet1 starting at row 2. For each row in the range BN2:BN45, the logic evaluated three conditions:
1. Checked if column BL (GCOA Ledger) equals 'LAG'
2. Verified if column BG (Expected Result in BG) contains either of the two specified text strings
3. Confirmed if column AY (FAH Parent Desc) contains either 'BOA_033E' or 'BOA_011G'

When all three conditions were met, the appropriate replacement value was written to the corresponding cell in column BN (COA_Child). The openpyxl library was used to load the workbook, iterate through rows, apply conditional logic, and save the modified file.
