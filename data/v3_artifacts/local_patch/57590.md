# Solution Note: Monthly Sum with Date Matching

The task required calculating a sum in cell B26 of Sheet1, where values from Column K (Cost) were summed only when the corresponding date in Column C (Date) matched the month specified in cell A26.

**Spreadsheet Structure:**
- Sheet Name: Sheet1
- Header Row: Row 1
- Data Start Row: Row 2
- Key Columns:
  - Column C: "Date" (contains datetime values)
  - Column K: "Cost" (contains numeric values to sum)
  - Cell A26: Contains the target month/date for filtering

**Python/openpyxl Approach:**

The solution read the value from cell A26 to determine the target month. Then it iterated through the data rows starting from row 2, reading:
1. Column C (Date column) to extract each transaction date
2. Column K (Cost column) to get the corresponding cost values

For each row, the logic compared the month and year of the date in Column C against the month and year from A26. When they matched, the corresponding value from Column K was added to a running total.

The final sum was written to cell B26 using the `cell()` method.

This approach replaced the SUMIFS formula with Python logic that performed month-based date filtering and summation, handling the datetime comparison programmatically rather than through Excel formulas.
