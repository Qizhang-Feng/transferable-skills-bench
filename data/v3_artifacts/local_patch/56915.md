# Solution Note: Populating COVER Sheet from DATA Sheet with Entity-Based Lookup

## Task Summary
The task required populating cells B18:E23 on the **COVER** sheet with data from the **DATA** sheet, based on an entity selected in cell B12 of the COVER sheet. The column headers were located in row 13 of the COVER sheet, and row headers were in column A (rows 18-23).

## Technical Approach
The solution involved creating INDEX-MATCH formulas that:
1. Read the entity value from cell B12 on the COVER sheet
2. Located matching entity rows on the **DATA** sheet by searching for the entity identifier
3. Matched column headers from COVER!B13:E13 with corresponding column headers on the DATA sheet
4. Retrieved intersection values and rounded them to 2 decimal places using ROUND function

## Key Challenges
The primary issue was that merged cells on the DATA sheet caused standard lookup formulas to fail for columns C, D, and E. The solution required:
- Using OFFSET or array formulas to handle merged cell ranges
- Implementing column-specific logic to account for the merged cell structure
- Ensuring decimal format for columns C and E (as opposed to other formats)
- Applying ROUND function with 2 decimal places to all retrieved values

## Implementation
The formulas were written using openpyxl to dynamically construct INDEX-MATCH-ROUND combinations that adapted to the merged cell layout on the DATA sheet, ensuring accurate data retrieval across all four columns (B through E).
