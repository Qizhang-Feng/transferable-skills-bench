# Solution Note: Automatic Debt Calculation in Payroll Table

## Task Summary
The task required automatically updating the 'DEBT' column (column J) in the 'NOMINA' sheet based on weekly deductions and additions for each worker.

## Spreadsheet Structure
- **Sheet Name**: NOMINA
- **Header Row**: Row 1
- **Data Range**: Rows 2-47
- **Key Columns**:
  - Column C: 'WEEK' - week number identifier
  - Column D: 'WORKER' - employee name
  - Column G: 'DEDUCTIONS' - amount deducted from debt
  - Column H: 'ADDITIONS' - amount added to debt
  - Column J: 'DEBT' - running debt balance (target column)

## Solution Approach
The solution iterated through rows 2-47 in the NOMINA sheet. For each row, the logic:

1. Retrieved the current worker name from column D and week number from column C
2. Searched previous rows (in reverse order) to find the most recent debt value for the same worker from an earlier week
3. Retrieved the previous debt value from column J of that matching row (defaulting to 0 if no previous record existed)
4. Read the current row's DEDUCTIONS (column G) and ADDITIONS (column H) values
5. Calculated new debt as: `previous_debt - deductions + additions`
6. Wrote the calculated debt value to column J of the current row

This approach maintained a running debt balance per worker across weeks, properly handling deductions (reducing debt) and additions (increasing debt).
