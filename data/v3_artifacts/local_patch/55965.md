# Solution Note: Display Last 10 Results Based on ID Match

## Task Summary
The task required populating columns G through P on sheet **Foglio1** with the last 10 results from sheet **DATI**, matching IDs between the two sheets.

## Data Structure
- **Sheet DATI**: Contains historical results in column J (labeled 'TOT') with corresponding IDs in column K (labeled 'ID'). Data starts at row 2.
- **Sheet Foglio1**: Contains dates in column A, IDs in column E (labeled 'IG'), and columns G through P (labeled '10' through '1') where results needed to be displayed. Headers are in row 1, data starts at row 2.

## Solution Approach
The solution used Python with openpyxl to:

1. Read all data from sheet **DATI**, specifically columns J (TOT scores) and K (ID values) starting from row 2
2. For each row in sheet **Foglio1** starting from row 2:
   - Read the ID value from column E
   - Filter all records from **DATI** where column K matched this ID
   - Extract the last 10 TOT values (column J) from the filtered results, ordered by their appearance
   - Write these values into columns G through P (right to left, so column G gets the 10th most recent, column P gets the most recent)
3. If fewer than 10 matching results existed, only the available results were written, leaving remaining cells empty

The logic preserved chronological order based on row position in the DATI sheet.
