# Solution Note: Deleting Rows Between Pattern-Matched Part Numbers

The task required deleting rows in **Sheet1** starting from a part number matching the pattern `400-*-A` (where * is any characters) up to but not including the next part number matching `400-*` (any ending).

## Approach Used

The data was located in **Sheet1, Column A** (the first column). The spreadsheet had no header row, with data starting from **row 1**.

The solution implemented the following logic using Python and openpyxl:

1. **Read Column A** from Sheet1 to identify all part numbers
2. **Scan for the trigger pattern**: Found cells matching the regex pattern `^400-.*-A$` (starting with '400-', ending with '-A')
3. **Identify the end boundary**: After finding a `-A` pattern match, continued scanning for the next cell matching `^400-.*` (any part number starting with '400-')
4. **Delete the range**: Removed all rows from the `-A` match (inclusive) up to but not including the next `400-*` match
5. **Handle edge cases**: If no `-A` pattern existed, no deletions occurred; if no subsequent `400-*` pattern was found after `-A`, deleted to the end of data

The deletion was performed by iterating through the identified row indices and using the worksheet's `delete_rows()` method. The operation preserved all other data outside the identified range.
