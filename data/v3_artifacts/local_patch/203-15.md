# Solution Note: Staff Allowances Transpose Task

## Task Overview
The task required transforming staff allowance data from a vertical (long) format in the 'Main' sheet to a horizontal (wide) format in the 'Output Required' sheet.

## Data Structure
**Source Sheet: 'Main'**
- Headers in row 1, columns A through J
- Staff information columns: A ('staff no.'), B ('Department'), C ('Location'), D ('Name'), E ('Position'), F ('Gender'), G ('Nationality')
- Column H contained 'Description' (allowance types: 'Basic', 'Housing Allowance', 'Transport Allowances', 'Other Allowances', 'Social Allowances')
- Column I contained 'Amount' values
- Data started from row 2

**Target Sheet: 'Output Required'**
- Headers in row 1 starting at A1
- First 7 columns (A-G) copied the staff information headers from Main sheet (excluding 'Nationality')
- Columns H onwards contained the transposed allowance types as headers
- Row 2 contained the unique staff member's information
- Row 3 contained the corresponding allowance amounts

## Approach
The solution read unique staff records from columns A-F of the 'Main' sheet, extracted unique allowance descriptions from column H, and pivoted the data so each allowance type became a column header. The corresponding amounts from column I were placed in row 3 under their respective allowance headers. The output structure placed staff info in columns A-F and allowance amounts in subsequent columns.
