# Solution Note: Extracting Numerical Values from Currency Text Strings

## Task Overview
The data was located in **Sheet1** with headers in **Row 1**. The source data was in column **'Amount'** (column B, cells **B2:B5**), and the results needed to be written to column **'I need: '** (column C, cells **C2:C5**).

## Approach Used
The task required extracting numerical values from mixed-format currency strings that contained:
- Currency symbols in various positions (prefix/suffix)
- Spaces as thousands separators
- Commas as decimal separators (European format)

## Python/openpyxl Implementation
1. **Read** the workbook and accessed **Sheet1**
2. **Iterated** through cells **B2:B5** in the **'Amount'** column
3. **Applied** text processing logic to each cell:
   - Removed currency codes (USD, EUR) using regex
   - Stripped whitespace
   - Identified the decimal separator by finding the last comma in the string
   - Removed all spaces (thousands separators)
   - Replaced the comma decimal separator with a period
   - Converted the cleaned string to a float value
4. **Wrote** the extracted numerical values to the corresponding cells in column **'I need: '** (**C2:C5**)
5. **Saved** the workbook with the populated results

The solution handled negative values (indicated by minus signs) and various currency symbol positions while correctly interpreting European number formatting conventions.
