# Solution Note: Transpose Runner Odds with Conditional Logic

The data was in sheet **Taul1** with headers in row 1 and data starting from row 2.

The task required transposing values from **column G (Odds)** to columns **H through S (labeled #1 through #12)** for rows 2-28, based on conditional logic:
- Only rows where **column B (Runner)** equals `1` contained the source data
- **Column C (Runners)** in those rows specified how many consecutive values to pull from column G
- The formula needed to read sequential rows in column G and distribute them horizontally

The Python/openpyxl approach implemented:
1. Read **column B (Runner)** to identify rows where value equals `1`
2. Read **column C (Runners)** from those identified rows to determine how many values to transpose
3. Read consecutive values from **column G (Odds)** starting from the current row
4. Write the transposed values horizontally across columns **H through S** (range H2:S28)
5. Preserve blank cells as blanks (not converting to zeros)
6. Apply cell formatting: shade range **H2:S28** with fill color **#E2EFD9**
7. Format numeric values to remove decimal places for whole numbers

The solution iterated through each row in the target range, checking the Runner column condition, then pulling the appropriate number of Odds values and placing them sequentially in the horizontal output columns.
