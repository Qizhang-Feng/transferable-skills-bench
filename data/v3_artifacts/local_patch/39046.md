# Solution Note: Counting Subreferences for Cancelled Items

The task was solved using Python with openpyxl to process data in **Sheet1**.

**Data Structure:**
- Header row: Row 1
- Data rows: Rows 2-6
- Column B ('REF'): Parent reference codes
- Column C ('STATUS'): Status values (checking for 'CANCELLED')
- Column D ('SUBREFS'): Subreference codes to analyze
- Column E ('Subs?'): Output column for Yes/No indicators
- Column F ('No. Subs'): Output column for count of matches

**Logic Applied:**

For each row (2-6), the solution:
1. Read the reference value from column B and status from column C
2. Checked if the status in column C was 'CANCELLED'
3. If cancelled, searched through ALL values in column D (rows 2-6) to find cells containing the reference from column B as a substring
4. Wrote 'Yes' to column E if at least one match was found, otherwise 'No'
5. Counted the total number of matches and wrote this count to column F

**Key Implementation Details:**
- The substring matching used Python's `in` operator to check if the parent reference appeared within any subreference string
- The search was performed across the entire column D range (D2:D6) for each parent reference
- Empty cells in column D were handled by treating them as empty strings
- Results were written to ranges E2:E6 and F2:F6
