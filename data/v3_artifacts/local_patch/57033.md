# 3-Way Match Solution Note

## Task Summary
The task required performing a 3-way match between **Sheet4** and **CBtrans** sheet, writing results to cells K2:K7 in **Sheet4**, then applying regular casing and hex color #FF66CC to those cells.

## Data Structure
- **Sheet4** contained headers in row 1 with data starting in row 2
- Key columns: Company (column B), account (column I), xchar (column E), Year (column J)
- Target column: Result (column K, cells K2:K7)
- **CBtrans** contained headers in row 1 with data starting in row 2
- Matching columns: company (column A), account (column H), xchar (column K), yearno (column E)

## Solution Approach
The Python/openpyxl solution read data from both sheets starting at row 2. For each row in Sheet4 (rows 2-7), the logic compared:
1. Company value (Sheet4 column B) against company (CBtrans column A)
2. account value (Sheet4 column I) against account (CBtrans column H)
3. xchar value (Sheet4 column E) against xchar (CBtrans column K)
4. Year value (Sheet4 column J) against yearno (CBtrans column E)

When all four criteria matched for any row in CBtrans, "MATCH" was written to the corresponding cell in column K of Sheet4. Otherwise, a dash "-" was written.

After populating K2:K7, the text was converted to regular casing (title case) and cells were formatted with fill color #FF66CC using openpyxl's PatternFill with RGB values (255, 102, 204).
