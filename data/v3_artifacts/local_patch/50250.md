# Solution Note: Conditional Sum Based on Reference Value

The task required summing numeric values from columns C, D, and E in **Sheet1** where the corresponding row in columns A or B matched the reference value in cell **G2** ('sela').

**Data Structure:**
- Sheet name: **Sheet1**
- Header row: Row 1
- Data rows: Rows 1-6
- Reference value location: Cell **G2**
- Lookup columns: **Column A** and **Column B**
- Value columns to sum: **Column C**, **Column D**, and **Column E**
- Output location: Cell **H2**

**Python/openpyxl Approach:**

1. Read the reference value from cell **G2** in **Sheet1**
2. Iterate through rows 1-6 of the spreadsheet
3. For each row, check if either cell in **column A** or **column B** matched the reference value ('sela')
4. When a match was found, examine the corresponding cells in **columns C, D, and E**
5. For each cell in columns C-E, verify if the value was numeric (integer or float type)
6. Sum only the numeric values from matching rows
7. Write the calculated sum to cell **H2**

The solution filtered rows based on the presence of 'sela' in columns A or B, then conditionally summed only numeric entries from the corresponding positions in columns C through E.
