# Solution Note: Summing Truck Quantities with Concatenated Truck Names

## Problem Context
The task required summing truck quantities from the 'Data' sheet where truck numbers were listed in separate rows, but matching them against concatenated truck numbers in the 'Report' sheet.

## Data Structure
- **Data Sheet**: Headers in row 2 ('Date', 'Supplier ', 'Truck', 'Quantity '). Data started in row 3. Column C contained individual truck numbers, column D contained quantities.
- **Report Sheet**: Headers in row 2 ('Date', 'Supplier ', 'Truck', 'Quantity '). Data started in row 3. Column C contained concatenated truck numbers (comma-separated), column D (D3:D10) required the summed quantities.

## Python/openpyxl Approach
The solution read data from the 'Data' sheet starting at row 3, extracting truck numbers from column C and quantities from column D into a dictionary mapping each truck to its quantity.

For each row in the 'Report' sheet (starting at row 3), the concatenated truck string from column C was parsed by splitting on commas and stripping whitespace. Each individual truck number was looked up in the dictionary, and matching quantities were summed.

The calculated sum was written to column D of the 'Report' sheet at the corresponding row position (D3:D10).

This approach handled the mismatch between individual truck entries in 'Data' and concatenated truck lists in 'Report', enabling accurate quantity aggregation across multiple trucks per supplier/date combination.
