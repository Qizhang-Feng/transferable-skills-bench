# Solution Note: Summing Quantities by Model Across Years

The data was located in **Sheet1** with headers in **row 3**. The source data included:
- Column B: Year values (starting row 4)
- Column C: Model names (range C4:C11)
- Column D: Quantity values (range D4:D11)

The lookup values were in column F (F4:F6 containing model names: Honda, Suzuki, Mitsubishi), and the results needed to be written to column G (G4:G6).

## Python/openpyxl Approach

The solution required reading data from **Sheet1** starting at row 4 (first data row after headers in row 3). The logic involved:

1. Reading the Model column (column C, cells C4:C11) and Qty column (column D, cells D4:D11) from the source data range
2. Reading the lookup model names from column F (cells F4:F6)
3. For each model in column F, iterating through all rows in the source data (C4:C11) to find matching model names
4. Summing all quantity values from column D where the corresponding model in column C matched the lookup model
5. Writing the aggregated sum for each model to the corresponding cell in column G (G4:G6)

This approach used SUMIF-like logic to aggregate quantities across multiple rows with the same model name, regardless of year values.
