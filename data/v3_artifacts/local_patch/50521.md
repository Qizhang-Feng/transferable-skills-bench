# Solution Note: Counting Blank Cells Between First and Second Numeric Entries

The data was located in **Sheet1** with headers in row 3 starting from column B (2020-01-01) through column M (2020-12-01). Customer names were in column A, with data rows starting at row 4.

The task required counting blank cells between the first and second numeric entries in each row. For each customer row (rows 4-6), the formula needed to:

1. Identify the position of the first numeric entry across columns B through M
2. Identify the position of the second numeric entry
3. Count the blank cells between these two positions (excluding the positions themselves)

The Python/openpyxl approach involved:
- Reading data from **Sheet1**, rows 4-6, columns B through M (B4:M6)
- For each row, iterating through the cells to find the first non-empty cell position
- Continuing iteration to find the second non-empty cell position
- Calculating the count of blank cells between these two positions using: `(second_position - first_position - 1)`
- Writing the results to column N, cells **N4:N6**

The logic handled cases where rows might have fewer than two numeric entries by returning 0 or an appropriate value. The solution processed each customer row independently and wrote the calculated blank cell count to the corresponding cell in column N.
