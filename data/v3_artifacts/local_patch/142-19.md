# Solution Note: Sorting Non-Blank Ranges in Sheet1

The data was located in **Sheet 'Sheet1'** with headers in **Row 1**. Column A contained 'Number' values (format: letter + space + number, e.g., 'Z 12', 'C 14') and Column B contained 'Name' values (e.g., 'John', 'Mary'). Data started from **Row 2** and extended through the sheet with blank rows interspersed throughout.

## Approach

The task required identifying and sorting each contiguous non-blank range independently while maintaining the relationship between Column A and Column B values.

**Steps taken:**

1. Read all data from **columns A and B** starting from **Row 2** in **Sheet1**
2. Identified contiguous non-blank ranges by iterating through rows and detecting where both Column A and Column B had values
3. For each identified non-blank range, extracted the data as paired tuples (Number, Name)
4. Sorted each range alphabetically based on Column A values (the 'Number' column) using Python's sort function
5. Wrote the sorted data back to the original cell ranges in **columns A and B**, preserving the blank row separations between different data blocks
6. Left blank rows unchanged to maintain the original structure

The solution preserved data integrity by keeping each name paired with its corresponding number throughout the sorting process, and each non-blank section was sorted independently without affecting other sections.
