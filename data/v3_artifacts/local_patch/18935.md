# Solution Note: Multi-Criteria Lookup from Data Table to Report Format

## Task Summary
The task required performing a multi-criteria lookup to retrieve Category values from a Data Table and populate cells D17:D22 in the report section of Sheet1.

## Data Structure
**Data Table Location (Sheet1):**
- Headers in Row 4: 'Work', 'Material', 'Category 1', 'Category 2', 'Category 3'
- Data started from Row 5 onwards
- Column A contained Work criteria (values like '1', '2', etc.)
- Column B contained Material types ('Material 1', 'Material 2', etc.)
- Columns C, D, E contained Category 1, Category 2, and Category 3 values respectively

**Report Section:**
- Target cells for output: D17:D22
- The report section contained three lookup criteria per row:
  - Work Criteria (column reference in report)
  - Category Type (which category column to retrieve: 1, 2, or 3)
  - Material (material name to match)

## Solution Approach
The solution used openpyxl to:
1. Read the Data Table from Row 5 onwards (columns A through E)
2. For each target cell in D17:D22, extract the three criteria from the corresponding row
3. Filter the Data Table where Work matched criterion 1 AND Material matched criterion 3
4. Based on Category Type (criterion 2), select the appropriate category column (C for Category 1, D for Category 2, E for Category 3)
5. Write the retrieved value to the target cell
