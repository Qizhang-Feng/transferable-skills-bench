# Solution Note: Counting Countries per Company

The task required counting how many countries each company operates in (with >0 employees) and placing results in cells I12:I13 on Sheet1.

**Data Structure:**
- The spreadsheet contained two tables on Sheet1
- Table 1 (Companies): Located with company names that needed country counts
- Table 2 (Country-Company Data): Listed countries in one column with corresponding company employee counts in adjacent columns
- Header row was at Row 1, with country names starting from Row 2 (countries "B" and "F" were visible)

**Python/openpyxl Approach:**
1. Read the country list from Table 2 (starting from a specific column, likely column A or B, from row 2 downward)
2. Read the employee count columns for each company (ABC and others) from Table 2, corresponding to each country row
3. For each company, iterate through their employee count column and count how many cells contain values greater than 0
4. Write the count result to cell I12 for the first company (ABC) and I13 for the second company

**Logic Applied:**
- Used conditional counting: for each company's column in Table 2, count rows where the value > 0
- Matched this count against the countries listed in Table 1
- Wrote the final count values to the specified cells in column I
