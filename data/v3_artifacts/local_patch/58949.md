# Solution Note: Pivoting Client Data from Rows to Columns

The task required transforming normalized data from sheet **'Extract'** into a pivoted format in sheet **'Desired Result'**.

## Source Data
The data was located in sheet **'Extract'** with headers in row 1:
- Column A: **'Person'** (client identifier)
- Column B: **'Car'** (item/detail name)
- Column C: **'Value'** (associated value)

Data rows started at row 2 and continued through row 5.

## Transformation Logic
The solution used Python with openpyxl to:
1. Read all data from columns A, B, and C in sheet **'Extract'** starting from row 2
2. Collect all unique values from column B (**'Car'**) to form the header row
3. Group data by column A (**'Person'**), creating a dictionary mapping each person to their car-value pairs
4. Build a pivot structure where each unique person appeared once, with their values aligned horizontally under the corresponding car column headers

## Output Location
The results were written to sheet **'Desired Result'**:
- Row 1: Header row with 'Person' in column A (B1), followed by all unique car names across columns B through I
- Rows 2-5: Each unique person in column A, with their corresponding values placed in the appropriate car columns (B2:I5)
- Empty strings filled cells where a person had no value for a particular car
