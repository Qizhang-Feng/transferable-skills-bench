# Solution Note: Sequential Numbering Based on Conditional Logic

The task was completed on the sheet named **"Sub CO's"**. The data structure had headers in **row 3** and data starting from **row 4**.

The objective was to populate column **G** (labeled "PCO #") with a sequential numbering system that incremented only when the corresponding cell in column **F** (labeled "Pass on to owner?") contained "Yes" (case-insensitive). When column F did not contain "Yes", column G should display 0.

The Python/openpyxl approach involved:

1. **Reading column F** starting from row 4 onwards to check each cell's value
2. **Maintaining a counter variable** initialized at 0
3. **Iterating through each row** in the data range (rows 4-10 in this case)
4. **Applying conditional logic**: 
   - If the cell in column F contained "Yes" (case-insensitive comparison), increment the counter and write the counter value to the corresponding cell in column G
   - If the cell in column F did not contain "Yes", write 0 to the corresponding cell in column G
5. **Writing results to column G** (cells G4:G10)

The solution required tracking state across rows (the running sequence number) rather than using independent cell formulas, which was why nested IF functions alone could not solve this problem. The sequential counter only advanced when the condition was met, creating a cumulative numbering system.
