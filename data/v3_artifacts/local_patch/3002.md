# Solution Note: Identifying Duplicate Mat Doc and Doc No Combinations

The task involved analyzing data in **Sheet1** to identify duplicate combinations of Mat Doc and Doc No values, then populating a Remark column and removing formatting.

## Data Structure
- **Header row**: Row 4
- **Data columns**: 
  - Column E: "Mat Doc" 
  - Column F: "Doc No"
  - Column G: "Remark" (output column)
- **Data range**: Rows 5-15 (E5:F15 for input data)
- **Output range**: G5:G15

## Solution Approach
The solution used Python with openpyxl to:

1. **Read the data**: Extracted values from columns E (Mat Doc) and F (Doc No) starting from row 5 through row 15 in Sheet1

2. **Identify duplicates**: Created a counter to track occurrences of each (Mat Doc, Doc No) combination across the dataset

3. **Apply logic**: For each row, checked if the combination appeared more than once in the dataset. If yes, marked as "Multiple" in column G; otherwise, left blank or marked as single occurrence

4. **Write results**: Populated the Remark column (G5:G15) with the appropriate values based on the duplicate analysis

5. **Remove formatting**: Cleared all cell colors (fill patterns) and borders from the output range G5:G15 to ensure clean presentation

The approach handled the modified Remark column structure while maintaining the duplicate detection logic.
