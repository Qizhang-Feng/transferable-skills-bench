# Solution Note: Deleting Rows Based on Group Filter

The task required filtering and deleting rows from **Sheet1** where column A ('Group') contained specific values: '@9T', 'SAL', or 'T9A'.

**Spreadsheet Structure:**
- Sheet name: **Sheet1**
- Header row: Row 1 with columns ['Group', 'ID', 'Location', 'Pay Level']
- Data started at row 2
- Column A contained the 'Group' values to filter
- The example showed 207 rows, but the actual dataset contained over 13,000 rows

**Python/openpyxl Approach:**

The solution read data from **column A ('Group')** starting from row 2 in **Sheet1**. The logic identified all rows where the Group value matched '@9T', 'SAL', or 'T9A'. 

To handle large datasets efficiently without excessive processing time, the approach collected row indices to delete in reverse order (from bottom to top). This prevented index shifting issues that occur when deleting rows sequentially from top to bottom.

The implementation iterated through the worksheet rows, checked each value in **column A**, and marked matching rows for deletion. Rows were then deleted in descending order using `ws.delete_rows()`.

This reverse-deletion method ensured the operation completed efficiently even with 13,000+ rows, as it avoided re-indexing overhead and maintained O(n) time complexity rather than O(n²) that would result from forward deletion.
