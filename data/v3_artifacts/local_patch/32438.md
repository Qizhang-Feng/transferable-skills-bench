# Solution Note: Extracting Time from DateTime in Column J

The task required extracting only the time portion from datetime values in column I and displaying them in 12-hour format with AM/PM in column J of Sheet1.

**Data Structure:**
- Sheet name: `Sheet1`
- Header row: Row 1
- Data rows: Rows 2-4
- Source column: Column I (`Time`) containing datetime values in format 'YYYY-MM-DD HH:MM:SS'
- Target column: Column J (column index 9, zero-based) for time-only output

**Python/openpyxl Approach:**

The solution read datetime values from column I (cells I2:I4) in Sheet1. Since the RIGHT() function approach returned strings without proper time formatting, the Python solution parsed each datetime value from column I and extracted only the time component.

For each data row (rows 2-4), the code:
1. Read the datetime value from column I
2. Extracted the time portion using Python's datetime parsing
3. Formatted the time as a 12-hour format string with AM/PM (e.g., "06:08:00 PM")
4. Wrote the formatted time string to the corresponding cell in column J

The cells in column J were populated with properly formatted time strings in 12-hour notation with AM/PM indicators, solving the issue where Excel's RIGHT() function returned unformatted numeric strings.
