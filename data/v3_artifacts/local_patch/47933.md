# Solution Note: IF Formula Correction for Hours-Based Rate Multiplier

## Task Summary
The task required correcting an IF formula in column E ('Sum') of the 'master' sheet to calculate values based on hours worked. The user's original formula had syntax errors including misplaced quotes, incorrect IF statement structure, and improper concatenation.

## Spreadsheet Structure
- **Sheet**: 'master'
- **Header Row**: Row 1
- **Data Range**: Rows 2-154
- **Input Columns**:
  - Column B ('Hours'): Contains hours worked values
  - Column D ('Rate'): Contains rate values
- **Output Column**: Column E ('Sum')

## Logic Applied
The formula needed to evaluate the hours in column B (B2, B3, etc.) and apply conditional multiplication to the rate in column D (D2, D3, etc.):
- If hours < 37.5: multiply rate by 2
- If hours ≥ 37.5: multiply rate by 1.5

## Python/openpyxl Approach
The solution read values from columns B ('Hours') and D ('Rate') starting at row 2 through row 154 in the 'master' sheet. For each row, the logic compared the hours value against 37.5 and calculated the result by multiplying the rate by either 2 or 1.5 accordingly. The computed results were written to column E ('Sum') in the corresponding rows.

The corrected Excel formula syntax was: `=IF(B2<37.5,D2*2,D2*1.5)`
