# Solution Note: SUMPRODUCT Formula Fix for Every 4th Column Sum

## Problem Analysis
The task was to fix a SUMPRODUCT formula in cell C15 of sheet **DATI** that sums every 4th column based on a criterion in column A.

## Spreadsheet Structure
- **Sheet Name**: DATI
- **Header Row**: Row 2 (empty in this case)
- **Data Start Row**: Row 3
- **ID Column**: Column A (A3:A10) - contains identifiers like '12345', '98321', '9449', '11341'
- **Category Column**: Column B (B3:B10) - contains categories 'AA', 'BB', 'CC', 'DD'
- **Data Range**: Columns C through N (C3:N10) - contains numeric values to be summed
- **Criteria Cell**: B15 - contains the lookup value to match against column A
- **Result Cell**: C15 - where the corrected formula result was written

## Python/openpyxl Approach
The solution involved reading the identifier values from column A (rows 3-10) and the data range from columns C through N (rows 3-10). The logic applied was to identify every 4th column starting from column C (columns C, G, K) and sum values from those columns only for rows where column A matched the criterion in B15. The result was written to cell C15 using openpyxl's cell assignment method. The formula issue stemmed from incorrect MOD logic that needed to be replaced with direct column index checking.
