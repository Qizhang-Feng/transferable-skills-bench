# Solution Note: Transposing Grouped Rows to Columns

The data was located in **Sheet1**, column **A** ('name'), starting from **row 1**. The dataset contained groups of 3 consecutive rows that needed to be transposed into columnar format.

## Approach

The task required reading data from **column A** in **Sheet1**, where every 3 rows represented a single record:
- Row 1 of each group: name information
- Row 2 of each group: address information  
- Row 3 of each group: city/state/zip information

The Python/openpyxl solution involved:

1. Reading all values from **column A** starting at **row 1** through the end of the data (21+ rows)
2. Iterating through the data in groups of 3 rows using a loop with step size of 3
3. For each group of 3 rows:
   - The first row value was written to **column D** of the output
   - The second row value was written to **column E** of the output
   - The third row value was written to **column F** of the output
4. Each group occupied a new row in the output range, starting at **row 1**

The output was written to the range **D1:F7** in **Sheet1**, where 7 rows represented 7 complete records (21 source rows ÷ 3 rows per group = 7 records). No headers were added to the output columns.
