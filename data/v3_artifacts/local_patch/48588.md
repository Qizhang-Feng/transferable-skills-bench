# Solution Note: Generating Unique IDs for Products

## Task Summary
The data was in sheet **'Sheet1'** with headers in row 1. Column A ('Product') contained product names starting from row 2, with multiple occurrences of the same products throughout the list (rows 2-43). Column B ('Unique ID (Expected Result)') needed to contain sequential unique numbers for each distinct product.

## Python/openpyxl Approach

The solution read product names from **column A ('Product')** in sheet **'Sheet1'**, starting from **row 2** (first data row after the header in row 1).

The logic applied was:
1. Loaded the workbook and accessed sheet 'Sheet1'
2. Iterated through cells in column A from row 2 onwards
3. Maintained a dictionary to track which products had been encountered and their assigned unique ID
4. For each product in column A:
   - If the product was new (not in dictionary), assigned it the next sequential number and stored it in the dictionary
   - If the product was already seen, retrieved its existing unique ID from the dictionary
5. Wrote the corresponding unique ID to **column B ('Unique ID (Expected Result)')** in the same row

This approach ensured that each unique product received a consistent ID across all its occurrences, with IDs assigned in the order products were first encountered in the list.
