# Solution Note: Text-Based Date Range to Days Calculation

The task involved processing text-based date ranges in **Sheet1** to calculate the number of days between dates.

## Spreadsheet Structure
- **Sheet name**: Sheet1
- **Header row**: Row 1 with columns ['Count', 'No. of days']
- **Input column**: Column A ('Count') starting from cell A2
- **Output column**: Column B ('No. of days') starting from cell B2
- **Data range**: The input data was in cell A2 containing the text '2 to 5'

## Python/openpyxl Approach
The solution read the text value from cell A2 in column A ('Count'). The text format '2 to 5' represented a date range (e.g., June 2 to June 5).

The logic applied was:
1. Parse the text string from cell A2 to extract the start and end day numbers
2. Split the string using ' to ' as the delimiter
3. Convert the extracted strings to integers (start day and end day)
4. Calculate the number of days as: (end day - start day) + 1
5. The "+1" was included because the range is inclusive (both start and end days count)

The calculated result was written to cell B2 in column B ('No. of days') of Sheet1. The workbook was then saved to preserve the calculated value.
