# Solution Note: Cleaning Non-Numeric Data in Range

The task involved cleaning alphanumeric and wildcard character contents from cells that should contain only numeric values in the **'Data'** sheet.

## Data Location
- **Sheet**: 'Data'
- **Target Range**: U3:AI20 (columns U through AI, rows 3-20)
- **Header Row**: Row 2 contained column headers including 'Opening Stock', 'Total Sales', 'Bulk Cancelled Sales', 'Total Loans done', 'Leads Received', etc.
- **Data Start Row**: Row 3 (first data row below headers)

## Approach
The solution used Python with openpyxl to:

1. Load the workbook and access the 'Data' sheet
2. Iterate through each cell in the range U3:AI20
3. For each cell, check if the value was numeric (integer or float)
4. If the cell contained non-numeric data (alphanumeric strings, special characters, wildcards like '¢±¹', 'ªâ>', 'á&v', etc.), replace it with 0
5. If the cell was already numeric or empty/None, leave it unchanged
6. Write the cleaned values back to the same cells in the range U3:AI20

The logic applied a simple conditional: if `isinstance(cell.value, (int, float))` was False and the value was not None, the cell was set to 0. This effectively replaced all non-numeric entries with zeros while preserving any existing numeric values.
