# Solution Note: Auto-Update Category Based on Age Lookup

## Task Summary
The task required creating a formula in column E ('Expected Result') of Sheet1 that automatically updates category values based on age calculations, with the ability to adjust when the report date in cell B1 changes.

## Spreadsheet Structure
- **Sheet Name**: Sheet1
- **Report Date**: Cell B1 (format: YYYY-MM-DD HH:MM:SS)
- **Data Range**: Rows 3-15
- **Header Row**: Row 2

## Key Columns
- Column C ('DATE OF BIRTH'): Birth dates for each person
- Column E ('Expected Result'): Target column for formula output
- Column H ('YEAR (age)'): Lookup reference ages (6, 7, 8, 9, etc.)
- Column I ('CATEGORY'): Corresponding categories (A, B, etc.)
- Column O ('age (year)'): Calculated age values used for matching

## Python/openpyxl Approach
The solution read the report date from cell B1 and birth dates from column C (C3:C15). For each row, the age was calculated by subtracting the birth year from the report date year. This calculated age was then used to perform a lookup against the reference table in columns H and I (starting from row 3) to find the matching category. The VLOOKUP-style logic matched the age value in column O against column H to retrieve the corresponding category from column I. The resulting category values were written to column E (E3:E15). The formula dynamically referenced cell B1, ensuring automatic updates when the report date changed.
