# Solution Note: Summing Part Number Quantities Across Multiple Columns

The task required creating a formula in cell M2 of sheet "PART LIST (2)" that sums quantities for the part number listed in cell A2 ("50303-TY2") across multiple occurrences in columns O onwards.

## Spreadsheet Structure
- **Sheet Name**: "PART LIST (2)"
- **Header Row**: Row 1 contained part number headers starting from column O (index 14)
- **Data Start Row**: Row 2
- **Target Cell**: M2 (column index 12, row 2)
- **Part Number Location**: Cell A2 (column index 0, row 2)

## Python/openpyxl Approach
The solution involved reading the part number from cell A2, then searching through columns O onwards (starting at column index 14) in row 1 to find all matching part number headers. The data structure showed part numbers in odd-indexed columns (O, Q, S, U, etc.) with corresponding quantities in the adjacent even-indexed columns (P, R, T, V, etc.).

The logic iterated through all columns from index 14 to the end of row 1, checking if each cell's value matched the part number from A2. When a match was found, the corresponding quantity value from row 2 in the next column (column index + 1) was added to a running total.

The final SUMIF-style formula or calculated sum was written to cell M2, aggregating all quantity values where the part number appeared in the header row.
