# Solution Note: Conditional Text Formula Based on Two Cell Values

The task required creating conditional logic in cells D1, D5, and D9 on **Sheet1** based on the text values in columns B and C of their respective rows.

## Spreadsheet Structure
- **Sheet**: Sheet1
- **Input cells**: Column B (cell B1, B5, B9) and Column C (cell C1, C5, C9)
- **Output cells**: Column D (cell D1, D5, D9)
- The pattern repeated every 4-5 rows with "Input:" labels in column A

## Logic Applied
The formula evaluated two conditions:
1. If column B contained "Description A" AND column C was blank → output "Single A"
2. If column B contained "Description B" AND column C was blank → output "Single B"
3. If column B contained "Description A" AND column C contained "Description B" → output "Multiple"

## Python/openpyxl Approach
The solution read values from column B and column C for rows 1, 5, and 9. For each target cell (D1, D5, D9), nested IF logic was implemented using openpyxl to write formulas that checked:
- First condition: B cell equals "Description A" with C cell empty
- Second condition: B cell equals "Description B" with C cell empty
- Third condition: B cell equals "Description A" AND C cell equals "Description B"

The formulas were written directly to cells D1, D5, and D9 using the `cell.value` property with Excel IF and AND functions.
