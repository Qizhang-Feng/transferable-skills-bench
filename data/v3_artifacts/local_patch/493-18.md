# Solution Note: Filtering and Cleaning Car Data Based on Filter List

The task was completed on **Sheet1** with headers in **Row 1** and data starting from **Row 2**.

**Data Structure:**
- Column A ('Car'), Column B ('Mileage(Miles)'), and Column C ('Colour') contained the main dataset
- Column F ('Filter_List') contained the filter criteria (cars to keep)

**Approach:**

1. **Read Filter List**: The filter values were read from **Column F (rows 2-6)** to create a set of cars to retain (Lamborghini, Audi, Porsche, Ferrari).

2. **Clear Non-Matching Rows**: Iterated through **Column A (rows 2-6)** and checked if each car name existed in the filter list. For rows where the car did NOT match any value in Column F, the cells in **Columns A, B, and C** for that row were cleared (set to None or empty string). Column F remained untouched.

3. **Compact Data**: After clearing, all non-empty rows in Columns A-C were collected and shifted upward starting from **Row 2**, eliminating gaps left by cleared rows.

4. **Preserve Autofilters**: The existing autofilter settings on Columns A, B, and C were retained by not modifying the worksheet's `auto_filter` property.

The final result had matching cars (Audi, Ferrari, Porsche) compacted in rows 2-4, with Column F unchanged.
