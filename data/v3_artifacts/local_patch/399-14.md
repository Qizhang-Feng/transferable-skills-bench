# Solution Note: Employee Hours Split (Regular vs Overtime)

## Task Summary
The data was located in **Sheet1** with headers in **Row 6**. Employee hours were recorded in **Column N ('REG')** starting from **Row 7**. The task required splitting hours into regular (≤40) and overtime (>40) hours.

## Approach Used
1. **Read Data**: Read values from Column N (REG hours) in Sheet1, starting from Row 7 through Row 23 (the data range N7:O23 was specified as the answer position).

2. **Logic Applied**: 
   - For each cell in Column N: if the value was ≤40, it remained unchanged in Column N and Column O was left empty or set to 0
   - If the value was >40, Column N was set to 40 and the excess (value - 40) was written to the corresponding cell in Column O (OT column)

3. **Write Results**: 
   - Updated values were written back to Column N (REG) for regular hours
   - Overtime hours were written to Column O (OT)
   - Both columns were in the range N7:O23

4. **Formatting**: The entire sheet was formatted using the '等线' (Dengxian) font family.

## Implementation
Used Python with openpyxl library to iterate through cells N7:N23, apply the conditional logic, and write results to both Column N and Column O accordingly.
