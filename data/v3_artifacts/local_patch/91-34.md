# Solution Note: Delete Duplicate Non-Employee Entries

The task required deleting duplicate entries from the **'SwiftMD'** sheet based on specific criteria. The data started at **row 3** (with headers in **row 2**) and extended through **row 42**.

## Approach

The solution read the following columns from sheet **'SwiftMD'**:
- **Column C**: 'Last Name'
- **Column D**: 'First Name'
- **Column F**: 'Date Of Birth'
- **Column G**: 'Duplicate?' (marked 'Yes' or 'No')
- **Column H**: 'Relationship' (e.g., 'Employee', 'Spouse', 'Child')

## Logic Applied

1. Identified all rows marked **'Yes'** in the **'Duplicate?'** column (column G)
2. Grouped these rows by the combination of **Last Name**, **First Name**, and **Date Of Birth**
3. For each group of duplicates, checked if **any** entry had 'Employee' in the **'Relationship'** column (column H)
4. If the group contained at least one 'Employee' entry, **no rows were deleted** from that group
5. If the group contained **only non-Employee** entries, deleted all but one row from that group

## Implementation

Used Python with openpyxl to iterate through rows 3-42, applied the deletion logic by removing entire rows using `worksheet.delete_rows()`, and preserved the modified data in the range **B2:O42**.
