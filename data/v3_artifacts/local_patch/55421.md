# Solution Note: Appointment Status Classification Formula

## Task Summary
The data was in sheet **'Sheet1'** with headers in row 1 and data starting from row 2 through row 20.

## Column Structure
- **Column A (MRN)**: Medical Record Numbers to be analyzed
- **Column D (Appt Status)**: Appointment statuses ('Sch', 'No Show')
- **Column E (Next Appt Date)**: Next appointment dates
- **Column F (Recshed/New Appt)**: Target column for results (F2:F20)

## Logic Applied
The solution used Python with openpyxl to:

1. Read all MRN values from column A (A2:A20)
2. Read corresponding appointment statuses from column D (D2:D20)
3. Read next appointment dates from column E (E2:E20)

For each MRN, the logic grouped all rows sharing the same MRN value and determined:
- **'FUTURE'**: If all statuses for that MRN were exclusively 'Sch'
- **'NS/SCHED'**: If the MRN appeared with both 'No Show' and 'Sch' statuses
- **'NO ACTION NEEDED'**: If the status was 'No Show' and column E contained a date value

The results were written back to column F (F2:F20) for each corresponding row. The solution required checking multiple rows per MRN to identify mixed status scenarios, making it a grouping and conditional logic problem rather than a simple row-by-row formula.
