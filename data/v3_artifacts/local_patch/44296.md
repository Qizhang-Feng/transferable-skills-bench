# Solution Note: Converting Data from Sheet1 to Sheet2 by Matching States and Years

The task required populating column D ('units') in 'Sheet2' by matching state codes and years from 'Sheet1'.

**Source Data (Sheet1):**
- State codes were located in column B, starting from row 3 (B3, B4, B5, B6, etc.)
- Year headers were in row 1, columns L through R (2015-2021)
- Corresponding unit values were in the data grid starting at row 3, columns L through R

**Target Location (Sheet2):**
- State codes were in column A, starting from row 3
- Years were in column B, starting from row 3
- The 'units' values needed to be written to column D, starting from row 3 (D3:D104)

**Python/openpyxl Approach:**
1. Read the year headers from 'Sheet1' row 1, columns L through R to create a year-to-column mapping
2. Read state codes from 'Sheet1' column B (rows 3 onwards)
3. For each row in 'Sheet2' starting at row 3:
   - Extract the state code from column A and year from column B
   - Locate the matching state row in 'Sheet1' column B
   - Find the matching year column in 'Sheet1' using the year mapping
   - Retrieve the intersection value from 'Sheet1'
   - Write this value to column D in 'Sheet2'

This approach eliminated the need for complex looping by using direct cell lookups based on matched indices.
