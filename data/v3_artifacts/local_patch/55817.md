# Solution Note: Transposing Symmetrical Column Data to Row Format

The task involved transposing symmetrical data from column A of Sheet1 into a structured multi-column format starting at cell B2.

**Data Structure:**
- Source data was located in column A, rows 2-6 of Sheet1
- The data pattern was: Question 1, Option A1, Option B1, Option C1, Option D1
- Target range was B2:H6 with headers in row 1: QS, OPT 1, OPT 2, OPT 3, OPT 4, Correct Ans, Explaination

**Approach:**
The solution read values from column A starting at row 2 (A2:A6). Each set of 5 consecutive cells represented one complete question record with its four options. 

Using Python/openpyxl, the data was extracted from column A and restructured into rows. For each question group:
- The first value (question text) was placed in column B
- The next four values (options) were placed in columns C through F
- Columns G and H were populated with corresponding answer and explanation data from the existing structure

The transposition logic iterated through column A in chunks of 5 cells, writing each chunk horizontally across columns B-F in successive rows (2, 3, 4, etc.). This transformed the vertical symmetrical pattern into a horizontal tabular format suitable for a question bank or quiz structure.
